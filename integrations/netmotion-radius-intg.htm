Typical workflow

Perform the following tasks, in the order shown:

Task

Description
Download the RADIUS agent
  • Download the Okta RADIUS Agent from the Settings Downloads page your in Okta org. Note that there are both Windows and Linux agents
Install the Okta RADIUS Agent.
  • Install the Windows or Linux RADIUS agent.
Configure application
  • In your Okta org, configure the NetMotion Mobility application.
Configure NetMotion with the RADIUS settings
  • Using the NetMotion Mobility Server, configure NetMotion Mobility to work with RADIUS.
Add Trusted root CA
  • On the Windows computer running the NetMotion Mobility VPN client, and using the Microsoft Management Console (MMC) add the required additional Trusted root CA.
