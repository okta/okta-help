Before you begin

Ensure the following:

  • User:
    • Must exist in an Okta org.
    • Must be assigned the Epic Hyperdrive app in Okta.
    • Must enroll in MFA prior to performing a ePCS transaction.
  • Administrator must configure servers to allow users write access to the log hyperdrive log folder specified during agent install.
    Typically C:\Program Files (x86)\Okta\OktaHyperdriveLoginDevice\logs

Typical workflow

Task

Description
Configure TLS Configure required Transport Layer Security (TLS), ensuring that TLS 1.2 or higher is configured.
Install and configure Epic Hyperdrive in Okta Configure groups, MFA, and the Epic Hyperdrive ePCS app.
