Install and configure the RADIUS Agent

The Okta RADIUS server agent can be installed on Windows and Linux servers.

For more information see:

  • Install and configure the Okta RADIUS Server agent on Windows
  • Install and configure the Okta RADIUS agent on Linux

Configuration

Configuring integrations typically requires several steps. Each agent has specific instructions. In general:

  1. Configure factor enrollment. Most RADIUS applications support multifactor authentication. Individual applications support different factor sets. If you have not done so already, enable multifactor authentication for your users:
    1. Sign in to your Okta tenant as an administrator.
    2. Navigate to Security Multifactor .
    3. Navigate to Security Authenticators .
    4. Add any additional required multifactor policies. For complete details, see Multifactor Authentication.
    5. Add any additional required multifactor policies. For complete details, see Multifactor Authentication.
  2. Create an application.
  3. Install and configure the RADIUS agent.
