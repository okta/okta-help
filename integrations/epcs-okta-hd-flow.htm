MFA for Electronic Prescribing for Controlled Substances - Flow

MFA flow for EPCS using Epic Hyperdrive.

Flow:

  1. User launches the Epic Hyperdrive client and is prompted for username/password and begins the ePCS workflow.
  2. Username/password combination is sent to Active Directory.
  3. Active Directory authenticates the user.
  4. EPCS server, which is hosting the Hyperdrive agent, calls out to Okta for MFA.
  5. User is prompted for MFA credentials as required.
  6. MFA is verified and the and the user is signed in to Hyperdrive and allowed to continue.
