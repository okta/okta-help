Configure Cisco ASA VPN

This guide details how to configure Cisco ASA VPN to use the Okta RADIUS Server Agent.

Okta and Cisco ASA interoperate through RADIUS. For each Cisco ASA appliance, you can configure AAA Server groups, which can be RADIUS, TACAS+, LDAP, and so on. Using RADIUS, Okta’s agent translates RADIUS authentication requests from the VPN into Okta API calls.
