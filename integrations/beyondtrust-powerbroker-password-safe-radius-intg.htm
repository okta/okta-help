Configure BeyondTrust

Configure BeyondTrust PowerBroker Password Safe to use the RADIUS Server agent.

Okta and BeyondTrust interoperate using either RADIUS or SAML 2.0. For each Password Safe deployment, you can assign one or more authentication providers. Each RADIUS authentication profile maps to a group of users using a filter (for example, All Users, All Local Users, All Domain Users, Domain Contains, and so on). Using RADIUS, Okta’s agent translates RADIUS authentication requests from BeyondInsight to Okta API calls.
