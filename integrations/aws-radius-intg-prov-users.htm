Provision users

AWS Workspace users are managed in Active Directory. In production configurations, users are pulled from Active Directory into Okta

To pull users from Active Directory:

  1. Sign in to your Okta Org as an administrator.
  3. Click Add Directory > Add Active Directory.
  4. Click Setup Active Directory.
  5. Complete the wizard as required.

See Manage your Active Directory integration to learn more about integrating Active Directory with your Okta tenant.
