MFA for Electronic Prescribing for Controlled Substances (ePCS) - Flow

MFA for ePCS flow using Epic Hyperspace.

Flow:

  1. User launches the Epic Hyperspace client and begins the ePCS workflow.
  2. User is prompted for a username/password combination which is sent to Active Directory.
  3. Active Directory authenticates the user.
  4. ePCS uses the agent to call to Okta for MFA.
  5. User is prompted for MFA credentials as required.
  6. MFA is verified and the and the user is signed in to Hyperdrive and allowed to continue.
