サポート対象指標

Access Gateway /指標REST APIエンドポイントは以下を含む指標を返します：

  • クラスター - Access Gatewayクラスターに関する情報。
  • CPU - CPU使用に関する情報。
  • ディスク使用 - ディスク使用に関する情報。
  • エンジン - Access Gatewayエンジンと現在処理されているリクエストに関する情報。
  • 入力/出力 - Access Gatewayノード入力/出力に関する情報。
  • メモリー - Access Gatewayノードのメモリー使用に関する情報。
  • ネットワーク - Access Gatewayネットワークインターフェイスに関する情報。
  • Swap- Access GatewayノードSwapの使用に関する情報。

指標モニタリングサブシステムによって返されるデータの一部を次に示します。完全なリストについては、curlコマンドを実行します：

curl http://{access gateway host}:8889/metrics | grep OAG_
