System

The System menu allows you to change the hostname, install or remove a package, perform an update, reboot, shutdown, or reset the Access Gateway instance.

Select 5 to open the System sub-menu.

System commands

  1. Change Hostname: Change the hostame of the Access Gateway instance.
  2. Configure High Availability: Configure and manage Access Gateway high availability.
  3. Install Package: Install a named packaged.
  4. Remove Package: Remove a named package.
  5. Update: Update the system.
  6. Reboot: Reboot the system.
  7. Shutdown: Shutdown the system.
  8. Reset: Reset the system to original default settings.

Change Hostname

  1. Press 1 to change the hostname.
    The current hostname is displayed.
  2. Enter the new hostname, and press Enter.
  3. Press y to confirm.
  4. Press Enter to return to the System menu.

Configure High Availability

The High Availability menu displays the current role of a Access Gateway node. Roles can be:>

  • Not Set - The node has not yet been configured as either a Worker or an Admin.
  • Admin - The node has been configured as an administrator for High Availability.
  • Worker - The node has been configured as a worker for High Availability).

  1. Press 1 to reset node keys
  2. Press 2 to Prepare an administration node to connect to a new Worker node.
  3. Press 3 to Prepare a node to become a worker node.
  4. Press 4 to List all known Worker nodes. Only applicable on the Admin node.
  5. Press 5 to remove a known Worker node. Only applicable on the Admin node.
  6. Press 6 to check High Availability status.
  7. Press 7 to enter the perform a rolling upgrade using Cluster manager.

For complete details of configuring high availability, see Configure and manage high availability.

The following is for reference only, to performing rolling upgrades see Perform admin renomination.

Install Package

  1. Press 2 to install a new package.
  2. Enter a package name, and press Enter.
  3. Press y to confirm.
  4. Press Enter to return to the System menu.

Remove Package

  1. Press 3 to remove an existing package.
  2. Enter a package name, and press Enter.
  3. Press y to confirm.
  4. Press Enter to return to the System menu.

Update

See Upgrade Access Gateway for a detailed discussion on upgrading Access Gateway.
Available in Access Gateway 2021.8.0 and earlier.

  1. Press 4 to begin the update process.
  2. Confirm you are ready to update and press y to proceed or n to cancel.
  3. Press y to confirm package downloads.
    Updates will be downloaded and the update will begin.
  4. Press Enter to return to the System menu.

Access Gateway selective upgrade supports both updating from a version of Access Gateway to a later version, and upgrade to the latest version.

See Selective upgrade workflow for a detailed discussion on upgrading Access Gateway.

Available since Access Gateway version 2021.9.3

Reboot

  1. Press 5 to reboot.

  2. Enter y to proceed or n to cancel.

  3. The system will then reboot.

Shutdown

  1. Press 6 to shutdown.

  2. Enter y to proceed or n to cancel.

  3. The system will then shutdown.

Reset

  1. Press 7 to reset.

  2. Enter y to proceed or n to cancel.

  3. The system will then return all settings to their default values.

  4. Note that the gateway appliance will shutdown after a reset and must be restarted.
