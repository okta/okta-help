Metrics monitoring events

Metrics based monitoring events found in the audit log and all logs.

Topics

  • Event fields
  • Enable, disable or list current monitoring status events
  • Add, delete or examine client IP addresses address
  • Change monitoring status interval

Event fields

Access Gateway audit log metrics monitoring events include the following information:

Field

Description

TIMESTAMP

Current system date and time

HOSTNAME

Hostname of node generating event

SUBSYSTEM

The sub system or host that generated the error, such as

  • OAG ADMIN CONSOLE MONITORING MONITORING

Child SUBSYSTEM
  • NODE_EXPORTER_SERVICE
  • OTEL_CLIENT_SCHEDULER_SERVICE
  • OTEL_COLLECTOR_SERVICE
MESSAGE Free form associated message.

Enable, disable or list current monitoring status events

Events logged when enabling. disabling or examining metrics monitoring.

Current status

Description: The metrics monitoring subsystem menu was entered.

Messages:

Node exporter status - [Not] Running.OAG Metrics Collection Scheduler status - [Not] RunningOtel Collector status - [Not] Running

Example(s):

2022-01-03T20:32:21.013-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING NODE_EXPORTER_SERVICE INFO SERVICE STATUS [USER="oag-mgmt"] Node exporter status - Not Running2022-01-03T20:32:21.032-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OAG_METRICS_COLLECTION_SCHEDULER_SERVICE INFO SERVICE STATUS [USER="oag-mgmt"] OAG Metrics Collection Scheduler status - Not Running2022-01-03T20:32:21.051-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR_SERVICE INFO SERVICE STATUS [USER="oag-mgmt"] Otel Collector status - Not Running2022-01-03T20:32:21.013-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING NODE_EXPORTER_SERVICE INFO SERVICE STATUS [USER="oag-mgmt"] Node exporter status - Running2022-01-03T20:32:21.032-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OAG_METRICS_COLLECTION_SCHEDULER_SERVICE INFO SERVICE STATUS [USER="oag-mgmt"] OAG Metrics Collection Scheduler status - Running2022-01-03T20:32:21.051-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR_SERVICE INFO SERVICE STATUS [USER="oag-mgmt"] Otel Collector status - Running
  • Structured data:

    USER - User accessing the monitoring menu

  • Corrective action :

    None

    • Enabled or Disabled

    Description: The metrics monitoring subsystem was enabled, disabled or examined.

    Messages:

    Current monitoring status "[Enabled|Disabled]"[Disabled|Enabled] Otel monitoring service

    Example(s):

    2022-01-03T20:50:33.696-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR_SERVICE INFO ENABLE SERVICE [USER="oag-mgmt"] Current monitoring status "Disabled"2022-01-03T20:50:33.696-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR_SERVICE INFO ENABLE SERVICE [USER="oag-mgmt"] Current monitoring status "Enabled"2022-01-03T20:50:33.696-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR_SERVICE INFO DISABLE SERVICE [USER="oag-mgmt"] Disabled Otel monitoring service
  • Structured data:

    USER - Admin examining metrics monitoring

  • Corrective action :

    None

    • Add, delete or examine client IP addresses address

    Events logged when a client address was added or deleted.

    Current status of configured IP addresses

    Description: An admin opened the monitoring metrics manage ip addresses menu.

    Messages:

    IP status "[Not]Configured" .

    Example(s):

    2022-01-03T20:32:21.356-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING CHECK_IP_CONFIGURED INFO IP STATUS [USER="oag-mgmt"] IP status "NotConfigured" 2022-01-03T20:32:21.356-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING CHECK_IP_CONFIGURED INFO IP STATUS [USER="oag-mgmt"] IP status "Configured"
  • Structured data:

    USER - User examining metrics IP addresses

  • Corrective action :

    None

    • Current list of configured IP addresses

    Description: An admin opened the monitoring metrics manage ip addresses menu.

    Messages:

    Address [IP address] was added to the metrics monitoring subsystem.

    Example(s):

    2022-01-03T20:32:21.663-08:00 tidal.wave.com OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR INFO MANAGE ALLOWED IP [USER="oag-mgmt"] Current list of allowed IP Address/CIDR ""2022-01-03T20:32:21.663-08:00 tidal.wave.com OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR INFO MANAGE ALLOWED IP [USER="oag-mgmt"] Current list of allowed IP Address/CIDR "192.168.1.19,192.168.1.200,192.168.1.40"
  • Structured data:

    USER - User examining metrics IP addresses

  • Corrective action:

    None

    • Add

    Description: A client address was added to the metrics monitoring subsystem.

    Messages:

    Added IP Address/CIDR "192.168.1.20", new list of IP Address/CIDR "192.168.1.20"

    Example(s):

    .2022-01-03T20:52:32.561-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR INFO ADD ALLOWED IP [USER="oag-mgmt"] Added IP Address/CIDR "192.168.1.20", new list of IP Address/CIDR "192.168.1.20"
  • Structured data:

    USER - User examining currently allowed monitoring IP addresses

  • Corrective action:

    None

    • Delete

    Description: A client address was removed from the metrics monitoring subsystem.

    Messages:

    DemovedIP Address/CIDR "192.168.1.20"

    Example(s):

    .2022-01-03T20:52:32.561-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR INFO REMOVE ALLOWED IP [USER="oag-mgmt"] Removed IP Address/CIDR "192.168.1.20"
  • Structured data:

    USER - User examining currently allowed monitoring IP addresses

  • Corrective action:

    None

    • Commit

    Description: A set of IP address changes was committed.

    Messages:

    Current list of allowed IP Address/CIDR ".

    Example(s):

    .2022-01-03T20:58:14.108-08:00 tidal.wave.com OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR INFO SAVE ALLOWED IP [USER="oag-mgmt"] Enabled monitoring and Saved IP Address/CIDR "192.168.1.20 192.168.1.40"
  • Structured data:

    USER - User disabled metrics monitoring

  • Corrective action:

    None

    • List

    Description: An admin opened the monitoring subsystem manage IP addresses menu.

    Messages:

    Current list of allowed IP Address/CIDR ".

    Example(s):

    .2022-01-03T20:32:21.663-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OTEL_COLLECTOR INFO MANAGE ALLOWED IP [USER="oag-mgmt"] Current list of allowed IP Address/CIDR ""
  • Structured data:

    USER - User disabled metrics monitoring

  • Corrective action:

    None

    • Change monitoring status interval

    Events logged when change is made to the monitoring interval

    Change interval

    Description: An admin opened the monitoring subsystem change interval menu.

    Messages:

    New oag metrics collection scheduler interval (in seconds) "200".

    Example(s):

    2022-01-03T20:49:12.589-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OAG_METRICS_COLLECTION_SCHEDULER INFO CONFIG SCHEDULER [USER="oag-mgmt"] New oag metrics collection scheduler interval (in seconds) "200"
  • Structured data:

    USER - User setting new metrics monitoring interval

  • Corrective action:

    None

    • Examine interval

    Description: An admin examined the scheduler interval .

    Messages:

    Current oag metrics collection scheduler interval (in seconds) "100"

    Example(s):

    2022-01-03T20:41:23.621-08:00 [admin or node.tld] OAG ADMIN_CONSOLE MONITORING OAG_METRICS_COLLECTION_SCHEDULER INFO CONFIG SCHEDULER [USER="oag-mgmt"] Current oag metrics collection scheduler interval (in seconds) "100"
  • Structured data:

    USER - User examining monitoring interval

  • Corrective action:

    None

    • Invalid interval

    Description: An admin entered an invalid scheduler interval .

    Messages:

    Failed to set new oag metrics collection scheduleInvalid oag metrics collection scheduler interval provided "aaa"

    Example(s):

    2022-01-03T20:49:12.589-08:00 tidal.wave.com OAG ADMIN_CONSOLE MONITORING OAG_METRICS_COLLECTION_SCHEDULER INFO CONFIG SCHEDULER [USER="oag-mgmt"] Failed to set new oag metrics collection scheduler2022-01-03T20:49:12.589-08:00 tidal.wave.com OAG ADMIN_CONSOLE MONITORING OAG_METRICS_COLLECTION_SCHEDULER INFO CONFIG SCHEDULER [USER="oag-mgmt"] Invalid oag metrics collection scheduler interval provided "aaa"
  • Structured data:

    USER - User examining monitoring interval

  • Corrective action:

    Enter a integer value in the allowed range
