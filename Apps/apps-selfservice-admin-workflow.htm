Workflow to configure Self Service request feature

The following table outlines the tasks required to enable the Self Service feature and assign app integrations to your end users:

Task

Description

Role Specifications
Enable Self Service request feature Understand and configure the Self Service feature settings for your org. Super admins
Configure the Self Service approval workflow Configure the workflow to grant access and assign an app integration using the Self Service feature. Super admins or app admins
Add app integrations as an end user Understand the types of app integration requests: add an org-managed app integration, add a personal app integration, or request that admins add an app integration to the org. End users
Handle app integration requests Learn how to address requests from end users for access to app integrations managed by the org. Super admins, app admins, or business application owners
