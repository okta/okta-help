Define group attribute statements

In the Group Attribute Statements (optional) section of the Create SAML Integration page, do the following:

  1. Enter a Name for the group attribute in your SAML app.
  2. Select a Name Format.
  3. Choose a Filter option for your expression.
  4. Enter the expression to match against GroupName values and add to the SAML assertion.
  5. Optional. Click Add Another, and then repeat steps 2–4 to add another attribute.

The Dynamic SAML feature enables apps in the Integration Network to process SAML attribute statements. Previously, the attribute statements were only available for apps created using the App Integration Wizard. This feature doesn't change how you enter attribute statements in the , or how the statements are processed.
