Provision applications

Lifecycle Management provisioning features automate account management for your organization. The System for Cross-domain Identity Management (SCIM) specification is used to perform provisioning actions between and cloud-based or on-premises applications. After you configure and select the attributes you want to sync with external applications, the user profile is automatically updated when changes are detected.
