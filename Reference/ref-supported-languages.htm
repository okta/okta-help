Admin Console languages

The supports the following languages:

  • English

  • 日本語 (Japanese)

If your language isn't supported, the Admin Console appears in English.

Some of the text in the can't be translated. These items always appear in English:

  • User-input values

  • Variables (for example, expressions)

  • System Log events

  • Notifications from third-party apps

  • The names and descriptions of default user groups and authentication policies

  • The Features page

, , , and are only available in English.
