View the default user profile

View the default user profile to view the base and custom attributes associated with the profile. There are 31 default base attributes for all users in an org. The only base attributes you can modify are First Name and Last Name.

  1. In the , go to Directory Profile Editor .
  2. Click Okta in the Filters list.
  3. For User (default), click Profile. If Profile is unavailable, click User (default).
