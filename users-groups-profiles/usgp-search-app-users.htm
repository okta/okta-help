Search for application users

Search for an application user to determine if the user is assigned to the application. You can search for users using a full or partial first name, last name, or email address. You cannot use the userName attribute to search for users.

  1. In the , go to Applications Applications .
  2. Enter the application name in the Search field.
  3. Click the application name and click the Assignments tab.
  4. Enter a full or partial user first name, last name, or email address in the Search field.
