Edit default group profile custom attributes

Edit the default group profile custom attributes when you want to change the display name, add or edit an attribute description, add or edit the attribute length, or indicate if the attribute is required.

  1. In the , go to Directory Profile Editor .
  2. Click the Groups tab.
  3. Click Okta in the Filters list.
  4. Click the Okta group title.
  5. Click Custom in the Filters list.
  6. Click the attribute display name.
  7. Edit the attribute fields and click Save.
  8. Optional. Repeat steps 6 and 7 to edit additional custom attributes.
