Add self-service password reset to an existing password policy

Adding a self-service password reset option to an existing policy lets users reset their passwords.

  1. In the , go to Security Authentication .
  2. Select an existing password policy and scroll down to the Add Rule area.
  3. Click Edit for an existing rule.

If a rule doesn't exist or can't be edited, click Add Rule, complete the mandatory and optional fields, select the options in step 4, and click Create Rule.

  1. In the THEN User can area, select the change password and perform self-service password reset check boxes.
  2. Click Update Rule.
