Enforce custom attribute uniqueness

  1. In the , go to Directory Profile Editor .
  2. Click Okta in the Filters list.
  3. For User (default), click Profile. If Profile is unavailable, click User (default).
  4. Click Custom in the Filters list.
  5. Scroll to the custom attribute you want to mark as unique.
  6. Click Edit.
  7. Select the Restriction check box and select that the value must be unique for each user.

    When the validation process to verify the existing data is unique is running, the check box is unavailable. Clear the check box to remove the uniqueness requirement.

  8. Click Save Attribute.

A status message on the Profile Editor page displays the validation progress and status.
