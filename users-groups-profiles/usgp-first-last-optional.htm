Make the user profile first and last name optional

Of the 31 default base attributes, the only base attributes you can modify are First Name and Last Name. You can make these two attributes required or optional for sourced users. If you want to import users with empty First Name and Last Name attributes, you'll need to make the attributes optional or the import fails.

  1. In the , go to Directory Profile Editor .
  2. Click Okta in the Filters list.
  3. For User (default), click Profile. If Profile is unavailable, click User (default).
  4. Click Base in the FILTERS list.
  5. Click information for the First Name attribute.
  6. Clear the Attribute required check box.
  7. Click Save Attribute.
  8. Repeat steps 5 to 7 for the Last Name attribute.
