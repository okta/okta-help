Configure voice call for self-service password resets

With voice call configured, users receive a voice call containing a recovery code to their mobile device or land line phone. These are the available languages:

  • Chinese (simplified)

  • Chinese (traditional)

  • Dutch

  • English (US)

  • English (UK)

  • English (Canada)

  • French
  • French (Canada)

  • German

  • Italian

  • Japan

  • Korean

  • Spanish

  • Taiwanese

The call is made in the user's default language and cannot be customized. If the user's default language is not in the list of available languages, the call is made in English.

  1. In the , go to Security Authentication .
  2. Select an existing password policy, click Edit in the right pane, and scroll down to the Account Recovery area.
  3. Select Voice Call.
  4. Click Update Policy.
