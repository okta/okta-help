Delete custom attributes from a user profile

Delete a custom user attribute when it is no longer required.

  1. In the , go to Directory Profile Editor .
  2. Click Okta in the Filters list.
  3. For User (default), click Profile. If Profile is unavailable, click User (default).
  4. In the Attributes area, select Custom in the Filters list.
  5. Locate the attribute in the list and click X.
  6. Click Delete Attribute.
  7. Optional. Repeat steps 5 and 6 to remove additional custom attributes.
