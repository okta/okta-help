Import users from an app

When you import users into your org from an app, you can schedule the import, define how imported users are confirmed and how the application fields are mapped to an username. If the app supports incremental imports, you can also indicate whether you want to import all of the data or only the data that has changed since your last import.

  1. In the , click Applications Applications .
  2. In the Search field, enter the name of the application.
  3. Click the application name link.
  4. Click the Import tab.
  5. Click Import Now.

    6. If the app supports incremental imports, select a full or incremental import and click Import.

  6. Optional. Click OK when the import finishes.
