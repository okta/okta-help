Remove groups imported from provisioning-enabled apps

Remove all groups imported from an application when they are no longer required. You cannot disable group import when group policy rules or mappings exist, or when a group selected for deletion is assigned to an app.

  1. In the , click Applications Applications .
  2. Select a provisioning enabled application.
  3. Click the Provisioning tab and select Integration in the Settings list.
  4. Click Edit.
  5. Clear the Import Groups check box.
  6. Click Continue in the Disable Import Groups dialog box.
  7. Click Save.

The application groups are removed from during the next scheduled or manual import.
