View the Import Monitoring dashboard

The Import Monitoring dashboard displays user attribute imports for a seven day period. Use the dashboard to view import progress, status, details, and logs.

The Import Monitoring dashboard can only be viewed by app admins with view permissions on all apps.

  1. In the , go to Reports Import Monitoring .
  2. Optional. In the Show All list, select an app to display imports for a specific app.

The Show All list displays a maximum of 200 applications.

  1. Optional. In the Status list, select Completed, In progress, or Failed to display only completed, in progress, or failed imports.
  2. Optional. Click expand to view details about in progress or completed imports.
  3. Optional. Click Logs in the Logs column to view import log information on the System Log dashboard.
