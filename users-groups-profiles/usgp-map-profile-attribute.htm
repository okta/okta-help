Map profile attributes

Map profile attributes to associate an app attribute such as company name with a similar attribute in . If you changed an attribute's Source priority to Override profile source, the attribute has a null value and must be mapped. Attributes that are not mapped from the primary profile source have a null value and a value is not pulled from other source apps in the priority list.
