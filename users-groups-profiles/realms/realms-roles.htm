Roles and permissions

Roles Description
Super admin Only Okta super admins can create or delete a realm.
Custom admin Realms don't have an admin by default. A custom admin role must be created to allow third-party admins to manage their realm users. Realm admins can manage users, assign apps to users through groups, and review which groups and apps are assigned to the realm.

User

Realm users can access applications assigned to them by the realm admin.
