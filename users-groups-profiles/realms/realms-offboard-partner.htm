Offboard a partner

To offboard a partner, you must deactivate and delete the IdP, delete all users from the realm, and then delete the realm.

Before you begin

  • Ensure that you’re signed to your Okta org as a super admin.

Start the task

  1. Deactivate and delete an IdP
  2. Deactivate and delete all users
  3. Delete a realm

Deactivate and delete an IdP

  2. Click the IdP you want to delete.
  3. Click Active Deactivate. A delete button appears.
  4. Click Delete to confirm.

Deactivate and delete all users

  1. In the Admin Console, go to Directory Realms.
  2. Select the realm that contains the users you want to delete.
  3. Select the user accounts that you want to deactivate, and then click Deactivate Selected.
  4. Click More Actions Deactivate.
  5. Click Deactivate. A delete button appears.
  6. Click Delete.
  7. Click Delete to confirm.

Delete a realm

  1. In the Admin Console, go to Directory Realms.
  2. Click Actions, and then select Delete.
  3. Click Delete realm to confirm.
