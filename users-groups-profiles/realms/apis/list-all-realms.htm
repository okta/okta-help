List all Realms

GET /api/vi/realms

OAuth 2.0: okta.realms.read

Lists all realms

Request

Query Parameters

limit integer <int32>

Default: 200

Specifies the number of results returned. Defaults to 10 if search is provided.
after string

The cursor to use for pagination. It is an opaque string that specifies your current location in the list and is obtained from the Link response header. See Pagination for more information.
search string

Searches for realms with a supported filtering expression for most properties.

sortBy string

Specifies field to sort by and can be any single property (for search queries only).

Example: sortBy=profile.name
sortOrder string

Default: "asc"

Specifies sort order asc or desc (for search queries only).

This parameter is ignored if sortBy is not present.
