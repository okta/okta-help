Manage realm users

You can move users from one group to another within a realm.

Before you begin

  • Ensure that you’re signed to your Okta org as a super admin.

Start the task

  1. Go to DirectoryRealms.
  2. Select a realm.
  3. Select the user that you want to move to another group.
  4. Go to the Groups tab on the user’s page.
  5. Delete the group that the user is associated with.
  6. Search and select the group to which to add the user.
