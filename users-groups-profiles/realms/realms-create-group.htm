Create a group

Groups simplify user management and allow realm admins to categorize users and provide them access to applications based on the user's identity and function. A user in a realm can belong to multiple groups within that realm. These groups can be edited by realm admins and Okta admins who have access to edit existing Okta groups.

  1. In the Admin Console, go to Directory Groups.
  2. Click Add Group.
  3. Enter the details for the group.
  4. Click Save.
