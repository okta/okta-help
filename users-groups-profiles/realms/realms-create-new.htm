Create a realm

As a super admin, you can create realms for your org and assign admin roles to users within those realms. When the realms feature is enabled, a default realm that contains all users is created for your org. The default realm can’t be deleted. You can create up to 500 realms per org, and add users to your realms manually in the , through Just-in-time (JIT) provisioning, or through the Users API.

Before you begin

  • Ensure that you’re signed in as a super admin.

Start the task

  1. In the Admin Console, go to Directory Realms.
  2. Click Create new realm.
  3. Enter a realm name and click Create realm.

You can now add an (IdP) and share apps with the realm.

Edit a realm

  1. In the Admin Console, go to Directory Realms.

  2. Click Edit for the realm that you want to edit.

  3. Edit the realm name and click Save.
