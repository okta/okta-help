Add an

When you configure an (IdP), you can choose a realm from where the user will be provisioned. If a realm isn't assigned to the IdP, the default realm is automatically selected. A realm can be associated with a maximum of 10 IdPs. Realm supports SAML 2.0 or OIDC s.

Users can only exist in one realm, in addition to the default realm. This means that an IdP with the same user can’t be configured for two realms.

Before your begin

  • Ensure that you’re signed in as a super admin.
  • You must have an existing realm. See Create a realm.
