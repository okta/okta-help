Manage self-service registration settings

This is where you'll find the information you need to manage self-service registration (SSR). SSR lets users use a custom app or the Homepage to self-register. After you enable SSR, a Sign up link appears in the Sign-In widget. When this link is selected, users are directed to a new Create Account registration form that is defined by a customized registration policy. To enable SSR, super admin permissions and the Sign-in widget version 2.9 or later are required.
