Remove group users manually

It is possible to remove users from a group manually through the individual user profile. You can remove the user if the user was assigned by a group rule that assigns users to one group only. You cannot remove the user if the user was assigned by a group rule that assigns users to multiple groups

  1. In , go to Directory Groups .
  2. Click the All tab and then select a group.
  3. Click the user name, and select the Groups tab.
  4. Click X for the user you want to remove from the group.
  5. Click Remove in the Remove Person from Group dialog box.
