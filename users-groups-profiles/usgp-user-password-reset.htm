Self-service password reset scenarios

This table details the steps users must take when self-service password reset is configured for your org.

Status Required Action
New user
  • Click Add phone number.
  • Enter a mobile phone number to receive an initial verification code.
  • Enter the verification code to authenticate in to .
Active user
  • Click the user name at the top of the home page, and then select Settings.
  • In the Forgot Password Text Message section, click Add phone number.
  • Enter a mobile phone number to receive an initial verification code.
  • Enter the verification code to authenticate in to Okta.
User needs to recover password
  • On the sign-in page, click the Forgot password? link.
  • Click Send Text Message and then continue through the reset prompts.

Invalid users who attempt entry through a Forgot Password or Unlock Account action don't see an error message, because an error message could reveal if the user account is valid. See Multifactor Authentication

User needs to reset or reconfigure their smartphone Users who lose a phone or get a new number can reset or reconfigure their phones by updating their Home Settings page.
