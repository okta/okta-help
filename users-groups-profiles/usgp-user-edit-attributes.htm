Allow users to edit attributes

Making some -sourced attributes editable lets users keep specific attributes current.

  1. In the , go to Directory Profile Editor .

  2. Click Profile for the user type.
  3. In the Attributes area, click information for the attribute that you want to edit.
  4. Select one of these options in the User permission list:
  • Hide — Select this option to hide the attribute field from the user.
  • Read Only — Select this option to make the attribute field read only.
  • Read-Write — Select this option to make attribute field editable.
  1. In the Source priority list, choose Inherit from Okta.
  2. Click Save Attribute.
