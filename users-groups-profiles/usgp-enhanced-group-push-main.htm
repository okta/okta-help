Manage Group Linking

Use Group Linking to push from to existing groups in specific apps. While generally you cannot push a group with a name that already exists within the target app, these apps do allow the enhanced capability. is the source for these exchanges.

When a group already exists in an application, pushing by rule fails because the creation of a duplicate group is not supported.

To determine if your application integration supports group linking, check the Capabilities tab for the application in the Integration Network integration catalog. See Okta Integration Network.

Group Linking is available for the Samanage app as Early Access. Contact Support to enable it.
