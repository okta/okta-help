Confirm Oracle Forms application attributes

Attributes are used to pass information from to the back end protected web resource using header fields. Oracle Forms applications require the SSO_USERID attribute but can take advantage of other attributes.
During this task we will confirm required attributes and add any additional attributes.

  1. The Attributes pane provides a list of attributes that are passed into the application as header fields.
