Launch a Nutanix VM

During this task we will launch a VM and open a console.

  1. Return to the Nutanix Prism Central console.
  2. In the main navigation menu select (Top menu) > VM.
    The VM pane will display.
  3. If not already displayed, select the Table tab.
  4. Select the row containing the VM created during task Create a Nutanix VM.
  5. Click Power on.
    The VM will power up and after a few moments the Launch Console option will become enabled.
  6. Click Launch Console.
  7. Login to the and continue normally.

The new VM is now up and ready for configuration.
