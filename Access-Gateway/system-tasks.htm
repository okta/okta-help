Common system tasks

application administrators perform several common tasks. Typically these tasks involve adding applications. Applications sometimes require additional policies, changes to application behavior, and other modifications.

Task Description

Backup and restore

Back up configuration to a local, non-appliance based disk, so that the configuration can be restored. See About backup and restore.

Download logs

supports downloading system logs. See About [%=okta-feature-names.Access Gateway%] logs.

Configure log forwarders

supports log forwarding to logging systems (for example, Graylog). See Administer logging

Upgrade latest workflow

Selective upgrade workflow

Upgrade to either the latest version or to a selected (intermediate) version. These tasks describe the process of selective (intermediate version) and traditional (latest version) upgrade. See Upgrade [%=okta-feature-names.Access Gateway%]

Manage trusted domains

Trusted domains, sometimes referred to as trusted origins, define a set of top level domains that are allowed for redirects. See About trusted domains

Proxy - should be in the OAG general tasks list.
