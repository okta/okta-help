Before you begin

Ensure that:

  • You have an external Oracle Demantra-based application that requires protection. supports Oracle Demantra v10g or later.
  • Appropriate DNS entries for both the legacy application and the exposed new URL exist.
  • Appropriate DNS entries for both the protected application and the external URL for example:
    ValueDescription
    https://demantra-app.externalexample.comExternal or customer facing application URL.
    https://demantra-app.internalexample.com

    Internal or protected application URL of Oracle Demantra application.
