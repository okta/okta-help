Architecture

Before you begin

Ensure that:

  • You have an external Oracle Forms-based application that requires protection. supports Oracle Forms v10g or later.
  • Appropriate DNS entries for both the legacy application and the exposed new URL exist.
  • Appropriate DNS entries for both the protected application and the external URL exist. For example:
    ValueDescription
    https://app-forms.externalexample.comExternal or customer facing application URL.
    https://app-forms.internalexample.com

    Internal or protected application URL of Oracle Forms application.
