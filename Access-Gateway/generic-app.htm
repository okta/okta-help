Generic apps

supports several generic app types.

  • Add a generic header application: This app type doesn't require specific settings. provides single sign-on (SSO) by sending HTTP headers to this type of app.
  • Add a Kerberos application: This topic describes how to add a Kerberos-based app to .
  • Add a portal application: Configure access policies for a header-based app that routes incoming traffic to specific back-end servers.
  • Add a SAML pass-through app: Security Assertion Markup Language (SAML) pass-through apps are a combination of apps in an Okta org, an SAML proxy app, and an associated configuration.
  • Add a SharePoint application: This topic describes how to add a SharePoint-based app to .
  • Add a Websocket-based application: Configure a header-based app and create a policy for handling traffic to WebSocket-based URIs.
