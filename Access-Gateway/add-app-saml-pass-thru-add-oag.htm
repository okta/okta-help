Create an SAML proxy application

During this task we will create the header application, which will proxy requests to the SAML pass through application for accesses from the general internet.

Create the application in

  1. On the row containing the oag username attribute, click the Delete icon as the attribute is not required.
  2. Click Yes to confirm the deletion.
  3. Click Next.
  4. Click Done.
