Reset and verify configuration

To reset the instance:

  1. Sign in to the using the [%=okta-feature-names.Access Gateway%] default credentials.
  2. Select 5 System.
  3. Select 7 - Reset.
  4. Select Y to clear the configuration.
  5. Select Y to initialize the system. The instance pauses for a short time during the reset.
  6. When the initialization is complete, press any key to return to the menu.
  7. Reboot after you complete the reset. See the Reboot section of System menu for instructions.

Verify configuration

  1. Use x to return to the main menu.
  2. Select 1 - Network .
  3. Select 2 - Test network configuration. attempts to contact www.okta.com and reports any errors.
  4. Select 7 - Connectivity test. Enter www.okta.com for the host and 443 for the port.
  5. Confirm that the connection was successful.
