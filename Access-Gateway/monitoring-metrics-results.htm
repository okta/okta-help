Supported metrics

The /metrics REST API endpoint returns metrics, including:

  • Cluster - information about the cluster.
  • CPU - Information about CPU use.
  • Disk usage - Information about disk use
  • Engine - Information about the engine and current requests being handled.
  • Input/output - Information about node input/output.
  • Memory - Information about node memory use.
  • Network - information about network interfaces.
  • Swap- Information node swap use.

The following is a partial list of data returned by the metrics monitoring subsystem. For a complete list, run the curl command:

curl http://{access gateway host}:8889/metrics | grep OAG_
