Enable FIPS mode

Federal Information Processing Standards (FIPS) mode provides cryptographic protection that's compatible with United States federal government standards.

Enable FIPS mode in

  1. Connect to the target instance of .
  2. Enter 5 - System.
  3. Enter 9 - Enable FIPS Mode.
  4. At the Enable FIPS mode for operating system? prompt, enter Y to enable FIPS mode, or n to return to the previous menu item.
  5. Press the Enter key to return to the main menu.
  6. Restart to apply the changes.
