Upgrade to a specific version

You can update to the latest version, or to a specific version later than your current version.

See Upgrade [%=okta-feature-names.Access Gateway%] to the latest version.

Before you begin

Upgrade to a specific version

Perform this procedure on the node that you want to upgrade:

  1. Connect to the target instance .
  2. Press 5 - System.
  3. Press 4- Update.
  4. Press 2 - View list of Access Gateway versions. . .. The list of available versions appears.
  5. Press the [up]/[down] keys or arrows to select a target and press Return or Enter to begin the upgrade process.
  6. In the upgrade confirmation dialog, press 1 to begin the upgrade.
  7. After the upgrade is complete, press any key to continue.
  8. Press y to view the yum upgrade log, or x to exit and return to the main upgrade menu.
