Content

The Content Update menu allows you to create and manage GIT repositories on your application. Note that only one repository is supported at a time.

Content update commands

  1. Setup Repository:Set up a GIT repository
  2. Change Branch: Create branch.
  3. Review Changes: Review all changes.
  4. Update Content: Push the content update to the repository.
  5. Rollback Latest Change: Rollback the latest change.

Setup Repository

  1. Press 1 to begin setting up a local Git repository.
  2. Enter the complete path to the Git repository, and press Enter. This replaces any existing repository.
  3. If found, the repository will be cloned locally.
  4. Press Enter to return to the Content menu.

Change Branch

  1. Press 2 to enter a new branch.
  2. Enter the branch and press Enter.
  3. Press Enter to return to the Content menu.

Review Changes

  1. Press 3 to review changes.
  2. Press Enter to return to the Content menu.

Update Content

  1. Press 4 to update content.
  2. Press y to proceed with update.
  3. Press Enter to return to the Content menu.

Rollback Latest Change

  1. Press 5 to rollback change.
  2. Press y to configure rollback.
  3. Press Enter to return to the Content menu.
