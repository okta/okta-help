Metrics monitoring

produces metrics that monitoring systems can use.

Metrics-based monitoring includes:

  • Metrics collector: implements a custom collector that tracks metrics such as swap, CPU, disk, I/O, and other metrics
  • Publication format: provides metrics in the Open Metrics format.
  • Transport: exposes metrics using either REST and a well-defined endpoint, or a push mechanism.

Before you begin

Ensure that:

  • Port 8889 is open between the monitoring implementation and nodes.

Description
