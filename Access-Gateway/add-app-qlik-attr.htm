Confirm Qlik application attributes

Attributes are used to pass information from to the back end protected web resource using header fields. Oracle WebLogic Server applications require the QVUSER attribute but can take advantage of other attributes.
During this task we will confirm required attributes and add any additional attributes.

  1. The Attributes pane provides a list of attributes that are passed into the application as header fields.
    Confirm the attributes match those required by Qlik.

  3. Leave all policies unchanged and click Done.
