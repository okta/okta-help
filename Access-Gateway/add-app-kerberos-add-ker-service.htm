Add Kerberos service

In order to interact with Windows using Kerberos, a Kerberos service is required.

  1. Sign into the .
  2. Select the Settings tab.
  3. Select the Kerberos pane.
  4. Click Add.
  5. Enter the service account details noted earlier.
  6. Expand the Windows Server Commands section.
    This section contains the commands that you must execute on the Windows server to create the required keytab.
  7. Expand the Configuration section.
  8. Click Choose File.
  9. Navigate to the directory containing the previously created keytab file and upload.
  10. Click Validate. The keytab file must be validated before continuing.
  11. Click Okay.
