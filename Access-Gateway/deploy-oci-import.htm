Import an OVA to OCI

During this task we will import an OCI image.

  • Console
  • Command Line

To Import the OVA:

  1. In the Oracle Cloud Console, navigate to Compute > Custom Images.
  2. Click Import image.
  3. In the Import Image dialog box:
    FieldValue
    NAMEAny appropriate name
    OPERATING SYSTEMLinux
  4. Select Import from Object storage bucket.
  5. From the Bucket in ... dropdown, select the bucket created in Upload an OVA to an OCI storage bucket.
  6. From the Object dropdown select the uploaded OVA.
  7. Click Import Image.
  8. Wait while the image is imported.

To import OVA

At the time of this writing import of OVA at the command line level was not supported by Oracle Cloud Infrastructure.
