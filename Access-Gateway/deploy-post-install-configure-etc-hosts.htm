Add admin entry to hosts file

To initially access , you must add an admin entry to your local hosts file. Your operating system determines the location of this file:

Operating System File

macOS

Linux

 /etc/hosts
Windows C:\Windows\System32\drivers\etc\hosts

Add an entry to your hosts file that has the following format:

<ip_of_instance> admin

You can determine the IP address of the by following the steps in Show a running configuration.

For example, if your gateway's IP address is 192.168.32.129, you'd add the entry:

192.168.32.129 admin # Okta Access Gateway admin entry

After you add the admin entry, you should be able to access the by opening https://admin in your local browser.

You only need the admin entry in the hosts file for the initial setup and configuration of . When that's complete, you can remove the entry.
