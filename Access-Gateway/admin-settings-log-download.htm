Download log files

enables you to download log files to a local disk.

A minimum of two gigabytes of free disk space is required to download log files.

  1. Go to the .
  2. Select the Logs tab.
  3. Select the Log Storage pane.
  4. Select the Download pane.
  5. Use the date picker to select the date or date range for the log events you want to download. If you receive an insufficient space error, select a smaller date range or configure a log forwarder.
  6. Click Prepare download.
  7. When indicates that the download is ready, click Download to download the log file.
