Create the application

During this task we will create the admin application.

  1. Select the Access Gateway AdminUI option from the left column menu, and click Create.

    The New Protected Application wizard starts and displays the Setting tab.

  2. In the Essentials pane enter:
    FieldValue
    LabelThe name of the application,
    For example: Access Gateway Admin UI
    Public DomainThe external facing URL of the gateway. Typically similar to gw-admin.[domain.tld].
    For example: gw-admin.myaccessgateway.com
    Post Login URL

    Leave unchanged.

    GroupEnter the name of group created in the prior step.

  3. Click Next. The Attributes page will display.

    The attribute set specified by the add application wizard is required by . Do not modify this list.

  4. Click Next. The Policy page will display.
  5. Click Done.
