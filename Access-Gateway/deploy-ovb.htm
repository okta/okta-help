Deploy the OVA file to Oracle VirtualBox

only supports using Oracle VirtualBox for development and testing. doesn't support deploying to Oracle VirtualBox for production use.

Download the latest OVA file

  1. Sign in to your Okta org as an admin.
  3. Go to the section and download the Virtual Appliance for VMWare OVA file to an appropriate location.

Import the OVA file into Oracle VirtualBox

  1. Open Oracle VirtualBox.
  2. Select FileImport Appliance.
  3. Click Folder to browse to the directory where the OVA file was downloaded.
  4. Select the Okta Access Gateway OVA file, and then click Open.
  5. Click Next.
  6. Specify the name as and verify that the Guest OS Type option is set to Red Hat (64-bit).
  7. Select the Reinitialize the MAC address of all network cards option, and click Import on the Appliance settings page. Oracle VirtualBox imports the OVA file.
  8. Click Settings.
  9. Select Storage.
  10. If the disk is associated with the SCSI device, select and drag it below the IDE device entry, and then click Ok.
  11. Click Start.
