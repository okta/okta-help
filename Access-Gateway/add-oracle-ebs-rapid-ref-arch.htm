Oracle E-Business Suite with Rapid SSO reference architecture

This reference architecture describes the components, flow and version requirements for integrating Oracle E-Business Suite and when configured for EBS Rapid SSO.

  • Architecture
  • Flow
  • Components

Architecture



Flow

  1. User sign in
  2. Send EBS user identity
  3. Verify EBS user (generates EBS session)
  4. Redirect to EBS with the EBS session cookie
