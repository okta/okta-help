Command Line Management Console reference

is a command line interface to the instance that allows you to modify configuration that are not available in the .

The purpose of this reference is to document the commands available within the oag-mgmt user account. These commands can be used to configure and monitor Management.

The following management areas are available with the :

  • First login
  • Network
  • Service
  • Kerberos
  • Monitor
  • System menu
  • Change [%=okta-feature-names.Access Gateway CMD%] password
  • Change Web Console Password
  • Content
  • Support connection
  • Client certificate chains

To avoid publication of public IP addresses that belong to Okta or any other organization, the IPv4 addresses used in Okta technical documentation are fictional and are from the private IP address range based on CIDR 192.168/16.
