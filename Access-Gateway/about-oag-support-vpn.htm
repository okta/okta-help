This access is restricted to select members of , Professional Services, and Engineering. It requires an access API token during configuration, and the client firewalls to allow outgoing TCP traffic on port 443.

Operations

When connected to a client appliance, staff can perform three types of operations:

  • Admin: support staff can sign in and execute operations using the to perform normal administration activities.
  • Command-line: support staff can use the tool to connect to and execute commands to enhance, diagnose, or correct instance issues.
  • File transfer: support staff can copy files to and from the appliance to upload and capture configuration, logs, and similar information.

When connects to the support VPN, a specific IP address is assigned. To determine the current IP address:
