Unbound log

Unbound is a validating, recursive caching DNS server. uses unbound as cache-only forwarding DNS server.

Unbound messages are useful when troubleshooting general connectivity issues.

Before you begin

  • See Download logs for details on downloading logs.
  • See Decompressing logs for details on decompressing log files.

Event fields

Field

Description

TIMESTAMP

The current system date and time

HOSTNAME

The hostname of the node generating event

SUBSYSTEM or HOST

The sub-system or host that generated the error, such as

  • systemd - system daemon running NGINIX
  • emerge
MESSAGE Free form message information about the error
