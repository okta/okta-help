Create the Oracle BI Enterprise Edition application

Create the Oracle BI Enterprise Edition application in .

  1. Select Oracle BI Enterprise Edition from the left menu. Click Create.

  2. In the Essentials pane specify the following:

    Field Value
    Label A name for the application, such as Oracle Hyperion application.
    Public Domain A fully qualified host name such as <yourexternalname>.<your domain>.
    In the example provided this would be obiee.externalexample.com.
    Protected Web Resource The URL of the internal, protected application. This example uses obiee.internalexample.com:<port>
    Where:
    • port: The port Oracle BI Enterprise Edition listens on for HTTP requests.

    See Configure load balancing.
    Group Enter the group containing the users who should have access to the application.

    Post Login URL

    The Post Login URL. By default this field is enabled and contains the value:

    https://obiee.externalexample.com/analytics/

    Description Optional. A description for your app.
