System menu

The System menu allows you to change the hostname, install or remove a package, perform an update, reboot, shutdown, or reset the instance.

Select 5 to open the System submenu.

System commands

  1. Change Hostname: Change the hostame of the instance.
  2. Configure High Availability: Configure and manage high availability.
  3. Install Package: Install a named packaged.
  4. Remove Package: Remove a named package.
  5. Update: Update the system.
  6. Reboot: Reboot the system.
  7. Shutdown: Shut down the system.
  8. Reset: Reset the system to its original default settings.

Change Hostname

  1. Press 1 to change the hostname.
    The current hostname is displayed.
  2. Enter the new hostname, and press Enter.
  3. Press y to confirm.
  4. Press Enter to return to the System menu.

Configure High Availability

  1. Press 1 to reset the node keys.
  2. Press 2 to prepare an administration node to connect to a new worker node.
  3. Press 3 to prepare a node to become a worker node.
  4. Press 4 to list all known worker nodes. This is only applicable on the admin node.
  5. Press 5 to remove a known Worker node. Only applicable on the Admin node.
  6. Press 6 to check High Availability status.
  7. Press 7 to perform a rolling upgrade using the Cluster manager.

For complete details of configuring high availability, see Configure and manage high availability.

For a reference on performing rolling upgrades see Perform admin renomination.

Cluster manager

  1. Authorize node nomination.
  2. Authorize node as Admin node.

Install Package

  1. Press 2 to install a new package.
  2. Enter a package name, and press Enter.
  3. Press y to confirm.
  4. Press Enter to return to the System menu.

Remove Package

  1. Press 3 to remove an existing package.
  2. Enter a package name, and press Enter.
  3. Press y to confirm.
  4. Press Enter to return to the System menu.

Update

See Upgrade [%=okta-feature-names.Access Gateway%] to the latest version for instructions on updating to the latest version.

See Upgrade [%=okta-feature-names.Access Gateway%] to a specific version for instruction on updating to a specific version.

Reboot

  1. Press 5 to reboot.

  2. Enter y to proceed or n to cancel.

  3. The system reboots.

Shutdown

  1. Press 6 to shut down.

  2. Enter y to proceed or n to cancel.

  3. The system shuts down.

Reset

  1. Press 7 to reset.

  2. Enter y to proceed or n to cancel.

  3. The system returns all settings to their default values.

The gateway appliance shuts down after being reset and must be restarted.
