Configure High Availability

The admin user interface (UI) is only available on admin nodes and nodes that haven't been configured as worker nodes. After an instance is configured as a worker, you can't access it using the admin UI. All administration of an cluster is done through the admin node and its admin UI.

Overview

Operations

To configure high availability:

  1. Reset keys - Reset the keys in both the admin instance and the worker instance.
  2. Add a worker node - Add one or more worker nodes to the cluster.
  3. Check configuration - Review the final configuration.
