Set DNS Servers

deployments often use the concept of split DNS, where two DNS servers handle requests for a single domain. One server handles requests from the public internet, and the other server handles requests from the internal network.

To configure DNS servers:

  2. Enter 1 - Network.
  3. Enter 8 - Manage DNS settings.
  4. Enter 1 - Manage DNS servers.
  5. Enter 1, 2, or 3 to set the primary, secondary, or tertiary DNS server, respectively.
  6. Repeat as required to set other DNS servers.
  7. Enter s to save or x to exit discarding changes.
  8. Enter x to return to the Manage network interfaces menu.

Depending on how the current IP address of is defined you may need to enter the static IP address into either and /etc/hosts file or DNS.
