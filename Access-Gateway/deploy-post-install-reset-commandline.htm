Reset the gateway

You don't need on-premises installations to reinitialize .

At the command-line main menu:

  1. Enter 5 - System.
  2. Enter 7 - Reset.
  3. Enter Y to clear the configuration.
  4. Enter Y to initialize the system.

The instance may pause for 1-2 minutes during reset.

After the system is initialized successfully, press any key to return to the menu.

Verify Configuration

  1. Enter x to return to the main menu.
  2. Enter 1 - Network.
  3. Enter 2 - Test network configuration. Okta attempts to contact www.okta.com and report any errors.
  4. Enter 7 - Connectivity test. Enter www.okta.com as host and 443 as port.
  5. Confirm that the connection was successful.
