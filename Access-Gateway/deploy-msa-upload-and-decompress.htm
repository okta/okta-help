Upload and decompress

After a disk is available, we can upload and decompress the Microsoft Azure disk image.

  1. At the command line, connect to the new VM using the IP address from the previous task and ssh.
    ssh okta@AA.BB.CC.DD.
  2. Using wget, download the latest Microsoft Azure fixed disk image.

    For example:

    cd /home/okta wget https://download.oag.okta.com/ga/oag_azure.vhd.gz
  3. Unpack the fixed disk image to a temporary location on the /datadrive.

    For example:

    sudo mkdir /datadrive/temp sudo cp oag_azure.vhd.gz /datadrive/ sudo gzip -d oag.vhd.gz --keep

Depending on the size of the disk and the speed of the VM, it can take anywhere from 60 to 90 minutes to uncompress the disk file.
