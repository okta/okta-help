Prerequisites

  • Review this topic: [%=okta-feature-names.Access Gateway%] deployment prerequisites.
  • Verify that is resolvable using DNS and is required to have a hostname assigned in DNS.
  • Verify that can use a static IP Address. doesn't support using DHCP for its static IP address.
  • Verify that a DNS Server and DNS Hostname (fully qualified domain name) are available to . must be resolvable using a DNS solution that end users leverage. If end users originate from the internet, the solution must be publicly resolvable. Okta recommends that you use split DNS so that internal users connect to an internal IP address and external users connect to a public IP address.

    The fully qualified domain names for all applications integrated with the must be resolvable in DNS.

Deploy to VMWare vSphere

Download the latest OVA image

  1. Sign in to your Okta Org as an admin.
  2. Download the Okta image from the Settings Downloads page.
  3. When prompted save the OVA file to an appropriate location.
