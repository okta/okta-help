Monitor with the

  1. Open a terminal. Use SSH to connect to the .

    Examples:

    ssh oag-mgmt@gw-admin.<domain.tld>

    ssh oag-mgmt@gw-admin.mysite.myco.com

  2. Login using the default or a previously set password.
  3. Go to the log monitoring page:
    1. Enter 4 - Monitoring.
    2. Enter 2 - Enable Debug.
    3. Enter 1 - Monitor logs.

    This shows a running display of all log messages. Return to the console and exercise the application being examined.

    See Monitor for a complete list of all monitoring and related commands.

  4. Enter [ctrl][c] to exit the log display.
  5. Enter 2 - Disable debug.

The debug logging level rapidly generates log messages. Always disable debug logging when you've finished examining logging.

Failure to disable debug logging could result in rapid log file growth and out of disk errors.
