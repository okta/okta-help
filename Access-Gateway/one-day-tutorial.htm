Okta one-day admin

This guide describes concepts, how to deploy and configure , and how to perform common administrative tasks.

Administrators following this guide should have a highly available cluster when complete. Developers can skip the sections that cover high availability focus on the sections that describe how to create a complete development environment.

To accomplish these tasks, you need a basic knowledge of networks, firewalls, and high availability.

This guide covers the following tasks:

  1. Deploy .
  2. Integrate with your Okta org as an IdP.
  3. Integrate an application.
  4. Configure an High Availability cluster.
