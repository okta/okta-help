Open metrics ports

communicates over port 8889 using TCP/HTTP.

During this task you configure your firewall to allow communication on port 8889.

Detailed firewall configuration is beyond the scope of this document. Consult the appropriate documentation for information on configuring your firewall. This page is meant as a general guide.

To open port 8889 on your firewall:

  1. Determine the IP address of all instances of being monitored. See Determine [%=okta-feature-names.Access Gateway%] IP address.
  2. For each address in the list:
    1. Add require configuration to allow traffic between monitoring clients, typically Prometheus, and cluster instances.
    2. Test traffic as required.
  3. Save all firewall changes.
