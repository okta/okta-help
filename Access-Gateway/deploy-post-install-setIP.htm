Set instance IP address

The following procedure details how to specify the IP address of the Okta virtual appliance.

  2. Enter 1 - Network.
  3. Enter 1 - Manage network interfaces.
  4. Enter 1 - eth0.
  5. Enter a - Assign new static IP address.
  6. Enter required IP address details:
    FieldExample
    IP Address192.168.100.1
    Netmask255.255.255.0
    Gateway10.0.0.1
  7. Enter a to abort or c to continue with change.
  8. Enter x to return to the Manage network interfaces menu.

Depending on how the current IP address of is defined you may need to enter the static IP address into either your local (client machine specific) /etc/hosts file or DNS.
