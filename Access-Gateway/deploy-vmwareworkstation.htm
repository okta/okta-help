Import the Open Virtual Appliance file into Workstation

This topic describes how to import the Open Virtual Appliance (OVA) file into Workstation.

  1. Open the Workstation Player.
  2. Click Open a Virtual Machine.
  3. Go to the folder that contains the Okta-.ova file.
  4. Select the file and then click Open.
  5. Enter a name for the virtual machine, such as Okta-AccessGateway.
  6. Click Import.
  7. imports the virtual machine.
  8. After the import has finished, click Play virtual machine to start the virtual machine.
