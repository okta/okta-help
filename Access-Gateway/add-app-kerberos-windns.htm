Add to Windows DNS

Windows is the DNS provider for . You can use Windows DNS Manager to perform common administrative tasks.

To add a Windows DNS entry for an instance:

  1. Sign in to your Windows server.
  2. Start the DNS Manager application.
  3. Select your domain (for example, idaasgateway.net).

  4. Click Action > New Host (A or AAAA).
  5. Enter appropriate values for name (FQDN or fully qualified domain name) and IP address. For example:
    • Name:gw-iis
    • FQDN:gw-iis.access-gateway.tld
    • IP Address: IP Address of Access Gateway instance.
  6. Click Add Host.
  7. Exit Windows DNS manager.
