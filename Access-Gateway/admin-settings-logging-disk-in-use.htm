Examine disk in use

To set the logging level:

  1. Navigate to your instance.
  2. Select the Logs tab.
  3. Select the Log Storage pane.
  4. Examine disk usage which should resemble:

 

Application verbosity

Application verbosity displays the number of applications in either debug and info mode.
Clicking View opens Application view.

Note that the Admin console application is represented in the application verbosity counts, as info, but not displayed in the application.
