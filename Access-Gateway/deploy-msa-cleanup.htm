Cleanup

During this task you will removed temporary resources.

Always ensure that all cluster members are up and functioning as expected before removing temporary resources.
Consider retaining the temporary (primary) disk until all cluster members are tested and proven ready for use.

Steps

  • Remove temporary VM
  • Remove temporary disk

Remove Temporary VM

  1. Sign in or return to the Microsoft Azure Portal.
  2. From the menu, select All Services.
  3. Search for Virtual Machines.. Alternatively, you can also select Virtual Machines from the favorites list.
  4. Click the name of the VM used to house the temporary disk.
  5. Click Stop. Wait for the VM to terminate.
  6. Click Delete.

Remove Temporary Disk

  1. In the bread crumbs menu, click Home.
  2. Click Disks.
  3. Click the name of the disk associated with the temporary VM.
  4. Click Delete.
  5. Repeat for the second disk.
