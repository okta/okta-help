Amazon Web Services deployment tasks

To deploy to Amazon Web Services (AWS):

Task(s)

Description
Download the latest OVA image Download current version of the OVA.
Install and configure AWS command line support Many of the tasks required for AWS can be done at the command line.
Determine your AWS region AWS regions represent geographical locations where virtual machines are housed.
Upload OVA:
  • Create AWS S3 Bucket
  • Upload an OVA to an S3 Bucket
 Uploading the into AWS involves:
  • Creating an S3 bucket.
  • Uploading the OVA to the bucket.
Convert OVA to AWS AMI:
  • Create and associate AWS roles
  • Import an OVA to AWS
 AWS requires AMI rather then OVA files. To convert an OVA to an AMI:
  • Create required roles and policy.
  • Grant role to bucket.
  • VMImport OVA from bucket.
Perform AWS Specific post installation tasks:
  • Launch AWS instance
  • Obtain AWS Elastic IP Address
 Most virtualization environments require some post installation tasks. For AWS this includes:
  • Launch instance.
  • Obtain and AWS Elastic IP Address.
