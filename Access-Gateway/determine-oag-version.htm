Understand version information

The following table describes the version information fields and values:

Field Value Comments
YYYY Four-digit year of the release. For example, 2020.  
MM Two-digit month of release. For example 10.  
# Release within month. For example, 1.  

Build

Internal build number.

Not present in UI.

OS:

Operating system prefix.

Not present in versions earlier than 2020.10.2.
Major The major version of the underlying OS used by the appliance. Not present in versions earlier than 2020.10.2.

Minor

The minor version of the underlying OS used by the appliance.

Not present in versions earlier than 2020.10.2.
