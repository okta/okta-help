Application settings

Application settings are the core elements of an application definition.

When configuring applications in , the Settings pane contains the core application configuration. It's divided into three sub-tabs or panes:

  • Essentials: Essential application configuration required by all applications, such as label, public domain, and similar fields. See About Application Essentials.
  • Advanced: Advanced settings, such as sessions duration, session time-outs, certificates, and so on. See About Application Advanced Settings.
  • Behaviors: Application behaviors, customizable to perform specialized operations on application events, such as sign out, error, and others. See About Application Behaviors.
