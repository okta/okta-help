Configure SharePoint to work with a reverse proxy

During this task you will configure SharePoint to work with a acting as a reverse proxy.

Description Example
SharePoint internal FQDN sharepoint-int.atko.biz
SharePoint external FQDN sharepoint-ext.atko.biz
  1. Connect to the SharePoint Central Administration service and login as SharePoint Admin.
  2. Select Configure alternate access mappings.
  3. Specify:
    • Public domain - sharepoint-ext-atko.biz
    • Internal URLs mapped to Default zone:
      http://sharepoint-int
      http://sharepoint-int.atko.biz
      https://sharepoint-int.atko.biz
    • Public URL mapped to Default zone:
      https://sharepoint-ext.atko.biz
  4. Save all changes and exit.
