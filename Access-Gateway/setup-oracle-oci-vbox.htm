Install Virtual Box Extension Pack

Overview

The purpose of this page is to describe the process of installing the Oracle Virtual Box Extension Pack which includes the ability to prepare and export an OVA image to Oracle Cloud Infrastructure.

Prerequisites

  • You have installed Oracle Virtual Box v6.0 or later.

What’s covered in this guide

  1. Download and install Oracle Virtual Box Extensions

Download and install Oracle Virtual Box Extensions

To download and installed the Oracle Virtual Box extensions.

  1. Open a browser to the Virtual Box Downloads page.
  2. Click the All supported platforms link to download the extensions.
  3. Start Virtual Box.
  4. Select Preferences.
  5. Click Extensions.
  6. Click the plus sign (+).
  7. Navigate to the directory where the extension pack was downloaded and select it, then click Open.
  8. Click Install.
  9. Follow the directions to complete the install of the extension pack.
  10. Restart Virtual Box.
