Before you begin

Ensure that:

  • The protected Oracle BI Enterprise Edition resource trusts incoming messages and supports Single Sign-On (SSO) (user identity and SSO cookie in header).
  • You're using a supported external app. supports the following versions of Oracle Business Enterprise Edition:
  • Appropriate DNS entries for the legacy application and its external URL exist.
  • Appropriate DNS entries for the protected application and its external URL exist. For example:
    ValueDescription
    https://obiee.externalexample.comExternal or customer facing application URL.
    https://obiee.internalexample.com

    Internal or protected application URL of an Oracle BI Enterprise Edition application.
