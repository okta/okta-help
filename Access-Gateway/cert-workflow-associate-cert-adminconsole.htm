Associate a default host certificate using the

You can associate a default host certificate and configure the default virtual hostname.

Before you associate a default host certificate, upload it to . See Upload certificates.

Associate a default host certificate

  1. Open the in a browser.
  2. Click SettingsCertificates.
  3. Enter a value in the Search field to narrow the list of certificates. You can also click Generate self-signed certificate to generate a new certificate.
  4. Select the default host certificate that you want to associate with .
  5. Click Update Certificate.
