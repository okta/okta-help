Create the application in

  1. Select the Header Based option from the application menu. Click Create.

  2. In the Essentials pane, specify the following:

    Field Value
    Label A name for the application.
    Public Domain A fully qualified host name, for example ext-header.example.com
    Protected Web Resource

    The URL of the protected resource. For example, int-header.example.com[:port].

    When you configure a protected web resource, you can optionally set up load balancing for the resource. See Configure load balancing.
    Group Enter the group containing the users who should have access to the application.
    Description Optional. An appropriate description for your application.
  3. Click Next. The Attributes pane appears.
