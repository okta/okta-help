Add a local Auth Module

A Local Auth Module is a simplified auth module which uses local accounts and is maintained within .

To add a local Auth Module:

  1. Add an Auth Module as described in Add an Auth Module and select Local.
    After you select Local , the Add New Auth Module page opens, configured for local authentication.
  2. Enter the following details:
    FieldDescriptionExample
    NameName used to identify the Auth ModuleLocal AM

    Accounts

    A set of accounts for authentication

    Use the(+) and Delete icons to add and delete accounts. Use the Edit icon to edit an existing account.

    Attributes

    Set of attributes provided to as part of the IDP assertion

    Email, cn, name, uid

    Description

    Optional. Enter description

    Local Auth Module

  3. Click the Not Validated ()button when complete. Okta validates the connection.
    If the validation is successful, the button changes to Valid ().
  4. Click Okay.
