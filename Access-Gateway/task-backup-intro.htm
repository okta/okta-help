Backup and restore

Backing up your appliances and configurations is a proactive safeguard against data loss, down-time, and other unwanted negative impacts to your business.

A best practice is to use both of the following backup methods:

  • Virtual environment-based backups: Create image-based backups using the built-in functionality found in hypervisor products such as VMware and VirtualBox. The benefit of image-based backups is that they back up the entire appliance and virtual machine.
  • -based backup and restore: This creates a backup of the current system configuration, including the Okta tenant (IdP), application, datastore, and logging.
