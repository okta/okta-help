Create a sample policy application

During this task we will create the sample policy application.

  1. Select the Access Gateway Sample Policy option from the left column menu, and click Create.

    The New Protected Application wizard will start and display the Setting tab for the application being added.

  2. In the Essentials pane specify the following:

    Field Value
    Label A name for the application. The default is set to Sample Policy App.
    Public Domain A fully qualified host name, such as policy.<your domain tld>
    Group Enter the group containing the users who should have access to the application.
    Description Optional. An appropriate description for your application.
  3. Move on to the next task or click Done.
