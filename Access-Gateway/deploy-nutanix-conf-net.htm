Configure virtual network

During this task we will configure a virtual network for use with a VM<.

  1. Open a browser to and sign in to the Nutanix Prism Central console.
  2. Click the Settings gear to open the setting panel.
  3. In the General section, select Network Settings.
    The Image Configuration pane will open.
  4. Click Storage container button.
    The Create Storage Container dialog will open.
  5. Enter:
    NameA unique name for the storage container
    Storage poolSelect a storage pool
  6. Click Save.

The storage container will then be created and made available to all hosts in the cluster.
