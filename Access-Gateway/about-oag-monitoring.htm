Monitoring

Monitoring is an important aspect of the day to day life of an installation.

provides several methods for monitoring, including:

  • Status REST Monitoring- A simple rest endpoint, that when called returns either a 200 success, a 400 error or a timeout.
  • Basic REST Monitoring - A simple rest end point that returns simple statistics from the requested node. Statistics include active connections, server connects and more.
  • Metrics monitoring - A more comprehensive set of human readable metrics, returned in the open metrics format, returning measures of Access Gateway instance status.
