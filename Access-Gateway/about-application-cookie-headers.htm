About application cookie headers fields

Application cookie fields are stored in the application requests Cookie header field as a semi-colon-separated list. They're created and populated by for application requests and are communicated to back-end protected web application, by only.
See Application attributes for more information about application attributes as well as all attribute sources and targets.
