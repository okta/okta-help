Troubleshoot a WebSocket integration

WebSocket connection closed error

When accessing my application I'm receiving one of WebSocket connection closed or WebSocket connection closed, Code: 1006.
Likely cause: the URI associated with the web socket does not have an appropriate policy.
Try the following solution:

  • Confirm that a policy exists for the given URI and that it contains the correct advanced directives.
    See Add required access policy for details on adding the required policy.
    Rerun the test.
