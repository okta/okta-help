Concepts & Architecture

Preparing for Identity Provider failover consists of:

  • Defining a secondary authentication module
  • Defining a redundant Identity Provider, which uses the secondary authentication module
  • Defining redundant applications, which replicate existing applications but use the secondary Identity Provider
