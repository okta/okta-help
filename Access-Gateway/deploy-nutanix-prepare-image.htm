Prepare Image for deployment to Nutanix

Nutanix requires the image disk be uploaded individually. In this task we unpackage the OVA file to gain access to the VMDK disk file.

  1. Open a terminal window and change directory to where the OVA was downloaded.
  2. Create a subdirectory to contain the OVA components.
    For example on Mac OSX, Linux or Windows:mkdir new-folderWhere new-folder represents some folder name.
  3. Change directory to the new folder and unpackage the OVA contents.
    On Mac OSX, or Linux:cd new-folder
    tar -xvf ../oag-2021.5.xyz.    On Windows:cd new-folder

    tar -xvf ..\oag-2021.5.xyz.ova
