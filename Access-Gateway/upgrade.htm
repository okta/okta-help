Upgrade

supports two upgrade types:

  • Latest released only, available in and earlier, and supports upgrading to the latest version of only.
  • Selective upgrade, available in and later, and supports upgrading to a later version of Access Gateway (if existing), or the latest version.

Reboot the node after you complete the upgrade. See the Reboot section of System menu for instructions.
