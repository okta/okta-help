Install the Okta Browser Plugin

After creating your Okta Personal account, install the Okta Browser Plugin for Chrome in a desktop browser. The Okta Browser Plugin enables you to use SSO for applications that require you to enter your credentials.

How it works

When you access an app from your dashboard, a new browser tab opens to the app’s URL. The Okta Browser Plugin uses an encrypted SSL connection to obtain your authentication from Okta Personal and then applies it to the app’s URL. The plugin doesn’t store credentials after authentication is complete. See Sign in from the plugin.

Install the Okta Browser Plugin

  1. In a desktop browser, go to your Okta Personal dashboard.

  2. In the Okta plugin banner, click Install.

  3. On the Okta Browser Plugin page of the Chrome web store, click Add to Chrome.

  4. Verify the permissions, and then click Add Extension.

To view the Okta Browser Plugin, click the puzzle icon in your taskbar. If you don't see it, click Manage Extensions and enable the plugin by toggling the switch.
