Set up AutoFill for Okta Personal

The AutoFill feature on your iPhone lets your Okta Personal account enter app credentials on your behalf.

Before you begin

Create an account

Set up MFA

Install the Okta Browser Plugin

Install Okta Personal for iOS

Add apps to Okta Personal

Start this task

  1. Open the Okta Personal app on your iPhone.

  2. Tap Set up AutoFill.

  3. Review the instructions, and then tap Go to passwords.

  4. Tap AutoFill Passwords, and then toggle it on.

  5. Allow filling from Okta Personal only.

  6. Return to your Okta Personal app.
