Save apps to Okta Personal

The Okta Browser Plugin prompts you to save your password whenever you enter your credentials in a desktop app.

Before you begin

Create an account

Set up MFA

Install the Okta Browser Plugin

Start this task

  1. In a desktop browser, enter the URL of the app you want to add.

  2. Sign in to the app using your current username and password.

  3. When the Okta Browser Plugin prompt appears, verify that your credentials are correct, and then click Save.
