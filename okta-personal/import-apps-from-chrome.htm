Import apps from Chrome

If you save your passwords in Chrome, you can import them into Okta Personal. Moving those passwords to Okta Personal gives you a single location to organize and secure your apps.

How it works

Importing apps from Chrome is a two-step process. First, export your Chrome passwords as a CSV file, and then import that file into Okta Personal. The Chrome apps then appear on your Okta Personal dashboard.

Be sure to delete the CSV file after you complete the import. It displays all of your saved app passwords in plain text.

Before you begin

Create an account

Set up MFA

Install the Okta Browser Plugin

Import apps

  1. Export your saved passwords from Chrome. See Show, edit, delete, or export saved passwords.

  2. In a desktop browser, go to your Okta Personal dashboard.

  3. Open the menu next to your name, and then select Import apps.

  4. On the source page, choose Chrome.

  5. Click Choose File.

  6. Select your Chrome Passwords CSV file.

  7. Click Import.

  8. After your apps are visible on the Okta Personal dashboard, delete the Chrome Passwords CSV file.

If you want to turn off the save password prompt in Chrome, see Start or stop saving passwords.
