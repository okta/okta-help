Set up MFA

You need a knowledge factor and a possession factor to secure your Okta Personal account. You set up your password during registration, so you should set up SMS the first time that you sign in. You can also add your fingerprint as an optional biometric factor.

If you choose to add a biometric factor, you need to set it up in your desktop browser. The Face ID feature on your iPhone doesn’t replace SMS or fingerprint.

Before you begin

Create an account

Set up SMS

  1. When you're prompted for fingerprint or face recognition after creating an account, click Try another method.
  2. In the Other Methods list, select SMS.
  3. On the Verify your account screen, enter your phone number to receive a 5-digit code through SMS.
  4. Enter your code, and then click Continue.

Set up fingerprint recognition

Fingerprint recognition is an optional factor, but if you set it up, you'll be prompted for it first when you sign in. SMS is then considered the backup factor.

  1. In a desktop browser, go to personal.okta.com.
  2. Enter your email address, and then click Continue.
  3. Click Continue to set up fingerprint or face recognition.
  4. At the Google Chrome prompt, tap your finger on the Touch ID pad.
  5. On the Okta Personal sign-in page, click Continue.
