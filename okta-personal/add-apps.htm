Add apps to Okta Personal

There are two ways to add apps to Okta Personal: from your dashboard or through the Okta Browser Plugin. Both methods require a desktop browser.

Before you begin

Create an account

Set up MFA

Install the Okta Browser Plugin

Add apps from the dashboard

  1. In a desktop browser, go to your Okta Personal dashboard, and then click Add App.

  2. Enter the app name in the search box.

    • If it’s already in the Okta catalog, select it from the list and then click Next.

    • If it’s not in the catalog, select Add “<name>”.

  3. Enter your sign-in credentials.

    • Enter the name that you want displayed on the dashboard. This can be different from the official app name.

    • Enter the login URL. This isn’t editable if the app exists in the Okta catalog.

    • Enter the username that you use to access the app.

    • Enter the password that you use to access the app.

  4. Click Save.

Add apps from the plugin

  1. In a desktop browser, click the extensions icon in your task bar.

  2. Select Okta Browser Plugin from the list of extensions.

  3. Click the plus sign to add an app.

  4. When your dashboard opens in a new browser window, enter the app name in the search box.

    • If it’s already in the Okta catalog, select it from the list and then click Next.

    • If it’s not in the catalog, select Add “<name>”.

  5. Enter your sign-in credentials.

    • Enter the name that you want displayed on the dashboard. This can be different from the official app name.

    • Enter the login URL. This isn’t editable if the app exists in the Okta catalog.

    • Enter the username that you use to access the app.

    • Enter the password that you use to access the app.

  6. Click Save.
