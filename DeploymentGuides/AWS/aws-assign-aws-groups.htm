Assign AWS management groups to the AWS app

After mapping Amazon Web Services (AWS) role groups to entitlements, you need to assign all AWS management groups to the AWS app in . If you have provisioning enabled and you can't assign a management group, disable provisioning and then assign all AWS management groups to the AWS app in .

  2. Enter AWS in the Search field.
  3. Click AWS Account Federation, then select the Assignments tab.

  4. Click Assign Assign to Groups .

  5. For each AWS management group, click Assign.
  6. Click Done.

Users are automatically assigned to AWS and can access the entitlements you defined.
