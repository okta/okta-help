Data collected by

collects the following data from Android and iOS devices for security and identification purposes:

Data Type Description
Device name Always collected
Model Always collected
Manufacturer Always collected. For example, Apple or Google
Platform Always collected. For example, iOS or Android
UUID (universally unique identifier) Always collected and stored in the system keychain. This ID is generated during the app installation to identify an app instance. It doesn’t identify the device.
OS version Always collected
Email Always collected and securely stored in system keychain
TOTP secret Always collected and securely stored in secure enclave
Encryption keys Always collected
Error, warning and info file logging Always collected. When users report an issue, the logs are attached to the bug report.
Diagnostic and crash data Always collected
Pendo collection Anonymized and collected for a random sample set of users
Instabug collection Anonymized and collected for a random sample set of users. The random sample set includes users who choose to provide feedback through the app (still anonymized).
