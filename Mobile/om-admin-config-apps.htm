Hide apps from Okta Mobile

As an administrator, you can configure what apps end users see in on their devices. You can hide applications that aren't ready for mobile use.

Start this task

  1. In the Admin Console, go to Applications.
  2. Click the app that shouldn't be available in .
  3. On the General tab, go to the App Settings section, and click Edit.
  4. For Application visibility, select Do not display application icon in the Okta Mobile app.
  5. Click Save.
