Edit an event hook filter

For you to edit an event hook filter, the event hook must be active.

  2. In the list of event hooks, click an event hook name.
  3. Click the Filters tab.
  4. Click Edit.
  5. Select the Apply filter checkbox to apply a filter to an event, or clear the Apply filter checkbox to delete a filter.
  6. For existing filters, you can use the Okta Expression Language to modify the filter. See Okta Expression Language and event hooks.
  7. Click Save.
