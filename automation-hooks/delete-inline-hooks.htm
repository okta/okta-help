Delete an inline hook

Delete an inline hook. By default, active inline hooks are listed first, but you can sort the table by type or name.

  1. In the , go to Workflow Inline Hooks .
  2. Locate the inline hook that you want to delete, and then click its Actions menu.
  3. Select Deactivate.
  4. Click the hook's Actions menu again, and then select Delete.
