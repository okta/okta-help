Verify an event hook

Verify that you are the owner and control the endpoint specified in the event hook. This is a one-time verification process. If an endpoint URL is used in a Workflows flow, it appears as verified.

  2. In the list of event hooks, click Actions, and then select Verify for the event hook you want to verify.
  3. Click Verify in the Verify Endpoint Ownership dialog.

    If the verification fails, an error message with details appears.

  4. Optional. Click Do this later to return to the Event Hooks page.
