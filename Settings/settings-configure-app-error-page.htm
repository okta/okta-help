Configure a custom application error page

You can redirect end users to a default Okta URL or to a custom URL if they try to access an app that isn't assigned to them. Unassigned users are more likely to try to access apps if you embed app links in your portal or other sites outside of Okta.

Note that app-specific URLs override this org setting. See Redirect unassigned users to a custom error page.

  3. Select Use a custom error page and enter the redirect URL.
