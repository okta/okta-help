Add translations

  1. Select a brand.
  2. Click Emails. 
  3. Choose an email template that you've customized. The default language version appears in the Custom Email table.
  4. Click Add Translation, and then select a language from the drop-down box. If the Add Translation button isn't available, this template hasn't been customized. See Edit a default email template.
  5. Make your translated edits, and then click Add Translation.
  6. Repeat steps 5 and 6 for more languages.

To delete all custom translations and revert to the Okta template, click Reset to Default.
