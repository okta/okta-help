Set a theme for your org

Branding begins with a theme. A theme is a collection of components you can apply to Okta templates, pages, and applications. Themes consist of a logo, a favicon, a primary color, a secondary color, and a background image.

Before you begin

You must be a super admin or an org admin, or have a role with customizations permissions.

Start this task

  2. Select a brand.

  3. Set the following components in Theme.

    • Primary color: Enter a 3-digit or 6-digit hex code. The primary color is used for buttons on email templates and Okta-hosted pages, and as a navigation menu color on the End-User Dashboard.
    • Secondary color: Enter a 3-digit or 6-digit hex code. The secondary color is used as background for Okta-hosted pages and email templates.
    • Logo: Click the pencil icon to upload your org logo. Logos are used on email templates and Okta-hosted pages. For best results, use a PNG file in landscape with a transparent background.
    • Favicon: Click the pencil icon to upload a favicon. Favicons are square icons used in browser tabs for Okta-hosted pages.
    • Background: Click the pencil icon to upload a background image. The background can be used on the Okta-hosted sign-in and error pages.
  4. Save your settings.
  5. Optional. Use the dropdown menu to preview your theme on other pages.
  6. Optional. Click Customize for each page preview. Refer to the following topics for information on each customization:

    • Customize your sign-in page

    • Customize an error page

    • Manage dashboard tabs for end users

    • Customize an email template
