Branding variables

After you create a theme for your org, you can use branding variables in your custom email templates.

Variable

Type
brand.theme.logo String (URL)
brand.theme.backgroundImage String (URL)
brand.theme.primaryColor String (hex code)
brand.theme.primaryColorContrast String (hex code)
brand.theme.secondaryColor String (hex code)
brand.theme.secondaryColorContrast String (hex code)
