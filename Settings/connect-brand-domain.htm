Connect a custom URL domain with a brand

Connect your custom URL domain with a brand so you can send traffic to a sign-in page. You can do this in Customizations > Domain or in Customizations > Branding.

About admin roles for this task

You must be a super admin or org admin to connect custom URLs with brands.

Start this task

From the Branding page

  2. Find the brand you want to connect to a custom URL, and then click Edit.
  3. Select a custom URL domain.
  4. Click Save.

From the Domain page

  2. Find the custom domain you want to connect, and then click Edit.
  3. Create or select a brand.
  4. Click Save.
