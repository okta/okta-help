Okta Active Directory Password Sync Agent version history

This page lists current and past versions of the Okta Active Directory Password Sync Agent. This page is updated whenever a new version of the agent is released.

To download the latest agent, go to the Admin Dashboard, select Settings Downloads and scroll to the agent you want. Click Download and run the installer.
