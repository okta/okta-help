Okta ADFS Plugin version history

This page lists current and past versions of the Okta Active Directory Federation Services (ADFS) Plugin. This page is updated whenever a new version of the agent is released to General Availability (GA) or Early Access (EA). 

For installation information, see Okta MFA for Active Directory Federation Services.

Current GA Version

 1.7.13

These agents support the following Microsoft Windows Server versions:

  • Windows Server 2022
  • Windows Server 2019
  • Windows Server 2016
  • Windows Server 2012 R2

These agents support the following ADFS versions:

  • ADFS 5.0
  • ADFS 3.0
  • ADFS 4.0
