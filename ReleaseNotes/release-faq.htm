How often does Okta deploy releases?

A monthly release usually includes these items:

  • Generally Available (GA) and Early Access (EA) features
  • Feature enhancements
  • Fixes
  • : New app integrations, updates, and fixes

A weekly update is deployed after a monthly release and includes these items:

  • Bug fixes and stability improvements
  • : New app integrations, updates, and fixes

What's the naming convention for monthly releases and weekly updates?

The version number follows the <year>.<month>.<release-type> convention.

For example:

  • 2023.09.0: Monthly release for September 2023
  • 2023.09.2: Weekly update for September 2023

What's the difference between Preview and Production releases?

A Preview release is deployed to preview orgs only. Similarly, a Production release is only deployed to production orgs.

Most features and enhancements are first deployed in a Preview environment. The Preview release moves to the Production environment after a month.

What are Early Access features and Generally Available features?

  • Early Access (EA): Opt-in features that super admins can enable or request Okta Support to enable for them. Super admins can enable or disable some EA features using the steps listed in Manage Early Access and Beta features.

  • Generally Available (GA): Features are made available to all orgs automatically based on your eligibility.
