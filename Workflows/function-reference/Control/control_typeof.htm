Type Of

This function card returns a type for an input value.

Input

Field Definition Type Required
value A string, numerical, Boolean or other value for which a type is returned. String TRUE

Output

Field Definition Type
type Type for the input value. Possible outputs:

  • Text

  • Number

  • True/False

  • Date & Time

  • Object

  • File

  • Flow

  • List

 String
collection Indicates whether the input value belongs to a list of that type. Boolean
