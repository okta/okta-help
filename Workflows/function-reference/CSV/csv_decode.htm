Decode

Decodes a CSV string into a list of objects.

Input Fields

  • csv: the CSV list you want to decode

Output Fields

  • list: the resulting list of objects from the CSV string

Example

The following is a CSV list containing a set of three records:

Name,Type,Active

Jane,Contractor,true

Bob,customer,false

Michael,employee,true

The Decode function card will produce three objects:

{ "Name": "Jane", "Type": "Contractor", "Active": "true" } { "Name": "Bob", "Type": "customer", "Active": "false" } { "Name": "Michael", "Type": "employee", "Active": "true" }
