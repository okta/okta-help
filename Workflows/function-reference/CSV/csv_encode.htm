Encode

Encodes a list of objects into a CSV string.

Input Fields

  • list: the list of objects you want to encode into a CSV string

Click the empty input field to add a second or subsequent JSON list of objects that you also want to split.

Output Fields

  • csv: the result of the CSV encoding

Example

This is a list of three JSON objects:

{ “user1”:  { "name":       “Jane”, “type”:      contractor, “active”:    true } } { “user2”:  { "name":       “Bob”, “type”:      customer, “active”:    false } } { “user3”:  { "name":       “Michael”, “type”:      employee, “active”:    true } }

The Encode function card will produce this CSV string:

Name,Type,Active

Jane,Contractor,true

Bob,customer,false

Michael,employee,true
