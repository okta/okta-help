Wait Until

This function card pauses the flow and waits until a specific date and time before resuming the flow once again.

Input

Field Definition Type Required
date & time

Date and time at which point the flow is resumed. This field accepts a value in ISO date format using UTC Z notation. For example, 2016-09-07T16:55:25.670Z.

 Date & Time TRUE

Output

This card has no output fields.
