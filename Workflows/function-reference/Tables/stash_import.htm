Import Data from CSV

You can import data into a table from a CSV file.

  1. Define table columns

  2. Click the Import button to bring up the Import Data dialog

  3. Drag/drop a CSV file into the dialog or click Choose CSV File.

  4. For each CSV field you want to import, choose which table column to which you want to map that field. If the names match, that table column will be the default suggestion.

  5. Click Import.
