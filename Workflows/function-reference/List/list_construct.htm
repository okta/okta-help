Construct

Construct a new list from a set of values.

Input

Field Definition Type Required
item User created fields.

To use this card, you must create correctly-named inputs and set your types correctly:

  1. Select the type for the list.

  2. Select the type for the output accordingly.

  3. For each item, set the type of each input.

  4. Populate the input values for each item in the list.
