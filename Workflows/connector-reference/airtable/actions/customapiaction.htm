Custom API Action

Make a custom, authenticated HTTP call to the API.

This action is unlike other cards. Refer to API.

Options

Input

Field Definition Type Required
Request
Relative URL URL address on the web server to which you are attempting to interact. Specify the relative URL as /{insert_remaining_URL}. You can specify query parameters in the relative URL using "?", or specify the query parameters as an object key pair in the Query input.

For example, for Airtable API endpoint https://api.airtable.com/v0/appu53qW5mpKdJAW2/Design%20project, the relative URL is /v0/appu53qW5mpKdJAW2/Design%20projects.

 String TRUE
Query JSON object that determines the action taken at the URL. Example: {"name":"something-urgent"} Object or string FALSE
Headers JSON object that determines the content type of the request. This will usually be similar to the following: Example: {"Content-type":"application/json"} Object FALSE
Body Data for your selected HTTP request. Object FALSE
