The new authorization method for connections from uses personal access tokens. See the developer guide on Personal access tokens.

The following procedure explains how to create the connection using either method.

Create a connection from an action card

  1. Click New Connection.

  2. Enter a Connection Nickname. Names are useful if you plan to create multiple connections to share with your team.

  3. Select the authorization method from the Authorize By dropdown menu. You can select either API Key or Personal Access Token.

  4. Provide either the API Key or the Personal Access Token in the text field.

  5. Click Create.
