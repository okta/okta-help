Authorize your Google Docs account

When you add a  card to a for the first time, you need to set up a configuration for that connector. Setting up a configuration allows you to connect to your Google Docs account, save the information, and reuse that configuration the next time you build an with the connector. You don't have to sign in every time you want to build an with this connector. After you set up one configuration, you can choose to use that configuration again, or set up a new one.
