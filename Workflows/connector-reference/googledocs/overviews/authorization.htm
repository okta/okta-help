Authorization

When you add a Google Docs card to a flow for the first time, you're be prompted to configure the connection. This enables you to connect your Google Docs account, save your account information, and reuse the connection for future Google Docs flows. See Guidance for Google Docs connector.

You can create multiple connections and manage them from your Connections page.

Create a connection from an action or event card:

  1. Click New Connection.
  2. Enter a string value in the Connection Nickname field. Make this nickname unique so you can authorize multiple Google Docs accounts.
  3. Click Create.
