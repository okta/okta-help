Search Channels

Search for channels.

Options

Field Definition Type Required
Result Set Choose one of the following options to display your search results:
  • First Matching Record - returns the first matching teams.
  • First 200 Matching Records - returns the first 200 matching teams.
Dropdown TRUE
Search By Choose whether to return teams that are an Exact Match of the search or Starts With the search expression. Dropdown TRUE

Input

Field Definition Type Required
Team      
ID Unique identifier of the team. String TRUE
Search
Display Name Display name of the channel. Maximum 50 characters.

Appears when Exact Match or Starts With are selected from Search By.

 String FALSE
Description Description for the channel.

Appears when Exact Match or Starts With are selected from Search By.

 String FALSE
Membership Type Type of the channel. Possible values are:

  • Standard: Channel inherits the list of all members of the parent team.

  • Private: Channel can have members that are a subset of all the members on the parent team. Limit of 30 private channels per team.

Appears when Exact Match is selected from Search By.

 Dropdown FALSE

Output

Field Definition Type
Channels    
ID Unique identifier of the channel. String
Display Name Display name of the channel. The default setting is message. String
Membership Type Specifies the type of the channel. String
Description Description for the channel. String
Email Email address for sending messages to the channel. String
Is Favorite By Default? Specifies whether the channel should automatically be marked favorite for all members of the team. Boolean
Created Date Time Date and time that the channel was created. Date and Time
