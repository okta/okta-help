Authorization

When you first add a card to a flow, you'll need to create a connection. This will allow you to connect to your account, save your account information, and reuse that connection next time you build a flow with the connector. You won't have to sign in every time you want to build a flow with this connector. Instead, use the connection that you've already set up. You can create and save multiple connections and share them with your teammates.

To set up a new connection in :

  1. Enter an Account Nickname. This should be unique. This is a helpful feature to enable you to differentiate between multiple connections.

  2. Click Create to launch a pop-up window where you will sign into your Account. This will also save your connection.
