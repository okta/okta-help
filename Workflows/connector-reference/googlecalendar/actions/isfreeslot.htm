Is Free Slot

Check a specific calendar to see if a time slot has an existing event or is available.

Options

Field Definition Type Required
Calendar A dropdown list of your available calendars on which to verify a time slot. Dropdown TRUE

Input

Field Definition Type Required
Min Time Date and time from which you want to check if your calendar is available. Input needs to be in the format YYYY-MM-DDTHH:MM:SSZ. Date & Time TRUE
Max Time Date and time before which you want to check if your calendar is available. Input needs to be in the format YYYY-MM-DDTHH:MM:SSZ. Date & Time TRUE

Output

Field Definition Type
Is Free Returns true if your calendar has no event during the given time slot; false if your calendar has an existing event during that time slot. Boolean
