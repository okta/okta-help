Search Transfer Request

Search for a transfer request in your account using transfer ID.

Options

Field Definition Type Required
Result Set Define the card output format. Choose from:
  • First Transfer: Returns the first matching transfer request
  • All Transfers: Returns all matching transfer requests
 Dropdown TRUE

Input

Field Definition Type Required
Transfer Status Status of the transfer; options are Completed, Failed, or In Progress. Dropdown FALSE
Old Owner Email Email of the user whose data is being transferred. String FALSE
New Owner Email Email of the user to whom data is being transferred. String FALSE

Output

Field Definition Type
Transfer ID Unique ID of the transfer. String
Transfer Status Status of the transfer; can be Completed, Failed, or In Progress. String
New Owner Email Email of the user to whom data is being transferred. String
Old Owner Email Email of the user whose data is being transferred. String
Request Time Time when the transfer was requested. Date & Time
