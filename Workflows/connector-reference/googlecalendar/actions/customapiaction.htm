Custom API Action

Make a custom, authenticated HTTP call to the API.

Options

Input

Field Definition Type Required
Relative URL URL address on the web server to which you are attempting to interact. Specify the relative URL as /{version}/{insert_remaining_URL}. You can specify query parameters in the relative URL using "?", or specify the query parameters as an object key pair in the Query input.

For example, for the API endpoint:
https://www.googleapis.com/calendar/v3/users/{userId}/calendarList the relative URL is: /v3/users/{userId}/calendarList.

See Google Calendar API.

 String TRUE
Query Specify any additional query parameters that should be included in object format (key value pairs). Object FALSE
Headers Specify any headers required in addition to authorization or content-type (these are already handled by this connector). Object FALSE
Body Specify a request body in JSON format. Only available for POST, PUT, and PATCH requests. Object FALSE

Output

Examples

calendarList.list

calendarList.update

calendarList.list

Returns a list of the know calendars the provided user id.
See also https://developers.google.com/calendar/api/v3/reference/calendarList/list

Request

Field Value
Request type GET
Relative URL calendar/v3/users/{user id}/calendarList
Query parameter(s)

Optional maxResults: integer, count of results to return.

See calendarList/list API docs for a complete list of query parameters
Header fields Not required.
JSON Body Not required.

calendarList.update

Updates the properties of the calendars the provided user id and calendar id.
See also https://developers.google.com/calendar/api/v3/reference/calendarList/update

Request

Field Value
Request type PUT
Relative URL calendar/v3/users/{user id}/calendarList/{calendar id}
Query parameter(s)

Optional colorRgbFormat: boolean. Whether to use the foregroundColor and backgroundColor fields to write the calendar colors (RGB). Default: false.

See calendarList/update API docs for a complete list of query parameters
Header fields Not required.
JSON Body

JSon containing one or more identifier/value pairs representing properties to update.
See RequestBody in https://developers.google.com/calendar/api/v3/reference/calendarList/update for a complete list.

{"colorId": "10"}

Response

Field Value
Status code 200 on success
JSON Body A result similar to:

{ "kind": "calendar#calendarListEntry", "etag": "\"1...0\"", "id": "id provided", "summary": "Birthdays", "description": "Displays birthdays, anniversaries, and other event dates of people in Google Contacts.", "timeZone": "America/New_York", "colorId": "10", "backgroundColor": "#b3dc6c", "foregroundColor": "#000000", "accessRole": "reader", "defaultReminders": [] }
