Download Message Attachment

Downloads an email attachment using the attachment ID in .

  • An attachment must be at least 1 byte in size, but cannot be larger than 2 GB. See Workflows system limits.

  • Files are downloaded to the Workflows platform file system.

Options

Field Definition Type Required
Mail Determines the type of user mail:

  • My Mail: owned by the connected user.

  • User Mail: user shared or delegated with the connected user.

 Dropdown TRUE
