AWS enables you to run code without the need to provision or manage servers, and pay only for the compute time that you use. You can invoke and list AWS Lambda functions associated with your account.

Authorize your account

When you add a card to a flow for the first time, you'll be prompted to configure the connection. This will enable you to connect your AWS Lambda account, save your account information, and reuse the connection for future AWS Lambda flows. See Authorization.
