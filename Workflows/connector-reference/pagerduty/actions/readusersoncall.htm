Read Users On Call

List all of the users on call in a given schedule for a given time range.

Input

Field Definition Type Required
Schedule
Schedule ID Unique identifier of the schedule that will be read. String TRUE
Since Start of the date range for the search.

For example, 2015-11-10T08:00:00-05:00.

 Date & Time FALSE
Until End of the date range for the search.

For example, 2015-11-10T17:00:00-05:00.

 Date & Time FALSE
