List Team Members

List all members of a team and their profile details in your instance.

Input

Field Definition Type Required
Team ID Unique identifier of the team profile whose members you want to access. String TRUE

Output

Field Definition Type
Users (Object)
User ID Unique identifier for the user profile who is a member of specified team in your instance. String
Summary A short string that provides succinct, information about the user. In many cases, this will be identical to Name, though it is not intended to be an identifier. String
HTML URL User profile's unique URL in the web application. String
Role Specific role within a user's role type. Role can be admin, limited_user, observer, owner, read_only_user, restricted_access, read_only_limited_user, or user. String
