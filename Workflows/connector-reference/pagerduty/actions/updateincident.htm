Update Incident

Update an incident in your instance.

Input

Field Definition Type Required
Update By
Incident ID Unique identifier of the incident that will be updated. String TRUE
From
Email ID Email address of the user who will update the incident. String TRUE
Incident
Type Type of integration that can be used to trigger an incident. The type can be incident or incident_reference. String TRUE
Status The state of the incident: acknowledged or resolved. String FALSE

Due to a PagerDuty API limitation, if an incident is triggered as incident, it can't be changed to incident_reference. But an incident can be triggered through the API as either type.

Output

Field Definition Type
Status Code Number
