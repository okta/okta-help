Revoke token

Revoke your PagerDuty account's API key.

If you delete a connection with PagerDuty in Workflows, you will also need to delete the API key manually. The API key is not revoked automatically.

You will need a super user account to perform this task.

To delete your API key:

  1. With your PagerDuty account credentials, sign in to your PagerDuty domain.

  2. From the Configuration menu, click API Access.

  3. Click Remove next to the API Key that you want to delete.

  4. Click OK to confirm.
