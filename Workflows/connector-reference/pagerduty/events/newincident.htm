New Incident

Use this event to trigger a flow when an incident is triggered in your instance.

Do not use the New Incident Event Card and the Create Incident Action Card together in the same flow. By doing so, you create a circular reference.

It is recommended that you preserve the default polling value of Every 5 minutes for this card. If you increase the time value for polling intervals, you may run the risk of not returning all of the records that are processed in that period. The connector supports a maximum of 1,000 records between polls.

No options available.
