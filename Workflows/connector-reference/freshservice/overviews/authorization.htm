Authorization

Authorize this connector by creating a connection to your account. You can reuse this connection the next time that you build a flow with this connector.

You can create multiple connections and manage them from your Connections page.

Create a connection

Prerequisites

  • Okta Workflows admin credentials

  • account credentials, if creating a connection using credentials

  • API key, if creating a connection using a API key

Procedure

  1. There are two ways that you can create a connection:

    • Add a card to your flow and click New Connection.
    • Open the Connections tab in the Workflows platform. Click New Connection and then click .

  2. Enter a Connection Nickname. This is useful if you plan to create multiple connections to share with your team.

  3. Enter your Domain value. This value is the sub-domain portion of your domain. For example, if your domain is example99.freshservice.com, your Domain value is example99.

  4. Choose whether to authorize your connection using Credentials or API Key from the Authorization By dropdown list.

  5. Enter your username and password if you chose to authorize your connection using credentials. Otherwise, enter your API key. See Where do I find my API key?

  6. Click Create.

The connector is now configured and this connection is ready to be used with any available cards for the connector.
