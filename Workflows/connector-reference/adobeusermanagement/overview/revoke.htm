Revoke connection

Delete a connection from Okta Workflows.

To revoke a connection, delete the private key that was used to create a connection.

1. Go to https://developer.adobe.com/.

  1. Sign in with system administrator role permissions.

  2. Click Console.

  3. Go to the project that was created to make a connection.

2. Click the Service Account (JWT) section.

3. On the Service Account (JWT) section page, click the recycle bin next to the public key value that was generated during the project creation.
