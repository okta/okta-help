Table Row Added or Updated

Start the flow when a new row is added to a table.

Options

Field Definition Type Required
Workbook List of workbooks available to the account. Workbook is another word for your Excel files. Dropdown TRUE
Worksheet List of worksheets available in the selected workbook. Worksheet is the individual spreadsheet within them, displayed as tabs in Excel files. Dropdown TRUE
Table List of tables available in the selected worksheet. Table in Excel is a formatted series of worksheets rows and column which have additional function. Dropdown TRUE

Output

Field Definition Type
Table
Row Number Table row number Number
