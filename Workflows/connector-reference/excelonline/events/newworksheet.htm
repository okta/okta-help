New Worksheet

This card will start flow when a new worksheet on you workbook.

Options

Field Definition Type Required
Workbook List of workbooks available to the account. Workbook is another word for your Excel files. Dropdown TRUE

Output

Field Definition Type
Worksheet
Worksheet ID Unique identifier for the worksheet String
Worksheet Name Name of the worksheet String
Position Worksheet position; this is displayed as the tab number of the sheet specified by the name that you type. String
Visibility Visibility of the worksheet. See Visibility. String
