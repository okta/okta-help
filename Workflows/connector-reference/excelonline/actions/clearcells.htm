Clear Cells

Clear formats or data from cells in an worksheet.

Options

Field Definition Type Required
Workbook Dropdown TRUE
Workbook ID String FALSE
Worksheet Dropdown TRUE
Apply to Select the content that will be cleared from the cells.

  • All: clear formats and content.

  • Formats: clear formats only.

  • Contents: clear contents only.

 Dropdown TRUE

Input

The cell reference cells are used to construct a grid of cells to apply the clear action.

Field Definition Type Required
Cells
Cell Reference Begin Value representing a cell (for example, A2) in the top left corner of the grid to be cleared. String TRUE
Cell Reference End Value representing a cell (for example, D4) in the bottom right corner of the grid to be cleared. String TRUE

Output

Field Definition Type
Status Code Number
