Get Cell Value

Read values and other properties for a cell in an worksheet.

Options

Field Definition Type Required
Workbook Dropdown TRUE
Workbook ID String FALSE
Worksheet Dropdown TRUE

Input

Field Definition Type Required
Cell
Cell Reference In format A1, for the first cell on the spreadsheet. String TRUE

Output

Field Definition Type
Cell Text Text in the specified cell. String
Cell Value Value of the cell. String
Type String
Format Format for content in the cell. String
Column Hidden Indicates if the column is hidden. Boolean
Row Hidden Indicates if the row is hidden. Boolean
Hidden Indicates if the cell is hidden. Boolean
Formula Excel online formula. String
