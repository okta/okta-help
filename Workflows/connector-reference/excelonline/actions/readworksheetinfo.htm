Read Worksheet Info

Retrieve properties of an worksheet.

Options

Field Definition Type Required
Workbook Dropdown TRUE
Workbook ID String FALSE
Worksheet Dropdown TRUE

Output

Field Definition Type
Worksheet ID Unique identifier of the worksheet. String
Worksheet Name Name of the worksheet. String
Position Position of a worksheet in a workbook. The first worksheet is automatically assigned the value 0, the second is assigned the value 1, and so forth. Number
Visibility Visibility status for the worksheet. String
