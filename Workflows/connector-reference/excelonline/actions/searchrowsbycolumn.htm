Search Rows by Column

Search for a row or rows that match a value in a certain column of an worksheet.

Options

Field Definition Type Required
Workbook Dropdown TRUE
Workbook ID String FALSE
Worksheet Dropdown TRUE
Column to Search Column on which the search will be conducted. Dropdown TRUE
Result Set Either the first matching row or all matching rows. Dropdown TRUE

Input

Field Definition Type Required
Column
Value Column value. This is the number displayed in Excel on the top of first row. String FALSE

Output

Field Definition Type
Result Matching row or list of rows. String
Row ID Unique identifier of the row. Number
