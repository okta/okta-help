Update Cell

Update a value within a row in an worksheet.

Options

Field Definition Type Required
Workbook Dropdown TRUE
Workbook ID String FALSE
Worksheet Dropdown TRUE

Input

Field Definition Type Required
Cell
Cell Reference In format A1, for the first cell on the spreadsheet. String TRUE
Value New value. String TRUE
Formula Add an excel formula. For example, =SUM(A2:A10), = CONCAT(A1;B5),=MAX (number1, [number2], ...). String FALSE
Format Number format. uses special inner format. It can be read via the Format output field on the Get Cell Value card.

Formats:

  • Number: 0
  • Currency: $#,##0
  • Date: m/d/yyyy
  • Percentage: 0.00%
  • Fraction: # ?/?
  • Scientific: 0.00E+00
  • Text: @
  • Accounting: _($* #,##0.00_);_($* (#,##0.00);_($* "-"??_);_(@_)
 String FALSE

Output

Field Definition Type
Status Code Number
