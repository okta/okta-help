Read Row

Read a row of values from an worksheet.

Options

Field Definition Type Required
Workbook Dropdown TRUE
Workbook ID String FALSE
Worksheet Dropdown TRUE

Input

Field Definition Type Required
Row
Row Number Number of the row you want to read; this is the number displayed in Excel to the left of the first column. Number FALSE

Output

In the field selection window, select the column letters from which you want to retrieve values.

Field Definition Type
Values
Column letter (A-ZZ) A letter that identifies the column from which to read the row data. Each of the fields will represent the corresponding cell of the read row. String
