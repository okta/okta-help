connector

is a collaborative IT Service Management (ITSM) solution designed to provide rapid service delivery. You can create, read, and update requests, and there are events that are triggered by each of those actions.

This connector is only supported for use with cloud. Use with on-premises servers is not supported. See Moving from server to cloud for developers.
