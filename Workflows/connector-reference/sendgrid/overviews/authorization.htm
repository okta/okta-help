Authorization

Authorize this connector by creating a connection to your account. You can reuse this connection the next time that you build a flow with this connector.

Tip

You can create multiple connections and manage them from your Connections page.

Prerequisites

  • Okta Workflows admin credentials

  • User admin credentials

  • A account.

  • Set up a API key.

  • Make sure that you have a Verified Sender.

To set up a API Key

  1. Go to SendGrid.

  2. Click Create API Key.

  3. Enter a name for the API Key.

  4. Choose Full Access token API Key type.

  5. Click Create & View.

Use the API Key when creating the connection.

Configure SendGrid Connector

To create a new connection:

  1. Do one of the following:

    • In the Okta Workflows Console, go to Connections New Connection .
    • From a event or action card, click New Connection.

  2. Enter a Connection Nickname. This is useful if you plan to create multiple connections to share with your team.

  1. Enter the API Key information.
  2. Click Create.
