Schedule Single Send

Schedule a single send in .

Options

Field Definition Type Required
Schedule Choose whether send a Single Send immediately or at a later time; options are Immediate or Set Time. Dropdown TRUE

Input

Field Definition Type Required
Singe Send
Single Send ID Unique identifier of the Single Send in . String TRUE
Send At (with the Set Time option) ISO 8601 time at which to send the Single Send. For example, 2021-03-15T13:18:57.985Z.

Optionally, enter now to send the single send immediately.

 Date & Time TRUE

Field Definition Type
Single Send
Status Current status of the Single Send. String
Send At ISO 8601 time at which to send the Single Send — this must be set for a future time. Date & Time
