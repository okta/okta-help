Example: { "first_name": "John", "state_province_region": "CA", "address_line_2": "", "last_name": "Smith", "alternate_emails": "jsmith@othercompany.com", "city": "Los Angeles", "address_line_1": "123 Joy Street", "postal_code": 92320, "country": "USA", "email": "john.smith@company.com" }

Due to API limitations, you can add a maximum of 30,000 contacts.
