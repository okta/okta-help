Find Deal Owner

Find a user or deal owner using a certain search field. If more than one user/deal owner is found matching the search term, then the first one will be returned.

Options

Field Definition Type Required
Search Field the deal owner field to search by. Options: First Name, Last Name, Email, Updated At, Created At Dropdown FALSE

Input Fields

Field Definition Type Required
Field FALSE
Value the value you would like to search for. This will be checked agains the "Search Field" chosen in the options step, e.g. if you selected "Email" as the Search Field, entering an email into "Value" will return the deal owner with that email. String FALSE

Output Fields

Field Definition Type Required
Owner FALSE
Owner ID The unique ID of this user/deal owner. Can be used in the **Create Deal **and **Update Deal **actions to assign the deal to this user, or can be used with the **Read Deal Owner **card to read additional details about this user Number FALSE
