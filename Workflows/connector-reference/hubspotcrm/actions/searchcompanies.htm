Search Companies

Find companies that match the provided search criteria.

To narrow your search results, choose All Matching Companies for your result set and search using a single field (for example, Email). You can follow this card with a Filter or Filter Custom function card to further filter the search results.

To ensure accurate results, wait 10 seconds after using the Create Company card before calling this card, and 60 seconds after using the Update Company card.
