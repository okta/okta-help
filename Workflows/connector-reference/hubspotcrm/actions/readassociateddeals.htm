Read Associated Deals

Read information about existing deals associated to a given contact or company, by using the contact or company ID.

Options

Field Definition Type Required
Associated to  Select the entity that this deal is associated to. If you'd like to get a list of deals associated with a contact, select "Contact". If you'd like to get a list of deals associated to a company, select "Company". dropdown option FALSE

Input

The input fields for this card depend on whether or not you'd like to read deals associated to a company or for a contact instead. If you selected "Contact" as your option, you'll need to input your Contact ID, and vice versa.

Field Definition Type Required
Contact/Company FALSE
Contact/Company ID  the ID of the company or contact you'd like to read associated deals from. Number FALSE

Output Fields

This card outputs a list of deals. The actual properties inside of each field are dynamic, meaning that if you have custom properties created inside of Hubspot CRM, they will show up for each deal found in this list. The below list of fields only includes the default properties.
