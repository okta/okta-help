Dissociate Contact from Deal

Dissociate an existing contact from an existing deal in HubSpot CRM.

Input Fields

Field Definition Type Required
Deal
Deal ID The unique identifier of the deal you're removing the contact from. (this ID can be found by going to the deal in and looking for the numbers at the end of the URL. For example, .../deal/132852761/) Number FALSE
Contact
Contact ID  The unique identifier of the contact you're removing from the deal specified. (this ID can be found by going to the contact in and looking for the numbers at the end of the URL. For example, .../contact/101/) Number FALSE
