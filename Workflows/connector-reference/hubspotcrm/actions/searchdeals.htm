Search Deals

Search all deals in your organization by a field of your choosing.

Options

Field Definition Type Required
Search By Select the field you would like to use as your search parameter to search deals by. The fields in this list are dynamically generated by the properties you have setup for Deal objects inside of Hubspot CRM. dropdown option FALSE

Input Fields

Field Definition Type Required
Field FALSE
Value  The value you would like to use to search your deals with, set as the search parameter property you selected as your Search By value under "Options" String FALSE

Output Fields
