Associate Contact with Another Object

Associate a contact with an object in .

Options

Field Definition Type Required
Associate With Type of object to associate with the contact. Dropdown TRUE

Input

Field Definition Type Required
Contact ID Unique identifier of the contact you're associating with an object. This ID can be found navigating to the contact in and looking for the numbers at the end of the URL. For example, in the URL https://app.hubspot.com/contacts/<your_portal_id>/contact/101/, the ID is 101. Number TRUE
Company ID Unique identifier of the company you're associating with the contact specified by Contact ID. Number TRUE

Output

Field Definition Type
Status Code Number
