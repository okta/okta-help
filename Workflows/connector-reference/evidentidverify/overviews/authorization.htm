Authorization

When you add an card to a flow for the first time, you'll be prompted to configure a connection to your tenant. This will enable you to connect your account, save your account information, and reuse the connection in new flows that include .

You can create multiple connections and manage them from your Connections page.

Create a new connection

  1. Either open the Connections tab in the Workflows platform or open an action card. Click New Connection, then click .

  2. Enter a Connection Nickname. This is useful if you plan to create multiple connections to share with your team.

  3. Enter the name of your account in the Account Name field.

  4. In the Account Type drop-down list, select one of the following options: Production or Demo.

    5. Select Demo to access your Sandbox/Demo account.

  5. Enter your API key in the API Key field. For example, Zi4QPf8SyxByStMM6BqxCCLXEW4SrIo4hc0m5rR9PuI. You can find your API key in the API settings page of your customer portal.

  6. Click Create.

  7. If prompted, click Allow.
