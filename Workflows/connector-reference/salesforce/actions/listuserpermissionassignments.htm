List User Permission Assignments

Search for user permission assignments in by user ID and permission set ID.

Options

Field Definition Type Required
List By List results by Permission Set Assignment, Permission Set Group Assignment, or Permission Set License Assignment. Dropdown TRUE

Input

Field Definition Type Required
User
User ID Unique identifier of the user to whom permissions are assigned. String False
Permission Assignment
Permission Set ID Unique identifier of the permission set. String FALSE
Permission Set Group ID Unique identifier of the permission set group. String FALSE
Permission Set License ID Unique identifier of the permission set license. String FALSE

Output

Field Definition Type
Permission Set Assignments
Assignment ID Unique identifier of the permission set assignment. String
User ID Unique identifier of the user to whom permissions are assigned. String
Permission Set ID Unique identifier of the permission set. String
System Modified Timestamp attribute of the permission set that indicates when it was last updated or assigned. Date & Time
Permission Set Group Assignments
Assignment ID Unique identifier of the permission set group assignment. String
User ID Unique identifier of the user to whom a permission set group is assigned. String
Permission Set Group ID Unique identifier of the permission set group. String
System Modified Timestamp attribute of the permission set group that indicates when it was last updated or assigned. Date & Time
Permission Set License Assignments
Assignment ID Unique identifier of the permission set license assignment. String
User ID Unique identifier of the user to whom a permission set license is assigned. String
Permission Set License ID Unique identifier of the permission set license. String
Created by ID Unique identifier of the admin who created the permission set license assignment. String
Created Date Date and time when the permission set license assignment was created. Date & Time
Is Deleted? Indicates whether the permission set license assignment is deleted; options are True and False. Boolean
Last Modified By ID Unique identifier of the admin who last updated the permission set license assignment. String
Last Modified Date Date and time when the permission set license assignment was last updated Date & Time
System Modified Timestamp attribute of the permission set license that indicates when it was last updated or assigned. Date & Time
