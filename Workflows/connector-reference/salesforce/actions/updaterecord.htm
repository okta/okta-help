The input fields for this Action are dynamically generated from your Account and dependent on the Record Type dropdown list in Options.

Record ID ID of the record inside . String TRUE

These input fields are dependent on the Record Type dropdown list, and are dynamically generated at design time based on the type of record you want to create. Custom fields are supported.

Multi-select picklist custom fields should be entered as a semicolon-delimited list of text. For example, "Labradoodle;Corgi;Bernese Mountain Dog". To read the available picklist options as part of your , use the Read Picklist Values action.
