Update Global Picklist Values

Add or remove the options that are available in a global picklist field. Changes will affect all picklists that inherit from the global value set.

Options

Field Definition Type Required
Record Type Dynamically generated list of Record Types from your account. String FALSE

Input

Field Definition Type Required
Picklist
Label API value of the picklist field. String TRUE
Name Display name of the picklist field. String TRUE
Options
Add Values Display names of the values that you want to add to the picklist. String FALSE
Remove Values Display names of the values that you want to remove from the picklist. String FALSE
