List Permission Set Group Assignments

Search for permission set group assignments in by permission set group ID and user ID.

Input

Field Definition Type Required
Permission Set Group
Permission Set Group ID Unique identifier of the permission set group. String FALSE
User
User ID Unique identifier of the user to whom permission set groups are assigned. String FALSE

Output

Field Definition Type
Permission Set Group Assignments
Assignment ID Unique identifier of the permission set group assignment. String
User ID Unique identifier of the user to whom permission set groups are assigned. String
Permission Set Group ID Unique identifier of the group if it's associated with a permission set. String
System Modified Timestamp attribute of the permission set group that indicates when it was last updated or assigned. Date & Time
