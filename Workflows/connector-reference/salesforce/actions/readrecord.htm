Read Record

Read dynamically generated fields from an existing record of the specified record type.

Options

Field Definition Type Required
Read
Record Type Dynamically generated list of record types from your account. Dropdown FALSE

Input

Field Definition Type Required
Read
Record ID ID for the record of the specified type within . String TRUE

Output

The output fields are dynamically generated from your account. Includes custom fields.

Multi-select picklist custom fields will be returned as a semicolon-delimited list of text. For example, "Labradoodle;Corgi;Bernese Mountain Dog".
