Deactivate User

Deactivate a user.

Input

Field Definition Type Required
User
User ID

Deactivating a Salesforce user prevents the user from logging into Salesforce and allows your organization to apply their license to another user. In cases when deactivating a user isn’t immediate, admins can freeze a user which prevents them from logging into Salesforce but still uses a Salesforce license.

 String TRUE

Output

Field Definition Type
Status Code Number
