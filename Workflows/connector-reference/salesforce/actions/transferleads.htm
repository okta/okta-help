Transfer Leads

Transfer all leads from one user to another.

Options

Field Definition Type Required
Ignore Assignment Rules Option to ignore assignment rules when transferring leads; options are Yes or No. String TRUE

Input

Field Definition Type Required
Transfer From
User ID Unique identifier of the user whose leads you want to transfer. String TRUE
Transfer To
User ID Unique identifier of the user who will receive the leads. String TRUE

Output

This action card has no output fields.
