Outbound Message

Start a flow when a Outbound Message is triggered.

This connector requires additional setup in your instance:

    In , open the Setup page.

    In the Quick Find/Search field, enter Workflow Rules.

    On the Workflow Rules page, click New Rule.

    Select an object type, and then click Next.

    Enter a rule name, evaluation criteria, and rule criteria.

    Click Save & Next.

    Click Done.

    On the Workflow Rules page, find the rule that you created, and then click Edit.

    In the Add Workflow Action drop-down menu, select New Outbound Message.

    Enter a name, description, endpoint URL, and the fields you want to include in the new message. Set up a flow beginning with the Outbound Message event card. Click the "</>" icon in the bottom right and copy the provided URL.

    Click Save.

    On the Workflow Rules page, find the rule you just created, and then click Activate.
