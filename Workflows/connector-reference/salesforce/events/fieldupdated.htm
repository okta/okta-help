Field Updated

Start a when an existing record is updated.

Field Updated does not monitor the Owner_ID field on any record type because API does not record ID changes in its system.

Field Updated can only monitor the most recent change made on a field.

Field Updated can only monitor fields that are history-enabled.

You must be a Admin to enable history:

    In your instance, navigate to Setup > Build.

    Click Customize.

    Select the record type that you want to monitor, and then click Add a Custom Field.

    Click Set History Tracking.
