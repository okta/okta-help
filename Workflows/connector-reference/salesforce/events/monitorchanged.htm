Monitor Changed Monitor Changed

Monitor for updated records of a specified type.

Note:

Do not use the Test functionality on monitors. Because records created in Salesforce are not immediately picked up by the API, monitors are set to a 15-minute delay. You can miss record changes if you test before the monitor runs.

Options

Field Definition Type Required
Record type Available record types in , including custom records. Dropdown FALSE

Output

Output fields are dynamically generated from your record type selection. Custom fields are marked with _c. Standard fields are listed below.

Field Definition Type
ID Unique identifier of the record that was updated in Number
IsDeleted true if the record has been deleted, false if it is still active Boolean
IsPrivate true if the record is private, false if it is public Boolean
Name Name of the record that was updated String
CreatedDate Timestamp when the record was created (UTC format, timezone of your instance) Date & Time
