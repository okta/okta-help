Search Groups

Search for a group by property values.

An Exact Match search returns groups whose properties match exactly, while Starts With searches return groups whose properties start with that value.

The Search Groups action card returns a maximum of 4,000 groups.

Options

Field Definition Type Required
Result Set

Choose to return either the First Matching Record, or return All Matching Records, which returns a list of objects as the output field on the card.

Search By

Choose the search operator when searching for groups.

  • Starts With returns groups whose properties start with the input values.
