SFTP connector

The Secure File Transfer Protocol () connector enables you to securely upload and download files between the file system and any remote  server.

Authorize your SFTP account

When you add a card to a flow for the first time, prompts you to create a connection. See Authorization.

Guidance for SFTP

For guidance and best practices when using an connector card in your flows, see Guidance for SFTP connector.

SFTP connector action cards

Action

Description

Download File

Download a file from an destination to the file system.

Upload File

Upload a file from the file system to an destination.

Write File

Write a text file to an server.
