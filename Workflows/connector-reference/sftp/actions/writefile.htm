Write File

Write plain text into a file on the given path.

This action can't be used to create files that contain anything other than plain text. For example, an document with a .docx extension.

See Guidance for SFTP connector.

Input

Field Definition Type Required
File
Text Content Text content that you want to write out to a file. Text TRUE
Destination File Path Path where the file is stored, including the folder and file name. Text TRUE

Output

Field Definition Type
Succeeded This is True if the file write is successful; otherwise False. True/False
