Download File

Download a file from an server destination to the file system.

See Guidance for SFTP connector.

Input

Field Definition Type Required
Source
File path Path of the file to download, including the folder and file name, on the remote server. Text TRUE

Output

Field Definition Type
File
File Content Pointer to the file content in the Workflows file system. File
Filename Name of the downloaded file. Text
Size Size of the downloaded file in bytes. Number
