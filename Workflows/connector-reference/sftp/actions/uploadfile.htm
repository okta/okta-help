Upload File

Upload a file from the file system to an server destination.

See Guidance for SFTP connector.

Input

Field Definition Type Required
Input
File Content Pointer to the file content on the file system. File TRUE
Destination File Path

Destination for the file, including folder and file name, on the remote server.

Text TRUE

Output

Field Definition Type
Succeeded This is True if the file uploaded successfully; otherwise False. True/False
