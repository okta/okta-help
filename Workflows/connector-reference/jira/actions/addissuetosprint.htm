Add Issue to Sprint

Add an existing issue to a sprint in .

The issue can be identified by the key (in the issue URL https://yourdomain.atlassian.net/projects/HT/issues/HT-22, HT-22 is the key) or mapped from a Read Issue or Create Issue card.

Input

  • Sprint

    • Sprint ID(text): ID of the sprint

  • Issues

    • Issue IDs or Keys(list of text): ID or key of the issue to add to the sprint

Output

There is no output for this card.
