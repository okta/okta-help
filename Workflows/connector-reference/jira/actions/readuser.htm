Read User

Read a user in .

Unless otherwise indicated, field types are text.

Input

  • User

    • Account ID: user's account ID

Output

  • User

    • Display Name: display name of the user

    • Account ID: account ID of the user; to obtain the account ID, you can perform a search by email address using the Search Users card, or note the last ID from the profile page. For example, in the URL https://oktainc.atlassian.net/jira/people/5d0c7af1ac9cfa0bc6eed1c7, the account ID is 5d0c7af1ac9cfa0bc6eed1c7.

    • Email Address: user's email address

    • Active? (true/false): indicator if the user is active
