Search Projects by Issue

Search for a project with an issue ID or key.

Field types are text.

Input

  • Issue

    • ID or Key: ID or key of the issue to search by. You can use the Search Issues card or find the key in the URL of the issue (https://yourdomain.atlassian.net/browse/AT-19).

Output

  • Project

    • Project ID: ID of the project

    • Key: project key

    • URL: URL for the project

    • API Resource URL: URL of the API resource for the project
