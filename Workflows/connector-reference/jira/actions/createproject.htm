Create Project

Create a project in .

Field types are text.

Input

  • Project

    • Name: name for the project

    • Key: short identifier for the project

    • Description: description of the project

    • Lead Account ID: account ID of the lead for the project; to obtain the lead account ID, you can perform a search by email address using the Search Users card, or note the last ID from the profile page. For example, in the URL https://oktainc.atlassian.net/jira/people/5d0c7af1ac9cfa0bc6eed1c7, the lead account ID is 5d0c7af1ac9cfa0bc6eed1c7.

    • Category: ID of the project's category

    • Type: type of project; values are Software, Service Desk, and Business

    • Issue Security Scheme: ID of the issue security scheme for the project, which enables you to control who can and cannot view issues

    • Permission Scheme: ID of the project's permission scheme, which contains information on permission grants for groups or users

    • Notification Scheme: ID of the project's notification scheme, which contains a list of events and recipients who will receive notifications for those events

Output

  • Project ID: ID of the project

  • Key: short identifier for the project (created above)

  • URL: URL for the project

  • API Resource URL: API Resource URL for the project
