Transition Issue

Transition a issue to a different status (for example, IN PROGRESS, DONE, or CLOSED,) using a transition name.

To change the status of an issue, you must use the name of the transition and not the name of the end status.

Input

Field Definition Type Required
Transition
Issue ID or Key ID or key of the issue that you want to transition to a new status. String TRUE
Transition To

Enter the transition name that matches the end state where you want to move the issue.

These transition names are specific to your project. Depending on your project settings, the transition names and issue statuses may be different.

You can find these names in the interface for your ticket, inside the transition issues dropdown menu.

For example:

Transition name

Issue status
Start Progress IN PROGRESS
Resolve Issue RESOLVED

Close Issue

CLOSED

String TRUE
Comment Comment about the issue. String FALSE

Output

Field Definition Type
Response
Status Code Number
