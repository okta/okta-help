Log Work

Log work on an issue in .

Note: To edit the visibility of , use the HTTP Request card with System Admin credentials.

Input

  • Issue

    • ID or Key(text): ID or key of the issue to log work on. You can use the Search Issues card or find the key in the URL of the issue (https://yourdomain.atlassian.net/browse/AT-19).

  • Worklog

    • Comment(text): comment about the logged work

    • Time Started(text): timestamp when the work began, formatted using ISO UTC (2016-09-07T16:55:25.670Z), offset notation (2016-09-07T16:55:25+00:00), or the expected format (2017-06-16T02:48:07.240+0000)

    • Time Spent (seconds)(number): amount of time spent on the issue (in seconds - 3600 for one hour)

Output

  • Response

    • Status Code(number): status of the attempt (status codes are defined here)
