Create Comment

Create a comment on an issue in . The issue can be identified by the key (in the issue URL https://yourdomain.atlassian.net/projects/HT/issues/HT-22, HT-22 is the key) or mapped from a Read Issue or Create Issue card.

Unless otherwise indicated, field types are text.

Input

  • Issue

    • ID or Key: ID or key of the issue you want to comment on

  • Comment

    • Body: content of the comment

Output

  • Response

    • Status Code(number): Status of the comment attempt. Status codes are defined here.
