Search Users

Search for a accountId in .

Unless otherwise indicated, field types are text.

Options

  • Result Set (drop-down): the first user or all users that match your search query

Input Field

  • User

    • Query: name or email address to search (for example, sarah or sarah@companyname.com)

Output Field

  • Account ID: user's account ID; to obtain the account ID, you can perform a search by email address using the Search Users card, or note the last ID from the profile page. For example, in the URL https://oktainc.atlassian.net/jira/people/5d0c7af1ac9cfa0bc6eed1c7, the account ID is 5d0c7af1ac9cfa0bc6eed1c7.
