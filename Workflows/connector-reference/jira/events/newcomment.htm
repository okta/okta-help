New Comment

Trigger a flow when a new comment is created in .

Unless otherwise indicated, field types are text.

Options

  • Project (drop-down): project to monitor for comments

Output

  • Webhook

    • Raw Data (object): raw webhook data for the comment

  • Issue

    • Issue ID: ID of the issue the comment was added to

  • Comment

    • Comment ID: ID of the comment

    • URL: URL to the comment

    • Body: content of the comment

    • Created Date (date and time): timestamp when the comment was posted

  • Author

    • Account ID: Account ID of the comment's author; to obtain the account ID, you can perform a search by email address using the Search Users card, or note the last ID from the profile page. For example, in the URL https://oktainc.atlassian.net/jira/people/5d0c7af1ac9cfa0bc6eed1c7, the account ID is 5d0c7af1ac9cfa0bc6eed1c7.

    • Display Name: display name of the comment's author

    • Active (true/false): indicator if the user is active

  • context

    • Execution ID: unique identifier of the flow execution
