List AWS Entitlements

List the entitlements for a specific principal.

Options

Field Definition Type Required
Region Dropdown TRUE
Instance ARN Dropdown TRUE
Account ID Dropdown TRUE

Input

Field Definition Type Required
Instance ARN String TRUE
Principal Type Dropdown TRUE
Principal ID GUID identifier of the principal from which to list the entitlements. String TRUE
Account ID String TRUE

Output

Field Definition Type
Entitlements (List of Objects)
Account ID Identifier of the AWS account. String
Permission Set Amazon Resource Name (ARN) of the permission set. String
