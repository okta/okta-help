Trigger a flow with the User MFA Factor Activated event card

To trigger a flow, you must use a mobile device and complete these steps:

  1. Using a test account, in the Admin Console, navigate to Security Multifactor .

  2. Verify that at least one factor is active. Factors that are already activated are designated as such with green check marks.

  3. To activate a factor, select one from the Factor Type list and select Activate from the drop-down for that factor.

    It is recommended that you activate the Okta Verify, SMS Authentication, or Security Question factors.

  4. In the top right corner of the Admin Console, click the account drop-down then click My settings.

  5. In the Extra Verification section, click Set up for the new factor that you selected previously, then complete the steps to activate that factor.

    When you have completed the activation of the new factor, any flow using the User MFA Factor Activated event card will be triggered.
