Trigger a flow with the User MFA Factor Deactivated event card

To trigger a flow, you must already have a factor activated. Complete these steps:

  1. Using a test account, in the top right corner of the Admin Console, click the account drop-down then click My settings.

  2. In the Extra Verification section, click Remove for the factor that you want to deactivate.

  3. Click Yes to confirm the removal of the factor.

    When factor is removed, any flow using the User MFA Factor Deactivated event card will be triggered.
