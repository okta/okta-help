Activate User

Activate an user by ID or username.

Options

Field Definition Type Required
Send Email? Indicates whether an email will be sent to the activated user. Options are Yes or No. Dropdown TRUE

Input

Field Definition Type Required
ID or Login ID or login of the user (usually in an email format). String TRUE

Output

Field Definition Type
Status Code String
Activation (if the Send Email? option is set to No)
Token Activation token used to create a custom activation link and complete a user's activation process. String
Url Activation link for the user to set up their account. String
