Get Primary Linked Object Value

For the associated user specified by ID, returns the user ID and self link for the primary user in the relationship specified by the primary name in .

Options

Field Definition Type Required
Primary Name Option to specify a primary name manually or to select a primary name from the dropdown list.

If you select -- Enter Primary Name --, then enter a string in the Primary Name input field.

 Dropdown TRUE

Input

Field Definition Type Required
User
Associated User ID ID of the associated user for whom you want to get the primary user ID String TRUE
Linked Object
Primary Name Name of the primary relationship being queried String TRUE

Output

Field Definition Type
Primary Users
User ID ID of the primary user in the specified relationship String
User Self Link For the associated user ID, returns the self link for the primary user in the relationship specified by the primary name. If the user specified isn't the primary user in any relationship, an empty array is returned. String
