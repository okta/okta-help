Reset Password

Reset password for an account.

Options

Field Definition Type Required
Convert to Federated User Convert a to Federated User. Possible values are:
  • Yes
  • No (default)
 Dropdown TRUE
Send Email When Convert to Federated User is set to No, it contains two options:

  • Yes

  • No

When Convert to Federated User is set to Yes, it contains only one option: No.

Send Email is not available on the API endpoint if Yes is selected for Convert to Federated User.

 Dropdown FALSE

Input

Field Definition Type Required
User      
ID or Username Unique ID or login username of the user in email format.

For example, samantha.smith@gmail.com

 String TRUE

Output

Field Definition Type
Status Code Number
Reset Password    
URL URL to reset the password.

Appears when No is selected from the both the Convert to Federated User and Send Email option.

 String
