List Users Assigned to Application

List all users assigned to an application.

The output fields in this card are dynamically generated based on your instance details.

Options

Field Definition Type Required
Application

Select which application you want to use for the user assignment search.

Choose Enter Application ID to select an app by its ID.

The maximum number of applications displayed is 100, and the maximum number of application instances displayed is 2000. If server or network connectivity issues cause a timeout, then Enter Application ID is the only option displayed.
