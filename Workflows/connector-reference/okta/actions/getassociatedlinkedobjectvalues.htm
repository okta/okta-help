Get Associated Linked Object Values

For the primary user specified by ID, return the list of user IDs and self links for the associated users in the relationship specified by associated names in .

Options

Field Definition Type Required
Associated Name Option to specify an associated name manually or to select an associated name from the dropdown list.

If you select -- Enter Associated Name --, then enter a string in the Associated Name input field.

 Dropdown TRUE

Input

Field Definition Type Required
User
Primary User ID ID of the user for whom you want to get associated user IDs String TRUE
Linked Object
Associated Name API name of the relationship to the primary user ID String TRUE

Output

Field Definition Type
Associated Users
User ID ID of each user in an associated relationship for the specified primary name String
User Self Link For the associated user specified by ID, returns the self link for each user in a relationship specified by the primary name String
