Search Groups

Search for a group in .

Options

Field Definition Type Required
Result Set Choose from First Matching Record and All Matching Records

First Matching Record: Returns the first group that matches your search. Use this option when you're looking for a specific group.

All matching Records: Returns a collection of groups that match your search. Use this option when you're looking for a filtered set of groups.

 Dropdown TRUE
Search Groups Choose one of the following options to display your search results:

By Prefix: returns groups whose properties start with the input value. This option supports a single word search and does not support searching by multiple words.

By Equality: returns groups when the field and the value match exactly. The full name is required.

    

Input

Field Definition Type Required
Group
Name String FALSE
Email Email associated with the group. String FALSE

You can use single words or a combination of words and spaces in your group search.

A search using single words will return all groups with partial matches. For example, to find the group "All Remote Users", you can use All or Remote or Users in your query.

If you use a combination of words and spaces in your group search, the query results will show only the group names that match this combination exactly. For example, to find the group "All Remote Users", you can only use All Remote Users in your query. Any other combination (for example, All Remote) will return null.

If you use only a space in your group search, the query results will also be null.

As a general guidance, don't add a space to the Name field in your group search unless you are performing an exact match search.
