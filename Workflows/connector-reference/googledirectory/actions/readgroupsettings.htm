Read Group Settings

Retrieves settings for a group, as identified by the group email address.

Input

Field Definition Type Required
Group
Email The email address used to identify the group in String TRUE

Output

Field Definition Type
Email Email address of the group. String
Name String
Description Detailed description of the group. String
Who Can Join Specifies what set of users can join the group. String
Who Can View Membership Permission level required to view the group membership list. String
Who Can View Group Permission level required to view the group messages. String
Allow External Members Identifies whether members external to your organization can join the group. String
Who Can Post Message Permission level required to post messages to the group. String
Allow Web Posting Allows posting to the group from the web. String
Primary Language The primary language for the group. This value is empty if the primary language is never set for the group. String
Is Archived Allows the group contents to be archived. String
Archive Only Specifies if the group is in an archive-only state. String
Message Moderation Level Specifies the moderation level for incoming messages. String
Spam Moderation Level Specifies the moderation level for incoming messages detected as spam. String
Reply To Who should receive the default reply message. String
Custom Reply To An email address used for replying to a message if the Reply To property is set to REPLY_TO_CUSTOM. String
Include Custom Footer Specifies if a custom footer is to be included in reply messages. String
Custom Footer Text Content for the custom footer. String
Send Message Deny Notification Allows a member to be notified if the group owner denies their message to the group. String
Default Message Deny Notification Text When a message is rejected, this text is sent to the message's author. String
Members Can Post As The Group Enables members to post messages using the group's email address instead of their own email address. String
Include in Global Address List Enables the group to be included in your Global Address List. See Manage your Global Address List. String
Who Can Leave Group Specifies who can leave the group. String
Who Can Contact Owner Specifies who can contact the group owner. String
Favorite Replies On Top Indicates if favorite replies should be displayed before other replies. String
Who Can Moderate Members Specifies who can manage membership for the group. String
Who Can Moderate Content Specifies who can moderate content for the group. String
Who Can Assist Content Specifies who can moderate metadata for the group. String
Custom Roles Enabled For Settings To Be Merged Specifies whether the group has a custom role that's included in one of the settings being merged. String
Enable Collaborative Inbox Specifies whether a collaborative inbox remains turned on for the group. String
Who Can Discover Group Specifies what set of users can discover the group. String
Default Sender Default sender address for members who can post messages as the group. String
