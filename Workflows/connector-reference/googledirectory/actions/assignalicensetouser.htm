Assign License to User

Assign a license to user.

For more information visit Google's license documentation page.

Input

Field Definition Type Required
Product
Licenses List of product licenses (or single license) that you want to assign to the user. List of Objects FALSE
User
Primary Email Email address of the user to whom you want to assign licenses. String TRUE

If you try to assign an already-assigned license to a user, the action card will return an error:

  • A 503 status error code will be thrown if the license you are trying to assign is automatically assigned for new users.

  • A 412 status error code will be thrown if the license you are trying to assign is not automatically assigned to new users but is currently assigned to the user who is specified on the card.

Note that if the Cloud Identity Free license is assigned by default for a Google Workspace account, then the license will be automatically assigned to all new users.

Output

Field Definition Type
License Name Name of the assigned license. String
Status Code Number
