Update Calendar

Update an existing calendar permission in using its ID.

Input

Field Definition Type Required
Update
Calendar Choose a calendar from the list.
 Dropdown TRUE
Calendar      
Name Name of the calendar. String  
Color Specifies the color theme to distinguish the calendar from other calendars in a User Interface.
  • Light Blue
  • Light Green
  • Light Orange
  • Light Gray
  • Light Yellow
  • Light Teal
  • Light Pink
  • Light Brown
  • Light Red
  • Max Color
  • Auto
 Dropdown  
Update By      
Permission ID Unique ID of the permission. String TRUE
Permission      
Role Specifies the color theme to distinguish the calendar from other calendars in a UI. The property values are:
  • None
  • FreeBusy Read
  • Limited Read
  • Read
  • Write
  • Delegate Without Private Event Access
  • Delegate With Private Event Access
 Dropdown FALSE

Output

Field Definition Type
Status Code Number

A 200 status code indicates that the event was updated.
