Search User By Email

Search by email address for a user's information.

Searching is limited to the team you selected when you authenticated . The card will return empty fields if the email address is not found.

Input

Field Definition Type Required
User
Email The email address of the user. String FALSE

Output

Field Definition Type
User
Full Name The full name of the user associated with the email address. String
Username The username of the user associated with the email address (without the @ symbol). String
User ID String
