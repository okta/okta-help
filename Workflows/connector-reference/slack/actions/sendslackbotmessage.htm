Send Slackbot Message

Send a notification to a user's @slackbot channel.

Options

Field Definition Type Required
Unfurl URLs? True, if should scan links found in messages and create attachments based on the link's contents Boolean TRUE

Input

Field Definition Type Required
Username User's username (without the @ symbol) String FALSE
Message Message text (may include emojis:, #channels, @users, and /commands) String FALSE
Slackbot Name Name of the sender from the slackbot channel. For example, Customer Tracking or Sales Bot. String FALSE

Output

Field Definition Type
Timestamp Timestamp when the message was sent String
