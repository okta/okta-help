Rename Channel

Rename a Slack channel.

By default, only the user that created the channel or a user who's a workspace admin or owner can rename a channel.

Options

Field Definition Type Required
Channel List of available channels. Dropdown TRUE

Input

Field Definition Type Required
Channel
Channel ID Unique identifier of the Slack channel. This field appears when Channel is set to -- Enter Channel ID --. String TRUE
Rename
Name New name for the channel selected in Channel or specified by Channel ID. String TRUE

Output

Field Definition Type
Status Code String
