Join Channel

Join a Slack channel. This action card adds the user who created the connection that's used by the card to the selected channel.

Options

Field Definition Type Required
Channel List of public channels of which the user who created the connection is not a member but can join.

If that user is already a member of the channel, then the Already in channel message will be returned.

A maximum of 200 public channels will be displayed.

The dropdown will display only those unarchived channels of which the user who created the connection is NOT a member. Even if the number of public channels may not exceed 200, not all applicable channels may display anyway. To specify a public channel that is not listed, select the Enter Channel ID option and populate the Channel ID input field.

If a failure occurs and no channels are listed in the dropdown, then only the Enter Channel ID option will be displayed.

 Dropdown TRUE

Input

Field Definition Type Required
Channel ID ID of the channel to join. String FALSE

Output

Field Definition Type
Channel ID ID of the channel that was joined. String
