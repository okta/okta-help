Update PreAuthorization

Update a preauthorization in .

Input

Field Definition Type Required
Project Name Target project name with which the preauthorization is associated. String TRUE
Authorization ID Identifier of the updated authorization. String TRUE
Disabled Indicates whether the authorization is enabled. Boolean FALSE
Expires At Update values for the date and time when the authorization will expire. Date & Time TRUE
User Name User name for the user whose preauthorization is updated. String TRUE

Output

Field Definition Type
ID Unique identifier for the preauthorization. String
Expires At Date and time when preauthorization expires. Date & Time
Disabled Indicates whether the preauthorization is still valid. Boolean
Username Username to which the preauthorization is assigned. String
