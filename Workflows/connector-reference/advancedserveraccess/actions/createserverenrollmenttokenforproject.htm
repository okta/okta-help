Create Server Enrollment Token for Project

Create a server enrollment token for an project. These tokens are used to enroll a server in a specific project in .

Input

Field Definition Type Required
Project Name Target project to create the enrollment token. String TRUE
Description Description for the enrollment token. String FALSE

Output

Field Definition Type
ID Unique ID of the enrollment token. String
Token Token attribute of the returned JSON object. String
Created By User User who created the server enrollment token. String
Issued At Date and time when the server enrollment token was issued. Date & Time
Description Description for the enrollment token. String
