Add User to Group

Add an user to a locally created group.

The Add User to Group action card can only be used with ASA groups that are created within ASA. Groups that are provisioned via SCIM from Okta cannot have their group memberships altered in ASA.

Input

Field Definition Type Required
User Name User name to be added to an group. String TRUE
Group Name Name of the locally created group to which a user is added. String TRUE

Output

Field Definition Type
Status Code Number
