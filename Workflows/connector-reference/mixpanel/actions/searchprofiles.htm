Search Profiles

Search for user profiles in .

Options

Field Definition Type Required
Project Name Name of the user's project Dropdown TRUE
Result Set Choose whether the search returns First Matching Profile or All Matching Profiles Dropdown TRUE
Search Type Type of data against which a search is performed; options are Property match or Custom expression;

Property match: Search for a profile with a given name that was created based on input values

Custom expression: Use a JQL expression to return a profile or list of profiles; see Write JQL.

 Dropdown TRUE

Due to limitations to the Mixpanel API, a 60-second delay is recommended between creating a profile in Mixpanel and searching for a new profile using the Search Profiles action card. See FAQs.

Due to limitations to the Mixpanel API, you can use the Search Profiles action card to make a maximum of 400 requests per hour.

This card supports dynamic fields. To add fields to this card, click the empty field at the bottom of the input field list under Click or drop here to create. To reorder fields, select Edit card from the Options menu and drag fields to their new location.
