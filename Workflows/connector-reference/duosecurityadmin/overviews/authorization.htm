Authorization

Authorize this connector by creating a connection to your account. You can reuse this connection the next time that you build a flow with this connector.

You can create multiple connections and manage them from your Connections page.

Prerequisites

  • Okta Workflows admin credentials

  • admin credentials

  • A account.

  • All scopes must be granted and settings must be saved.

To configure connector in Workflows:

  1. In the Okta Workflows Console, go to Connections New Connection Duo Security Admin .

  2. Enter a Connection Nickname. This is useful if you plan to create multiple connections to share with your team.

  1. From the Duo Security Application Admin API page, enter a value for the following fields:
    • Integration Key
    • Security Key
    • API Hostname. For more information on the API Hostname value, see Duo Auth API.
  2. Click Create.
