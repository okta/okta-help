Revoke Token

Revoke your account's API key.

If you delete a connection with in Workflows, you will also need to delete the API key manually. The API key is not revoked automatically.

Note: You will need a super user account to perform this task.

To revoke your API key:

  1. With your account credentials, sign in to your account.
  2. Go to Account Settings > Security.
  3. Click X in the Forget App column, next to the API Key that you want to revoke.
  4. Click OK to confirm.
