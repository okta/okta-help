connector

is a service that enables the storage and sharing of files in the cloud. You can create, update, and delete users and folders, and make custom API calls to the Box API.

Authorize your account

When you add a card to a flow for the first time, you'll be prompted to configure the connection. This will enable you to connect your account, save your account information, and reuse the connection for future flows. See Authorization.

To revoke an API key from your account, see Revoke Token.
