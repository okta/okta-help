Assign Task

Assign a task to a user.

Input

Field Definition Type Required
Task Assignment
Task ID Unique identifier of the task.

For new tasks, you can get this value from the output of the Create Task card.

 String TRUE
Assign to ID Unique identifier of the user assigned the task. String FALSE
Assign to Email Email address of the user. String FALSE

Output

Field Definition Type
Task Assignment
Task Assignment ID Unique identifier of task assignment. String
Time Assigned Time stamp when the task was assigned. String
