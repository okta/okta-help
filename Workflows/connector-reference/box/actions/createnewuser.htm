Create User

Create a new user in your Account. You must a Box Administrator to perform this action.

Field Definition Type Required

Status Status of the user (for example, Active or Inactive). Dropdown FALSE
Timezone Timezone of the user (for example, Active or Inactive). String FALSE
Allotted Space (GBs) Amount of space (in gigabytes) allotted to the user. Number FALSE
Can See Managed Users If true, the user can see managed users. Boolean FALSE
Is Exempt From Device Limits If true, the user is exempt from device limits. Boolean FALSE
Is Exempt From Login Verification If true, the user is exempt from login verification. Boolean FALSE
Is External Collab Restricted If true, external collaboration is restricted. Boolean FALSE
Is Sync Enabled If true, the sync is enabled. Boolean FALSE
