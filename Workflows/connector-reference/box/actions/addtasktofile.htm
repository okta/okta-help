Add Task to File

Attach a task with a due date to a file.

Input

Field Definition Type Required
Task
File ID Unique identifier of the associated file. String TRUE
Message Message about the task. String FALSE
Due Date Date and time when the task must be completed.

For example, 2012-12-12T10:53:43-08:00.

 String FALSE

Output

Field Definition Type
Task
Task ID Unique identifier of the task. String
Time Created Timestamp when the task was created. String
