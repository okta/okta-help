Add Comment to File

Add a comment to any of the files in using the File ID.

Input

Field Definition Type Required
Comment
File ID Unique identifier of the file to comment on String TRUE
Message Text of the comment (if message contains a tag, leave this blank) String FALSE
Tagged Message Text of the comment if it includes a tagged user @[userid:Username]

If message does not contain a tag, leave this blank.

 String FALSE

Output

Field Definition Type
Comment
Comment ID Unique identifier of the new comment String
Time Created Timestamp that the comment was created.

For example, 2012-12-12T11:25:01-08:00.

 String
