Search Folders

Search folders in by using filters.

Options

Field Definition Type Required
Result Set First matching record: return the first folder to match the search parameters

All matching records: return all folders that match the search parameters

 Dropdown TRUE

Input

Field Definition Type Required
Search By
Name Name of the folder String TRUE
Start Date Start date of a created at range (End Date must be used in conjunction) String FALSE
End Date End date of a created at range (Start Date must be used in conjunction) String FALSE
User ID String FALSE
Parent ID Unique identifier of the parent folder String FALSE
