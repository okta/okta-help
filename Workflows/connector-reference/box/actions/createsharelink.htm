Create Share Link

Create a share link for an item.

Input

Field Definition Type Required
Item
ID Unique identifier of the source file or folder String TRUE
Share Link
Access Accessibility of the file (options are open, company, or collaborators) String FALSE
Date Unshared Expiration date String FALSE
Password Password for the link String FALSE
Can Download Indicator of downloadable content String FALSE
Can Preview Indicator of viewable content String FALSE

Output

Field Definition Type
Share Link
Share URL URL to view the file String
Download URL URL to download the file String
