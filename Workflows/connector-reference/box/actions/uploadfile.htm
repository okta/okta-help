Upload File

Upload a file into .

Input

Field Definition Type Required
File
Name Name of the file. String TRUE
Parent Folder ID Unique identifier of the target folder. String TRUE
File Content Unique ID used to identify a file in the Workflows file system. File TRUE

A file must be at least 1 byte in size, but cannot be larger than 2 GB. See Workflows system limits.

Only files that are already stored in the Workflows platform file system can be uploaded. Local files cannot be uploaded to using this action card.

The maximum file size for uploads to Box will depend on your Box account type. See Understand the Maximum File Size You Can Upload to Box.

Output

Field Definition Type
File
File ID Unique ID used to identify and retrieve a file in the Workflows file system. String
