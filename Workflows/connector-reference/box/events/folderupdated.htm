Folder Updated

Use this event to trigger a flow when a file is uploaded to or any of its root folders.

Output

Field Definition Type
Event
Event ID unique identifier of the event String
Event Type type of event that created this file (for example, FILE_UPLOAD) String
Folder
Folder ID unique identifier of the folder String
Name name of the folder String
Description description of the folder String
Version Tag version tag of the file (tracked for users with premium accounts only) String
Sequence ID unique identifier for use with events endpoint String
Size size of the folder String
Path path to the folder (for example, /pictures/2016/4/...) list of text
Time Created timestamp when the folder was created String
Time Modified timestamp when the folder was last modified String
Time Trashed timestamp when the folder was last trashed String
Time Purged timestamp when the folder was deleted permanently String
Time Content Created timestamp when the file in the folder was originally created String
Time Content Modified timestamp when the content of the folder was modified String
Folder Upload Email email address that can receive files for upload to this folder String
Parent Folder Name name of the parent folder String
Parent Folder ID unique identifier of the parent folder String
Item Status status of the folder (active, trashed, or deleted) String
Share Link
Share URL URL to view the folder String
Download URL URL to download the folder String
Vanity URL customizable URL String
Password Enabled indicator of password String
Time Unshared timestamp when the folder was unshared String
Download Count number of times the folder was downloaded String
Access Level accessibility of the share link (for example, Edit or View) String
Can Download indicator of downloadable content String
Can Preview indicator of viewable content String
Created By
Name name of the user who created the folder String
User ID String
Email email address of the user String
Modified By
Name name of the user who most recently modified the folder String
User ID unique identifier of the user String
Email email address of the user String
Owned By
Name name of the user who owns the folder String
User ID unique identifier of the owner String
Email email address of the owner String
Context
Execution ID unique identifier associated with the execution of the flow String
