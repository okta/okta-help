Search Users

Search users in group using the defined criteria.

Options

Field Definition Type Required
Accounts Choose from available accounts. Dropdown TRUE
Result Set Choose whether the search returns First Matching Record or All Matching Records. Dropdown TRUE

Input

Field Definition Type Required
Query
Email Email address of the user. String FALSE
Group ID Unique ID of the group to which the user belongs. String FALSE
Login Status Indicates whether the user is currently logged in or not. String FALSE
Status Status of the user's account, choose from Activation Required, Activation Sent, Active, Closed, or Disabled. Dropdown FALSE
Username Substring Substring of the username. String FALSE

Output

Field Definition Type
Username Username of the user. String
User ID String
User Type Type of the user. String
Is Admin When true, the user's feature set is actively set as part of the plan. String
User Status Status of the user's account, can be of one of these: ActivationRequired, ActivationSent, Active, Closed, or Disabled. Dropdown
URI URI containing the user ID. String
Email Email address of the user. String
Created Date Time Date and the time when the user was created in . String
User Added to Account Date Time Date and the time when the user was added to the account. String
First Name First name of the user. String
Last Name Last name of the user. String
Job Title Job title of the user. String
Company Name of the company with which the user is employed. String
Permission Profile ID Unique ID of the permission profile associated with the user, can be one of these: 2301416 (for the Viewer profile), 2301415 (for the DocuSign Sender profile), or 2301414 (for the Account Administrator profile). String
Permission Profile Name Name of the permission profile associated with the user. String
