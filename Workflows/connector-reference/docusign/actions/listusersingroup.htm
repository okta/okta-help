List Users in Group

Retrieve a list of users in a group.

Options

Field Definition Type Required
Accounts Choose from available accounts. Dropdown TRUE

Input

Field Definition Type Required
Group ID Unique ID of the group. String TRUE

Output

Field Definition Type
Users
Username Username of the user. String
User ID String
User Type Type of the user. String
User Status Status of the user's account; can be of one of these: ActivationRequired, ActivationSent, Active, Closed, or Disabled. String
URI URI containing the user ID. String
Email Email address of the user. String
Login Status Indicates whether the user is currently logged in or not. String
Send Activation Email When true, an activation email is to be sent. String
Activation Access Code Access code provided to the user to activate the account. String
