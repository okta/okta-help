Upload Document to Envelope

Upload a document to an existing envelope.

Options

Field Definition Type Required
Accounts Choose from available accounts. Dropdown TRUE

Input

Field Definition Type Required
Envelope ID Unique ID of the envelope. String TRUE
File Content File content. String TRUE
File Name String TRUE
File Extension File extension. String TRUE

The size of the file can't exceed 9 MB. Larger files may cause this action card to produce errors and cause your flow to fail.

Output

Field Definition Type
Envelope ID Unique ID of the envelope. String
Document ID Unique ID of the document. String
