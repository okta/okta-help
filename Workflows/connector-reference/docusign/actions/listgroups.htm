List Groups

Retrieve a list of groups in a account.

Options

Field Definition Type Required
Accounts Choose from available accounts. Dropdown TRUE

Input

This card doesn't require any inputs.

Output

Field Definition Type
Groups
Group ID Unique ID of the group. String
Group Name Name of the group. String
Group Type Type of group; can be one of these: adminstrators, everyone, customGroup, or sharedSigningGroup. String
Permission Profile ID Unique ID of the permission profile associated with the group, can be one of these: 2301416 (for the Viewer profile), 2301415 (for the DocuSign Sender profile), or 2301414 (for the Account Administrator profile). String
User Count Total number of users in the group. String
