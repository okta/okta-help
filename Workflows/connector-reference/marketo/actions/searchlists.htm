Search Lists

Search one or more lists in .

Options

Field Definition Type Required
Result Set Choose from whether the search applies to the First Matching List or All Matching Lists. Dropdown TRUE

Input

Field Definition Type Required
List Names Names of lists. List FALSE
Program Names Names of programs associated with the lists. List FALSE
Workspace Names Names of specific workspace groups, often used for international purposes, see API Docs. List FALSE

Output

Field Definition Type
List ID Identification number associated with list. Number
Name Name/title of the program. String
Description Description of the program, if any, maximum 2000 characters. String
Program Name Name of the program the list is in. String
Workspace Name Name of the workspace the list is shared with. String
Created At Date & Time
Updated At Most recent date when the campaign was updated. Date & Time
