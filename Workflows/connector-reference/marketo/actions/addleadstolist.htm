Add Leads to List

Add leads to a list by using lead IDs.

Input

Field Definition Type Required
List ID Identification code associated with the list to which you want to add the leads. The list ID is a 4-6 digit number which can be found within the URL after the #SL. To find a list ID, open the target list in . Number FALSE
Lead IDs Identification code of the leads you wish to add to the list. You can find a lead's ID by using the Search Lead action in . Note: you can add a maximum of 300 leads at a time. List FALSE

Output

Field Definition Type
Leads
Lead ID Identification number of the leads. Number
Status Status of the added leads. String
