Remove User from Group

Remove a user from a group in Atlassian.

Input

Field Definition Type Required
Group
Group ID Group's unique identifier. String TRUE
User
User ID String FALSE

Output

Field Definition Type
ID Unique identifier of the newly created group. String
External ID String
Display Name Name of the group that is visible to other team members. String
Members
Resource Type Resource type of the group member. String
ID Unique identifier of the group member. String
Username Login username of the group member. String
Ref Reference ID of the group member. String
Meta
Resource Type Resource type of the group. String
Location Geographic designation for the group. String
Created String
Last Modified Date when the group was modified. String
