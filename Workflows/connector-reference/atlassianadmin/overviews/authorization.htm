Authorization

When you add an Atlassian card to a flow for the first time, you'll be prompted to configure the connection. This will enable you to connect your Atlassian account, save your account information, and reuse the connection in new flows that include Atlassian.

Tip

You can create multiple connections and manage them from your Connections page.

To create a new connection from an Action card:

  1. Click New Connection.

  2. Enter a Connection Nickname. This is useful if you plan to create multiple Atlassian connections to share with your team.

  3. Enter a value in the API Token and Directory ID fields. To learn more about enabling Atlassian SCIM API integration in Okta to obtain those values, see Configure user provisioning with Okta.

  4. Click Create.
