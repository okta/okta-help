Download Attachment

Download an attachment from an email. For additional information, refer to documentation.

Input

Field Definition Type Required
Download
File Name Name of the attachment. String FALSE
Attachment ID Unique ID of the attachment. String TRUE
Message ID Unique ID of the message containing the attachment. String TRUE

To find message and attachment IDs, open the message and click More (to the right of Reply). In the drop-down menu, select Show original.

Output

Field Definition Type
Attachment
File Content Content of the downloaded attachment. File
File Name Name of the downloaded attachment. String
Size Size of the attachment. String
File type Type of attachment (for example, .png, .jpg, or .pdf). String
