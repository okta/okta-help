Authorization

When you add a card to a flow for the first time, prompts you to configure the connection. This connection links to your account and saves your account information, so you can reuse the connection for future flows.

Paid accounts have a rate limit of 10 million complexity points per minute. Free and trial accounts are limited to 1 million complexity points per minute. See Rate Limits for details on the complexity points structure.

Prerequisites

You need the following elements to create a connection to account:

  • administrator authority

  • A administrator account

  • A  API key. The API key is found on the Administration page within your admin account. See Accesing API tokens.

Create connection

To create a connection from an action card, complete the following steps:

  1. Click New Connection.
  2. Enter a Connection Nickname. This is useful if you plan to create multiple connections to share with your team.
  3. Enter the API Token information.
  4. Click Create.
