Custom API Action

Make a custom, authenticated HTTP call to the API.

Options

Input

Field Definition Type Required
Relative URL URL address on the web server to which you're attempting to interact. Specify the relative URL as /{insert_remaining_URL}.

You can specify query parameters in the relative URL following a ? character, or specify the query parameters as an object pair in the Query input.

For example, for the API endpoint https://example-org.us.auth0.com/api/v2/users/{userid}, the relative URL is /api/v2/users/{userid}.

 Text TRUE
Query Specify any additional query parameters that should be included in object format as key value pairs. Object FALSE
Headers Specify any headers required in addition to authorization or content-type, because this connector already handles these. Object FALSE
Body Specify a request body in JSON format. Not available for GET requests. Object FALSE

Output
