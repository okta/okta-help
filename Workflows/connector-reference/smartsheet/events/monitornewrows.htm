New Row - Webhook

Trigger a flow when a new row is created in a sheet. Waits to receive an incoming request from that contains information on newly created rows. This request occurs whenever a new row is created within a sheet.

Options

Field Definition Type Required
Your Smartsheet sheets Select from the list of all available sheets associated with your instance. Dropdown TRUE

Output

Field Definition Type
Row
The fields in this section are dynamically generated based on the columns available in the selected sheet. Fields are assigned according to the names of the columns within the sheet.
Metadata
id Row ID within . String
rowNumber Row number within the sheet. String
createdAt Time when a new row was created. String
modifiedAt Time when a new row was modified. String
