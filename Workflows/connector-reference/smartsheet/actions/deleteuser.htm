Delete User

Remove a user from an organization account.

A removed user is transitioned to a free collaborator with read-only access for the following items: owned dashboards, reports, sheets, workspaces, and any shared templates (unless those are optionally transferred to another user).

Options

Field Definition Type Required
Remove From Sharing? Choose one of the following options:

  • No: Empty fields are not updated.
