Remove User from Organization

Remove a user from an organization in .

Options

Field Definition Type Required

Transfer Ownership?

Choose one of the following options:
  • Yes: Transfer groups or workspaces to another account if the user being removed owns the groups or workspaces. Allows the transfer of sheets to another account.
  • No: Do not transfer groups, workspaces or sheets to another account.
 Dropdown TRUE

Input

Field Definition Type Required
User
ID ID of the user. String TRUE
Transfer Ownership      
User ID User ID.

User must have group admin privileges in order to view transferred groups.

 String TRUE
Transfer Sheets?

Choose one of the following options:

  • Yes: Transfer the removed user's sheets to another user.
  • No: Do not transfer the removed user's sheets to another user.

This field appears when Yes is selected from the Transfer Ownership? option.

 Boolean FALSE
Remove From Sharing? True if the user was removed from sharing all sheets and workspaces in the organization account; otherwise, False. Boolean FALSE

Output

Field Definition Type
Result
Group Transferred Number of groups that are transferred. Number
Sheet Transferred Number of user sheets that are transferred. Number
Workspaces Transferred Number of workspaces that are transferred. Number
Sheet Removed From Sharing Number of sheets removed from sharing. Number
Workspaces Removed From Sharing Number of workspaces removed from sharing. Number
