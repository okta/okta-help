Comment on Sheet

Adds a comment to a sheet.

Options

  • Your sheets (dropdown list): A list of all available sheets associated with your account or "-- Insert ID --" if you want to provide an ID to a sheet at runtime.

Input

  • Comment (text): The text of the comment

  • Sheet ID(text): The ID of the sheet you want to comment on. (This field only appears if '-- Insert ID --' was selected in the 'Your Sheets' dropdown menu in the Options tab)

Output

  • Comment ID(text): The ID of the resulting comment. You'll need this ID to do something else with the comment, such as Attach URL to Comment.
