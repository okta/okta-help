Attach File to Comment

Attach a file to a comment using the comment ID.

Required fields are indicated in red.

Options

  • Your sheets (dropdown): Select from the list of all available sheets associated with your instance. Choose Insert ID to select a sheet by its ID, which is entered in the Input section.

Input

The input fields for this card are dynamically generated based on your instance.

Sheet

  • ID (text): The unique identifier of the sheet you want to use.

Comment

  • ID (text): The unique identifier of the comment to which the file will be attached.

File

  • File name (text):  The name of the file to be attached.

  • File content (file): The contents of the file to be attached.

Output

Attachment

  • ID (text): The unique identifier of the attached file.
