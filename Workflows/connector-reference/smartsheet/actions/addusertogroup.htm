Add User to Group

Adds a user to a group in Smartsheet.

Input

Field Definition Type Required
Group
ID Unique ID of the group. String TRUE
User      
Email Primary email address of the current user. String TRUE

Output

Field Definition Type
Result
Operation Status Message that indicates the outcome of the request:

  • SUCCESS

  • PARTIAL_SUCCESS

 String
Failed Items List of objects that represents the items that failed to be added or updated. List of Objects
