Search Buckets

Search buckets owned by the authenticated user in AWS S3.

There is a default limit of 100 buckets, and an upper limit of 1000 buckets per account.

Options

Field Definition Type Required
Result Set Choose a method to filter search results:
  • First Matching Record: returns the first record that matches.
  • All Matching Records: returns all matching records.
