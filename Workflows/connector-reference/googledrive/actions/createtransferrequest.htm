Create Transfer Request

Create a request to transfer files and folders from one user to another within a domain.

Input

Field Definition Type Required
From User Email of the user from whom the files and folders are being transferred. String TRUE
To User Email of the user to whom files and folders are being transferred. String TRUE

Output

Field Definition Type
Transfer ID Unique ID of the transfer. String
Transfer Status Status of the transfer; can be completed, failed, or inProgress. String
Request Time Time when the transfer was requested. Date & Time
