Search Files

Search for files by parameters.

Options

Field Definition Type Required
Result Set All files (default) returns all files in the search results; First file returns the first file that matches the search criteria. Dropdown TRUE
Parent Folder Target parent folder for the file search. Dropdown TRUE

Input

Not all of the fields are mandatory. Most often the search will only contain the Name field. But if you want to limit the search to more specific locations, then use more inputs to limit the scope of the search.

Field Definition Type Required
Name Name of the file. String FALSE
Starred
  • True: A folder has a "Starred" label applied to it.
  • False: A folder does not have a "Starred" label applied to it
 Boolean FALSE
Trashed
  • True: A folder has a "Trashed" label applied to it.
  • False: A folder does not have a "Trashed" label applied to it
 Boolean FALSE
Parent Folder ID Unique ID of the parent folder. Dropdown FALSE
Owners Name of the persons who own the file. String FALSE
Writers List of names of people who have edit access to the file. String FALSE
Readers List of names of people who have viewer access to the file. String FALSE
Commenters List of names of people who have commenter access to the file. String FALSE

If you don't select a search criteria or leave input fields empty, the search results will contain all of the existing files in your instance.

Output

Field Definition Type
Files
File Name Name of the files matching searched criteria. String
File ID Unique ID of the files matching searched criteria. String
