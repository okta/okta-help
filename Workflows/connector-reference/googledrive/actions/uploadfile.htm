Upload File

Upload a file to .

Input

Field Definition Type Required
Name Name of the file. String TRUE
Parent Folder ID ID of the folder to which you want to upload the file; leave empty for root. String FALSE
Description Description of the file. String FALSE
File Content Unique ID associated with the file content in the Workflows file system. Object TRUE
Convert to Google format? Indicates whether to convert the file to Google format. Boolean FALSE

A file must be at least 1 byte in size, but cannot be larger than 2 GB. See Workflows system limits.

Only files that are already stored in the Workflows platform file system can be uploaded. Local files cannot be uploaded to using this action card.

Output

Field Definition Type
File Name String
File ID Unique ID of the file. String
