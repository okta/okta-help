Read User

Retrieves a user profile within your instance.

Options

Field Definition Type Required
Field Type Options are Email, User ID, or External ID. Dropdown TRUE

Input

Field Definition Type Required
User
Email

Email address of the updated user.

This field appears when you choose Email for the Field Type option.

 String TRUE
ID

This field appears when you choose ID for the Field Type option.

 String TRUE
External ID

This field appears when you choose External ID for the Field Type option.

 String TRUE

Output

Field Definition Type
ID String
Account ID Unique identifier of the account associated with the user profile. String
Member Folder ID Unique identifier of the user's folder within a team space. String
Email Email address associated with the user profile. String
Secondary Emails Additional email addresses that are associated with the user profile. String
First Name User's first name. String
Last Name User's last name. String
Display Name User's name that is visible to other team members. The display name may be different than the user's full name. String
External ID String
Status Status of the user's Dropbox account. String
Verified Email? Indicates whether the email address associated with the user profile has been confirmed. Boolean
Role Indicates the type of admin role assigned to the user profile. String
Invited On Date on which an invitation email was sent to the user. String
Joined On Date on which the user created a Dropbox account. String
Membership Type Type of account associated with the user. String
Groups Team groups with which the user profile is associated. List of Text
