Create Group

Creates an empty group within your instance.

Input

Field Definition Type Required
Name Name for the new group. String TRUE
External ID String FALSE
Add Creator As Owner? Specifies if you want to add the group creator as the owner for the group. Dropdown TRUE
Management Type Dropdown TRUE

Output

Field Definition Type
Name Name for the new group. String
User ID String
External ID String
Management Type Management type of the group. String
Created At Timestamp for the group creation. Date & Time
Member Count Number of members in the group Number
Members Information for group members List of Objects
