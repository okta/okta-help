Read Incident

Read an incident's information based on its ServiceNow System ID.

Options

Field Definition Type Required
Display
  • System Values: unique identifier associated with the full name. For example, "46b87022a9fe198101a78787e40d7547" would be the database's version of "Joe Smith."
  • Display Values: value shown in the ServiceNow UI. For example, a full name like "Joe Smith."
 Dropdown TRUE

Input

Field Definition Type Required
System ID Unique identifier of the incident. The system ID can be obtained by searching for an incident with our Search Incidents card or in ServiceNow itself. String TRUE

Output

Field Definition Type
Incident
Dynamically Generated Fields Fields are dynamically generated based on fields available to the account selected. This includes standard fields and custom fields, if needed. Click the cog wheel in the bottom right of the card in order to look at the full list of output options. String
