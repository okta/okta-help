Authorize your account

When you add a card to a for the first time, you need to set up a configuration for that connector. Setting up a configuration will allow you to connect to your  account, save the information, and reuse that configuration next time you build a with the connector. You won't have to sign in every time you want to build a with this connector. After you've set up one configuration, you can choose to use that configuration again, or set up a new one.
