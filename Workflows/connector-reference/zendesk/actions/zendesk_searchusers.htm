Search Users

Search for one or more users.

Options

Field Definition Type Required
Return Choose from Single user and All matching users.

Single user - If selected, this card returns the first user that matches your search. Use this option when you're looking for a specific user.

All matching users - If selected, this card returns a collection of users that match your search. Use this option when you're looking for a filtered set of users.
