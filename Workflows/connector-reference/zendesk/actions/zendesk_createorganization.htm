Create Organization

Create a new organization on Zendesk.

Input

Field Definition Type Required
Organization
External ID String FALSE
Name String FALSE
Domain Names Drop down menu of possible domain names to associate with organization. String FALSE
Details Any details you want to store about the user, such as an address. String FALSE
Notes Any notes you want to store about the user. String FALSE
Group ID Identification number of group organization is associated with. String FALSE
Shared Tickets? If true is marked, tickets are not shared. String FALSE
Shared Comments true if the comments within organization are shared. String FALSE
Tags Drop down menu of tags related to organization. String FALSE

Output

Field Definition Type
Membership
Organization ID Identification code of organization user is associated with. Number
