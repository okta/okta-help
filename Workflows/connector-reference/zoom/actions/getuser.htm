Get User

Get detailed information of a user.

Input

Field Definition Type Required
User
User ID String TRUE

Output

Field Definition Type
Profile
User ID String
Username Sign-in username of the user. String
Primary Email Primary email address associated with the user. String
First Name User's first name. String
Last Name User's last name. String
Display Name Name of the user that is visible to other team members; the display name may be different than the user's full name. String
Department Name of user's department. String
User Type Type of user. See Account Types. String
Basic User without a paid license. String
Pro User with a paid account who can host unlimited meetings on the public cloud. String
Corp User with paid account who can host unlimited meetings with the on-premises meeting connector. String
Role Indicates the type of admin role assigned to the user profile. String
Active? True if the user account is active; False, otherwise. Boolean
Login Using SSO? True, if the new user will access using single sign-on; False, otherwise. Boolean
Login Using Work Email? True if the new user will access using a work email address; False, otherwise. Boolean
