Create Organization Sharing Link

Create an organization or anonymous sharing link for a drive item in .

Options

Field Definition Type Required
Source Dropdown TRUE
Scope Choose a type of scope for the sharing link:

  • Organization: Anyone signed into your organization (tenant) can use the link to get access. Only available in for Business.

  • Anonymous: Anyone with the link has access, without needing to sign in. This may include people outside of your organization. Anonymous link support may be disabled by an administrator.

 DropDown TRUE

Type Password of the sharing link that is set by the creator. Possible values are:
  • View: creates a read-only link to the file or folder.
  • Edit: creates a read-write link to the file or folder.
  • Embed: creates an embeddable link to the file or folder. This option is only available for files in personal.
String TRUE

Additional information

By default, you cannot create an anonymous type sharing link for a file or folder in a group. You can only create an anonymous sharing link of file or folder in an organization.
