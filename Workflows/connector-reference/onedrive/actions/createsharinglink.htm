Input

Field Definition Type Required

 String TRUE

 String TRUE
User      
User ID Unique ID for the user object who owns the .

 String TRUE
Sharing Link      
File or Folder ID Unique ID of the file or folder. String TRUE
Type Password of the sharing link that is set by the creator. Possible values are:
  • View: creates a read-only link to the file or folder.
  • Edit: creates a read-write link to the file or folder.
  • Embed: creates an embeddable link to the file or folder. This option is only available for files in personal.
String TRUE
Scope Scope of the link to create. Possible values are :

  • Anonymous: anyone with the link has access, without needing to sign in. This may include people outside of your organization. Anonymous link support may be disabled by an administrator.

  • Organization: anyone signed into your organization (tenant) can use the link to get access.

String TRUE
Password Password of the sharing link that is set by the creator.

Appears when Drive ID, Group ID or User ID is selected from the Location option.

 String  
Expiration Date String with format of yyyy-MM-ddTHH:mm:ssZ of DateTime indicates the expiration time of the permission.

Appears when Drive ID, Group ID or User ID is selected from the Location option.

 Date and Time

Additional Information

To specify certain recipients of the shared link, use the Send Sharing Link card.

By default, you cannot create an anonymous type sharing link of file or folder in a group. You can only create an anonymous type sharing link of file or folder in an organization.
