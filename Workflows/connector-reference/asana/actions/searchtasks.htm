Search Tasks

Search for a task in your instance.

This action card is available for accounts with Premium access or higher.

Options

Field Definition Type Required
Organization Organizations in your workspace. Dropdown TRUE
Result Set First Matching Task: returns the first task that meets the search criteria.

All Matching Tasks: returns all tasks that match the search criteria.

 Dropdown TRUE

Input

Field Definition Type Required
Task
Text Option for full-text search on both a task name and description. String FALSE
Assignee Any Identifier of the user to whom a task is assigned. List of Text FALSE
Projects Any Identifier of the project that contains the task. List of Text FALSE
Project Sections Any Identifier for the section that contains a task. List of Text FALSE
Tags Any Identifier of the tag that is assigned to a task. List of Text FALSE
Teams Any Identifier of the team that is assigned to a task. List of Text FALSE
Followers Any Identifier of the follower of a task. List of Text FALSE
Created By Any Identifier of the creator of a task. List of Text FALSE
Assigned By Any Identifier of the assigner of the task. List of Text FALSE
Created At After Only returns tasks that are created after this time. Date & Time FALSE
Completed At After Only returns tasks that have been completed after this time. Date & Time FALSE
Modified At After Only returns tasks that have been modified after this time. Date & Time FALSE
Is Blocking? Indicates whether task has dependents. Boolean FALSE
Is Blocked? Indicates whether task has dependencies. Boolean FALSE
Has Attachment? Indicates whether task has attachments. Boolean FALSE
Completed? Indicates whether task is completed. Boolean FALSE
Sort By Key to sort tasks; options are due_date, created_at, completed_at, likes, or modified_at; default is modified_at. Dropdown FALSE

Output

A maximum of 100 tasks can be returned.

Field Definition Type
Tasks
Task ID Identifier for the new task. String
Task Name Name of the task. String
