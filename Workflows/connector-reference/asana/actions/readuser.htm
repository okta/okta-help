Read User

Read a user profile.

Input

Field Definition Type Required
User
User ID or Email Unique identifier or email address associated with the user; to use a user ID as input, you must have an Enterprise account. String TRUE

Output

Field Definition Type
Fields
User ID String
Name String
Email User's email address. String
Secondary Emails One or more of the user's secondary email addresses; values will be returned if you provided a Service Account Token when authorizing a connection for the connector. String
Workspaces Workspaces with which the user is associated. String
