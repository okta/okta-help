Delete Worksheet

Remove a single worksheet and all of its data.

Input

Field Definition Type Required
Spreadsheet ID Unique ID of the spreadsheet.

For example, the spreadsheet ID in the URL https://docs.google.com/spreadsheets/d/abc1234567/edit#gid=0 is abc1234567.

 String TRUE
Worksheet ID Worksheet's unique ID (identified by gid in the URL of the spreadsheet). String TRUE
