Read All Rows

Read all non-header rows from a worksheet and return them as a collection.

Options

Field Definition Type Required
Spreadsheet List of spreadsheets in your account.

Due to an API limitation, a maximum of 100 spreadsheets are returned in a search. To search for a specific spreadsheet, use the Enter Spreadsheet ID option.

 Dropdown TRUE
Worksheet List of worksheet tabs in the spreadsheet. Dropdown TRUE

Output

Field Definition Type
Rows Column values associated with the row List of objects
Row Number Location number of the row Number
Columns Cell content in the row (each cell represents a column) Object
