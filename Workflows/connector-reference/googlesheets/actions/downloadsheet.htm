Download sheet

Options

Field Definition Type Required
Spreadsheet List of spreadsheets in your account. Dropdown TRUE
File Format File format for your download (.xlsx, .ods, .pdf, or.csv format). Dropdown TRUE

Output

Field Definition Type
Size File size in kilobytes. String
Name File name. String
File Content Actual downloaded file (type determined by the format option selected). String
