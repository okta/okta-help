Create Team

Create a new team.

Make sure that the API key that's used to create the connection for this card is not team-specific. Otherwise, any input value in the Team ID field on this card will not override the team that's specified by a team-specific API key. It is recommended to create and manage API keys on the Opsgenie API key management page.

Input

Field Definition Type Required
Name Name of the new team. String TRUE
Description Description of the new team. String FALSE

Output

Field Definition Type
Team
ID Unique ID of the new team. String
Name Name of new team. String
