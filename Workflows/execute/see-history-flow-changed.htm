See execution history after data changes

When you click a record in the Execution History pane,  shows a view of the flow with the actual values used for that particular execution.

This view is available only if there's accessible data for a record. The save data feature must have been active when the flow executed. See Save flow data. Also, you can't see data that removed after the 30-day retention limit.

However, if the flow itself has changed since that prior execution, the values may no longer show correctly:

  • There will be no values shown for any cards or fields added after the execution.

  • For any cards that were moved, the values show in the new position.

  • No values appear for any fields or cards that were deleted.
