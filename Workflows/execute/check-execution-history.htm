Check execution history

You can check on a flow after it has run and view every input and output field value for every stage in your flow. By doing so, you can confirm that a flow is running properly and to determine where an error may be occurring.

Data for an executed flow is available to view for 30 days.

To use this feature, open a flow that has run and click on the Flow History tab. Click on an item in the list displayed in the right pane to see the data that passed through the flow for a particular execution.

Since flows can be used to process personally identifiable information, you must opt in to saving Flow History. Flow History can be enabled in two ways:

  1. Check the Save all data checkbox in the Save dialog. This can be accessed when first saving the flow or by reopening the dialog by clicking on the flow's name.

  2. Click on the Enable Save Data link in the right pane of the Flow History page.

To stop saving Flow History for a flow, open the Save dialog, click the flow name and uncheck the Save all data checkbox.

To clear data from a flow history, see Clear All Data in Flow History.
