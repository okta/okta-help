Save flow data

If you enable the save data feature, saves the data that passes through your flow each time the flow executes. See Save flow data.

Review flow results

After enabling the save data feature for a flow, you can see the values for all input and output fields at every step.

The Execution History page shows you a list of all previous executions of the flow. When you select any record, the page shows the data from that execution. This helps to confirm that a flow is running properly or can show you where an error is occurring. The data remains available for 30 days.
