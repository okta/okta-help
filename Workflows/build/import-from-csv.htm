Import data from a CSV file into a table

You can import data into a table from a CSV file.

  1. Define the columns for a table.

  2. Click the Import button to bring up the Import Data dialog.

  3. Drag/drop a CSV file into the dialog or click Choose CSV File.

  4. Select a table column to which you want to map each CSV field. If the names match, that table column will be the default suggestion.

  5. Click Import.
