Configure a connection

When you first add a card to a flow, you need to select or create a connection to authenticate against that third-party API.

After your remote account connection is configured, saves your account information, so you can reuse the connection when you build any other flow that uses the same connector.

You can create and save multiple connection configurations and share them with other Workflow admins in your org.

New connection

To set up a new connection configuration, complete these steps:

  1. In the Account Nickname field, enter a value for this specific connection. This should be unique, so if you're connecting multiple accounts to the same external application or service, you can tell them apart.

  2. Enter any necessary credentials. This may be a specific domain, an API token, or a paired access key and secret. The security requirements vary depending on the external application or service.

  3. Click Create to complete the configuration. If required, prompts you to enter your sign in credentials for the destination site.

For existing connections, see View the Connections page.
