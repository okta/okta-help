Step 0: Before you begin

  1. Create an Okta user profile. The user must have a first name, last name, email address, and department (Sales, Marketing, or Operations). These attributes are case-sensitive.

  2. Configure a Salesforce application in Okta. If you do not have a Salesforce account, set up a Developer account here. For this template, we recommend that you disable Provisioning (the application is only used for Sign-On).

  3. Authorize your connections:

    • To authorize an Okta connection, see Authorization.
    • To authorize a Salesforce connection, see Authorization.
