Step 3: Connect the flows

  1. Return to the parent flow, and in the For Each card, click Choose Flow.

  2. Select Import Users - Helper Flow.

  3. In the Rows drop-down menu, select Item.

  4. Save the parent flow and all of its data. Note that you can't save a parent flow that contains a helper flow unless that helper flow is active.

  5. Toggle the Flow is OFF switch to ON.
