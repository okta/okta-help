Step 2: Add a helper flow

  1. In the Workflows console, click New Flow.

  2. Click Add Event, and then select Helper Flow.

  3. In the Helper Flow card, add the following Inputs to this flow:

    • id
    • type
    • price

  4. Click Add Another Function Text Compose .

  5. In the Compose card, enter id, and then drag and drop the id field from the Helper Flow card. Drop down a line, and then repeat for type and price.

  6. Save and name your helper flow.

  7. Toggle theFlow is OFF switch to ON.
