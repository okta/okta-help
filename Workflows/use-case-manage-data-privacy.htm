Manage data privacy requests

Summary

Problem: Under GDPR, CCPA, or other privacy regimes, customers have a right to have their data retrieved or deleted (or both) by service providers. These requests are typically highly manual and time-consuming, and failure to comply with them can lead to steep penalties.

Solution: Tools like DataGrail can automate the process of discovering and retrieving customer data. Okta Workflows can call out to these tools based on identity events, such as user inactivity or a user being in a particular group.

Applications: . For the full list of available Workflows connectors, see Connectors.

Sample Flow 1

API Request → Data Download Request → Get Data Download Details → Log in Table

Search users → Filter by last login > 120 days → Data Download Request
