Create and edit connector settings

Manage the basics of the branding for your cards on the Settings page.

  1. From the Overview tab, go to the Connector Settings page for your connector, and in the Connector Name field, enter a name for your connector.

    This will usually be the name of your service (for example, Salesforce, Slack, or ServiceNow). This will appear in the in-product Workflows catalog.

    This will appear in both the public OIN catalog and in-product Workflows catalogs.

  2. Note the string in the Key field. The key value is generated automatically and cannot be changed. The value is based on the name of your project.

  3. In the Key field, add a text string that will serve as an ID for your connector. The string value must remain unchanged, and it will be viewed by Workflows users.

  4. In the Description field, add a description for your connector.

    The description field may be used in the public catalog, as well as in-product search indexing.

    Limit the length of a description to two sentences. The description of the connector will be added to the connectors catalog and be subject to in-product search indexing.

  5. Click Upload to add an icon image for your connector.

    An image must be not larger than 300px x 300px transparent .png or .jpg file with a maximum size of 3MB. The image must be a square.

  6. In the Card Header Color field, add a 3 or 6 character hexadecimal color representation.

    For example: #fc0 or #00297A including the number sign.

  7. In the API Documentation field, add a link to the documentation for your connector's APIs.

  8. In the User Documentation URL field, add a link to a site that is hosting the user documentation for your connector's authorization settings as well as content to support the use of each action card. The link to the site will be displayed when a user clicks the help icon on any card.

  9. In the Support Email Address field, add an email address for additional support when using your connector. The email address will be displayed when a user clicks the help icon on any card.
