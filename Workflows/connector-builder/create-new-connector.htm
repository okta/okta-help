Create a connector

Create and store connectors as connector projects in .

Before creating your connector, consider which org you need it for. After you create the connector, it can't be moved between orgs.

To create a connector:

  1. Click New Connector. Or, if you already have existing connectors, click Connectors + in the upper left corner of to add another.

  2. Add a name and an optional description for the connector project in the New Connector Name dialog.

    The platform generates an immutable key for your specific connector. For example, if you name a connector project Google Docs, then generates a unique key such as googledocs_RHP4567. adds this randomly generated string to your connector name to ensure that the connector is unique across all environments.

    You can see the key in the Settings section of the page.

The connector appears in the left side of your page, which has four tabs:

  • Overview - Settings and Authentication

  • Flows

  • Test Connections

  • Deployment
