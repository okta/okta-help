Add static values for a dropdown options field

Add an options dropdown composed of a set of static values.

  1. In the dialog, add string values to the Group Name and Label fields.

  2. Select Dropdown from the Field Type field.

  3. Select Text or Number from the Data Type field.

  4. Select Static Values for the Dropdown values field.

  5. Add Option and Value pairings. For example, if you wanted to display language options, these can be possible pairings:

  • option: English, value: en_US

  • option: Chinese (Simplified), value: zh_CN

  • option: Danish, value: da

  1. Specify which pairing will display as the default in the dropdown, or select No default option.

  2. Add one or more additional dropdown fields, as required.

  3. Click Save.
