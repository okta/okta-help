アクティベーションリンクまたは秘密キーを使ってAndroidデバイスのOkta Verifyをセットアップする

デバイスでQR コードをスキャンしない場合、使用デバイスの電子メールまたはSMSアプリに送信されたアクティベーションリンクを使用してOkta Verifyをセットアップできます。また、秘密キーを使ってOkta Verifyをアクティベーションすることもできます。

開始する前に

  • ソフトウェアの適切なバージョンを使用していることを確認します。「サポートされているプラットフォーム、ブラウザー、オペレーティングシステム」をご覧ください。
  • 一部の組織では、モバイルデバイスで組織にサインインしてOkta Verifyアカウントをセットアップできます。詳細については、「アカウントにサインインしてAndroidデバイスでOkta Verifyをセットアップする」をご覧ください。
  • 管理者からのOktaようこそ/アクティベーションの電子メールを開きます。この電子メールには、組織のOktaのサインインページへのリンクおよびユーザー名（通常は電子メールアドレス）が含まれています。アカウントをセットアップするにはこの情報が必要となります。このメールを受信していない場合、管理者にご連絡ください。
  • 重要： 秘密キーを使ってOkta Verifyをセットアップした後、プッシュ通知で認証することはできません。代わりに、Okta Verifyで生成された6桁コードを使って、アプリのアカウントにサインインする必要があります。

このタスクを開始する

  1. ご使用のコンピューターで開始します。ウェブブラウザーでOktaのようこそ電子メールを開いて、組織のOktaサインインURLをクリックします。 サインインURLが分からない場合、管理者にお問い合わせください。
  2. ユーザー名（電子メールアドレス）およびパスワードを入力し、[Sign In (サインイン)]をクリックします。

    組織でOkta Verifyを必要とする場合、セットアップするよう求められます。IT部門の構成に基づき、[Set up multifactor authentication (多要素認証をセットアップする)]または[Set up authenticators (認証子をセットアップする)]のいずれか1つのページが開きます。

  3. [Setup (セットアップ)]をクリックします。
  4. デバイスとしてAndroidを選択して、[Next (次へ)]をクリックします。
  5. [Scan barcode (バーコードをスキャンする)]ページで、QRコード下の[Can't scan? (スキャンできませんか？)]リンクをクリックします。

    管理者がアカウントをどのように設定するかにより、秘密キーを使うか、電子メールまたはデバイスのメッセージングアプリに送信されたアクティベーションリンクを使って、Okta Verifyに手動で登録することができます。

    重要： [Setup Okta Verify (Okta Verifyのセットアップ)]画面で[Next (次へ)]をクリックしないでください。

  6. ブラウザーにどのバージョンの画面が表示されるかによって、次を実行します。
    • 画面にドロップダウンメニューがある場合、最適なオプションを選択し、画面の手順に従います。
      • Send activation link via SMS（SMSを通してアクティベーションリンクを送信する）このオプションを選択する場合、携帯電話に送信されるメッセージ内でアクティベーションリンクを受け取ります。電話番号を入力し、[Send (送信)]をクリックします。
      • Send activation link via email（電子メールを通してアクティベーションリンクを送信する）このオプションを選択する場合、電子メールでアクティベーションリンクを受け取ります。このオプションを選択した後、[Send (送信)]をクリックします。

        Okta Verifyをセットアップする際、バーコードをスキャンする代わりに[Set up Okta Verify via email link (電子メールリンクを通してOkta Verifyをセットアップする)]オプションを選択する場合、主要電子メール（管理者からOktaの案内ノートを受信したことのあるアドレス）を入力します。その他の電子メールアドレスを入力すると、エラーが生成されます。

      • Set up manually without push notification（プッシュ通知なしで手動でセットアップする）このオプションを選択する場合、秘密キーを書き留めてください。後でデバイスのOkta Verifyアプリで秘密鍵を入力する必要があります。このオプションを選択した後、プッシュ通知を使用して認証することはできません。Okta Verifyによって生成されたコードを使用してのみ認証できます。
    • 画面にドロップダウンメニューがない場合、秘密キーを書き留めておきます。後の手順でデバイスのOkta Verifyアプリで秘密キーを入力する必要があります。

      重要： [Setup Okta Verify (Okta Verifyのセットアップ)]画面で[Next (次へ)]をクリックしないようご注意ください。

  7. デバイスでGoogle Play Store - Okta VerifyからOkta Verifyをダウンロードして、インストールします。
  8. アカウントをアクティベーションします。
    • [Send activation link via SMS（SMSを通してアクティベーションリンクを送信する）]オプションを選択する場合、デバイスのメッセージアプリを確認し、登録リンクをクリックします。Okta Verifyが開き、アカウントが正常にアクティベーションされたことを確認するメッセージが表示されます。
    • [Send activation link via email (電子メールを通してアクティベーションリンクを送信する)]オプションを選択する場合、デバイスの電子メールアプリを確認し、[Activate Okta Verify Push (Okta Verifyプッシュのアクティベーション)]ボタンをクリックします。Okta Verifyが開き、アカウントが正常にアクティベーションされたことを確認するメッセージが表示されます。
    • [Set up manually without push notification (プッシュ通知なしで手動でセットアップする)]オプションを選択する場合、Okta Verifyが開き、手順に従います。
      1. [Add Account (アカウントの追加)]をタップし、追加するアカウントを選択します。Okta Verifyで[Organization (組織)]を選択します。
        • Organization (組織)：認証プロバイダーとしてOktaを使用する仕事、学校、またはビジネス用に多要素認証（MFA）アカウントを追加します。
        • Other（その他）： Okta Verifyを使用して、銀行または証券口座などの個人のアカウント用に1回限りのパスワードを生成します。
      2. [Scan QR code (QR コードをスキャン)]をタップします。

      3. [Enter Key Manually (キーを手動で入力する)]をタップします。

      4. [Enter Key (キーの入力)]画面で認証情報を入力します。次に、[Add Account (アカウントを追加する)]をタップします。
        • Account Name（アカウント名）：案内メールで受け取ったユーザー名を入力します。
        • Key（秘密キー）： コンピューターに表示されている鍵を入力します手順6をご覧ください。

      5. コンピューターのブラウザーの[Setup Okta Verify (Okta Verifyをセットアップする)]ページで[Next (次へ)]をクリックします。

      6. ブラウザーの[Enter code displayed from the application (アプリから表示されたコードを入力する)]ページで、デバイスのOkta Verifyでアカウントの下に表示される数字を入力します。
      7. [Verify (確認)]をクリックします。これでデバイスが登録されました。

Oktaで保護されている組織のアプリにサインインする際の認証に、Okta Verifyを使用できます。
