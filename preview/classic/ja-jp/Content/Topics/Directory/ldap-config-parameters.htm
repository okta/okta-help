LDAP構成パラメーター

以下の表は、LDAP構成パラメーターであり、Okta LDAP エージェントをインストールした後に変更できるパラメーターを示しています。

下のコマンドを使用してOkta LDAP エージェントのパスワードを変更できます。

/opt/Okta/OktaLDAPAgent/scripts/update.sh [-w|--ldap-admin-password] "<NewLDAPPassword>"

ldapPort = 389非暗号化接続に対するLDAPサーバーのポート番号、インストール中に設定されます。はい*

ldapUseSSL = true暗号化または非暗号化接続の間で選択。trueの場合、エージェントは暗号化された接続を使用。インストール時のデフォルト オプションは、非暗号化接続の使用です。推奨されるオプションは、インストール時に安全な接続を使用することです。はい*
ldapAdminDN = cn=ldsadmin, cn=admins, dc=example, dc=net, dc=localエージェントがLDAPサーバーにバインドする際のユーザーの識別名。はい**

LDAP構成パラメーターのアップデート

LDAP パラメーターの構成値を変更するには、OktaLDAPAgent.confファイルで値をアップデートします。LDAPエージェントの構成ファイルを開いたり編集する前に、Windowsサービスの下でOkta LDAP エージェント サービスを停止します。OktaLDAPAgent.confファイルをアップデートして変更を保存した後、Okta LDAP エージェントを再起動して変更内容を有効化する必要があります。

Windows

Windows環境で、以下の場所でOktaLDAPAgent.confファイルを見つけます：C:\Program Files\Okta\Okta LDAP Agent\conf\OktaLDAPAgent.conf

Linux

Linux環境で、以下の場所でOktaLDAPAgent.confファイルを見つけます：/opt/Okta/OktaLDAPAgent/conf/OktaLDAPAgent.conf.
