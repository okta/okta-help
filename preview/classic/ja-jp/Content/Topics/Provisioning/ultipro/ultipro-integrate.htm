UltiProをOktaと統合する

  1. UltiProレポートとレポートIDを作成します。UltiProレポートとレポートIDを作成するを参照してください。

  2. UltiPro WebサービスのURLと、ユーザーおよび顧客のAPIキーを見つけて記録します。
    1. 管理者権限でUltiProインスタンスにサインインします。
    2. メニュー> [システム構成] をクリックし、[ セキュリティ]の下のWebサービス を選択します。
    3. User API Key および Customer API Key フィールドに表示されている値をコピーして、テキストエディタに貼り付けます。
    4. ビジネスインテリジェンスWebサービスエンドポイント の横に表示されるベースURLをコピーして、テキストエディターに貼り付けます。ベースURLは、表示されるすべてのフィールドに共通するURLの部分です。
  3. UltiProアプリをすでにOktaに追加している場合は、手順4に進みます。まだ追加していない場合は、今すぐ追加します。
    1. 管理コンソールで、[アプリケーション] > に移動します。［Applications（アプリケーション）］に移動します。
    2. ［Add Application（アプリケーションの追加）］をクリックします。
    3. Search フィールドに、UltiProと入力し、 UltiProを選択します。
    4. Select Add.
    5. ［General Settings（一般設定）］ページのフィールドに入力し、［Next（次へ）］をクリックします。
    6. ［Sign-On Options（サインオン オプション）］ペインの［Sign On Methods（サインオン方法）］で、サインオンのオプションを選択します。SAML 2.0を選択した場合、［View Setup Instructions（セットアップ手順を表示）］をクリックして指示に従います。
    7. ［Done（完了）］をクリックします。
  4. 管理コンソールで、［Applications (アプリケーション)］に進み、アプリケーションのリストから「UtilPro」を選択します。
  5. ［Provisioning（プロビジョニング）］タブをクリックし、［Configure API Integration（API統合を構成）］をクリックして［Enable APIIntegration（Okta API統合を有効化）］チェックボックスを選択します。
  6. 以下のフィールドに入力します。
    • WebサービスのベースURL：手順2 dでコピーした値を貼り付け、URLプロトコルをhttpからhttpsに変更します。たとえば、http：//Servicet.ultipro.comはhttps://Servicet.ultipro.comになります。

    • Admin Username（管理者ユーザー名）：UtilPro管理者のユーザー名を入力します。

    • Admin Password（管理者パスワード）：UtilPro管理者のパスワードを入力します。

    • ユーザーAPIキー：手順2でコピーした値を貼り付けますc。

    • カスタマーAPIキー：手順2でコピーした値を貼り付けますc。

    • Oktaレポートパス：次の形式を使用してUltiProレポートIDを貼り付けます： storeID（ "レポートID"）。UltiProレポートとレポートIDを作成するを参照してください。

    • ユーザー属性に基づいてグループを作成（一度設定すると変更できません）： なし、 組織レベル1、 組織レベル2、 組織レベル3、 を選択します組織レベル4、または カスタム。

      • 選択したユーザー属性は、UltiProレポートに含まれている必要があります。Oktaテンプレートには、組織レベル1、2、3、および4が含まれています。 カスタム を選択した場合は、[グループ]のカスタムユーザー属性に属性値を入力します。

      指定されたユーザー属性値ごとにグループが作成され、値が一致するインポートされたユーザーがそのグループに追加されます。

    • グループのカスタムユーザー属性：オプション。カスタムグループ属性を入力します。属性名とテーブル名は一致する必要があります。

    • Pre-Start Interval（雇用前インターバル）：（任意）従業員がアクティブとみなされる雇用開始日前の日数を入力します。

    • デフォルトの電話の国番号（USA、CAN、UKRなど）：オプション。電話番号の国番号プレフィックスを入力します。

    • インポートグループ：オプション。グループをインポートする場合はこのオプションを選択します。

  7. （任意）［Test API Credentials（API資格情報をテスト）］をクリックしてAPI統合をテストします。

  8. [保存]をクリックします。
