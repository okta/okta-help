Okta Hyperspaceエージェントのインストール

このタスクでは、Okta Hyperspaceエージェントをインストールします。

タスク

  • クライアント シークレットおよびIDの定義
  • エージェントのインストール

クライアント シークレットおよびIDの定義

  1. 組織の管理者は、OktaサポートにOkta Hyperspaceエージェントのダウンロード用リンクを要求する必要があります。
    エージェントのバージョン履歴については、「Okta Hyperspaceエージェント バージョン履歴」をご覧ください。
  2. Oktaテナントに管理者としてサインインします。
  3. Epic Hyperspace EPCS (MFA)アプリケーションに進みます。
  4. ［General（一般）］タブを選択します。
    このタブにはOkta URL、クライアントIDおよびエージェントのインストールに必要なクライアント シークレットが含まれています。
    これらの値は後の手順で必要になるため、このタブを開いたままにしてください。

    注

    プロンプトが現れたら、次のいずれかの形式でOkta URL を入力します：https://<<YOUR OKTA ORG>>.okta.comまたはhttps://<<YOUR OKTA ORG>>.oktapreview.com。

エージェントのインストール

対話型

エージェントをホストするマシンにエージェントをインタラクティブにインストールします。

  1. HyperSpaceエージェントのMSIファイルをダウンロードして解凍します。
  2. OktaHyperspaceAgent.msiを実行し、すべてのデフォルト設定を承認します。
  3. プロンプトが現れたら、 Epic Hyperspace EPCS (MFA)アプリケーションの［General（一般）］タブに表示されているようにURL、クライアントID、クライアント シークレットを入力します。

サイレント

  1. https://www.microsoft.com/en-us/download/details.aspx?id=48145に記載されているように、Microsoft Visual C++ 再配布可能ライブラリをダウンロードし、インストールします。
  2. HyperSpaceエージェントのMSIファイルをダウンロードして解凍します。
  3. コマンドプロンプトを開き、ディレクトリをMSIをダウンロードし解凍した場所に変更します。
  4. 以下のようなコマンドを使用してMSIインストーラーを実行します：
    msiexec /qb /log log.txt /i OktaHyperspaceAgent.msi CLIENT_ID="ClientID" CLIENT_SECRET="ClientSecret" OKTA_URL="Url"
    ここで、
    Url、ClientIdおよびClientSecretの値はEpic Hyperspace EPCS (MFA)アプリケーションの［General（一般）］タブに表示されているものです。


注

OktaにインストールされているバージョンEpic.Core.Authentication.dllが古い場合、EPIC library Epic.Core.Authentication.dllはOktaバージョンを新しいEPIC DLLに置換します。

プラグイン インストール ファイルのデフォルト位置は、通常はc:\Program Files (x86)\Okta\OktaHyperspaceLoginDevice\です。
