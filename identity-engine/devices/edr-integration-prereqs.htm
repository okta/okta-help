Endpoint security integration prerequisites

Verify the following prerequisites before integrating with your endpoint detection and response (EDR) solution:

  • org is enabled
  • is configured

    • See Configure [%=okta-feature-names.Okta FastPass%] .

  • EDR is enabled for your Okta org
  • Targeted devices are enrolled in (macOS devices must be registered and managed; Windows devices must be registered but don't need to be managed)

    • See Device registration.

  • version:
    • macOS client version 1.9.0 or later
    • Windows beta client version 1.3.100.3 or later
  • Device operating system:
    • Windows 10 (1709), 32 and 64-bit enrolled in (registered)
    • macOS 10.15 (Catalina) or later enrolled in (registered)
  • Endpoint security integration solution:
    • CrowdStrike: Falcon Agent 6.14 or higher (supported by for both macOS and Windows)
    • Windows Security Center ( for Windows only)
