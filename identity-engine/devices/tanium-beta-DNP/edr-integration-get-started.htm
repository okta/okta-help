Get started with endpoint security integrations

Complete the following tasks to integrate with your endpoint detection and response (EDR) solution:

  1. Review the system requirements. See Endpoint security integration system requirements.
  2. Ensure that targeted devices are enrolled in . See Device registration.

    • macOS devices must be registered and managed.

    • Windows devices must be registered but don't have to be managed.

  3. Ensure that is configured. See Configure Okta FastPass.
  4. Add an endpoint security integration.
  5. Create or edit an authentication policy that evaluates EDR signals. See Manage endpoint security integrations.
  6. Manage plugins to extend the functionality of the Okta endpoint security integrations. 
    • Manage endpoint security integration plugins for macOS.
    • Manage endpoint security integration plugins for Windows.
  7. Validate your endpoint security integration.
