Add an endpoint security integration

When you add an endpoint security integration, the server requests integration signals from the endpoint security client. The client collects the signals from the device plugins, and then sends a response back to the server. The server uses these signals to make app access decisions according to .

  2. Click the Endpoint security tab.

  3. Click Add endpoint integration.

  4. Select one of the following:

    • Windows Security Center

    • CrowdStrike

    • Tanium

    You can only add one integration at a time.

  5. If you selected CrowdStrike, select your platform: macOS, Windows, or both.

  6. If you selected Tanium, upload your organization's public key. Browse to the folder where you save the key and click Add. To learn how to download the certificate see the Tanium documentation.

  7. Click Add.
