Delete an endpoint security integration

  1. Locate any of your that are associated with an endpoint integration, and then remove the association.

    If an is associated with an endpoint integration, users wont' be able to access that app when the endpoint integration is removed.

    See Create an endpoint security integration [%=okta-feature-names.app sign-on policy%] .

  3. Click the Endpoint security tab.

  4. For the endpoint integration that you want to delete, click Actions Delete .

  5. Click Delete.
