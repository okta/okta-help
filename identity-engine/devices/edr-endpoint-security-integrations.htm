Manage endpoint security integrations

The Device integrations Endpoint security tab allows you to manage endpoint security integrations, such as endpoint detection and response (EDR) integration, with Windows Security Center and CrowdStrike. EDR integration extends device posture evaluation by enabling to capture signals collected by your EDR client running on the same device.
