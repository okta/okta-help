Device health attributes in System Log events

When users sign in with , the following System Log events provide device health details:

  • POLICY_EVALUATE_SIGN_ON
  • USER_AUTHENTICATION_VERIFY
  • USER_AUTHENTICATION_AUTH_VIA_MFA
  • USER_SESSION_START

The Device object shows device health signals such as Disk Encryption Type, Jailbreak, Screen Lock Type, and Secure Hardware Present.

You can use the device health data to analyze device security against device assurance policies.
