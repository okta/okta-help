Before you begin

Before you can enable the OTP authenticator, you need to configure the YubiKeys and generate a OTP secrets file (also known as the Seed File) using the . The OTP secrets file is a .csv that you upload into to activate the YubiKeys. See Programming YubiKeys for Okta Adaptive Multi-Factor Authentication for instructions. When you have finished generating the OTP secrets file, save it to a secure location.

Don't create a OTP secrets file manually. Only the Personalization Tool can populate the public and private key information for each . If this information is missing, the YubiKeys may not work properly.

After you've configured the YubiKeys and uploaded the OTP secrets file to , you can distribute the YubiKeys to your end users.
