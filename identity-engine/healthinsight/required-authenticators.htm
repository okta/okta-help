Set required authenticators for MFA enrollment policies

Enabling at least one required authenticator for your org ensures that end users assigned to a given policy are enrolled in multifactor authentication (MFA).

Once a required authenticator is set, you can also update your sign-on policy to prompt users to enroll in the authenticator the next time they sign in.

HealthInsight task recommendation

Set require authenticators to ensure that end users assigned to a given policy are enrolled in authenticators.

recommends

Require at least one authenticator in every MFA enrollment policy.
Security impact

Moderate
End-user impact

None

If an authenticator is required as part of the MFA enrollment policy, end users must enroll in the authenticator before they can sign in to their org. Setup varies depending on the authenticator specified.

Prompt an end user to enroll in a required authenticator

To prompt an end user to enroll in a required authenticator, you may do one of the following options:

  • Option 1:

    • Set a sign-on policy rule that prompts a user for authenticator enrollment.
    • Set a authenticator enrollment policy rule that allows a user to enroll in an authenticator when challenged for MFA.

  • Option 2: 

    • Set a factor enrollment policy rule that prompts the user to enroll in an authenticator the first time they sign in to their org.

Set a sign-on policy rule that prompts for authenticator enrollment

  2. Select the policy and from the list of associated rules, click Edit to start modifying an existing policy rule.
  3. Optional. You can also create a new rule.
  4. From the Edit Rule window, select Prompt for Authenticator.
  5. Click Update Rule to continue.

Set a factor enrollment policy rule that prompts a new user to enroll in a factor the first time they sign in to their org

  1. In the menu, click Security Multifactor .
  2. Click Factor Enrollment.
  3. Choose one of the active policy rules in the list and click Edit. The Edit Rule page appears.
  4. Under the condition THEN Enroll in multifactor, select the first time a user signs in.
  5. Click Update Rule to save your changes.
