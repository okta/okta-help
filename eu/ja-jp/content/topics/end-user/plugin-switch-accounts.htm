Okta Browser Pluginでアカウントを切り替える

Okta Browser PluginでAccount Chooserを使用すると、複数のOktaアカウントを容易に切り替えることができます。Okta Browser Pluginを介してアカウントに初めてアクセスしたときに、ブラウザーを信頼するか拒否するよう求められます。これを繰り返すうちに、プラグインを介してOktaアカウントのリストが構築され、さまざまなアプリに容易にアクセスできるようになります。

はじめに

Okta Browser Pluginがインストールされていることを確認します。ツールバーにOkta Browser Pluginアイコンが見つからない場合は、非表示になっている可能性があります。プラグインの非表示を解除するには、次の手順を完了します。

  • Chrome：Chromeメニュー（）をクリックし、［More Tools（その他のツール）］［Extensions（拡張機能）］に移動してOkta Browser Pluginが有効であることを確認します。プラグインが有効になっていることを確認したら、ブラウザーの右上部分にあるジグソーパズルアイコンをクリックし、Okta Browser Pluginの横にあるピンアイコンを選択します。
  • Firefox：Firefoxメニュー（）、［Customize（カスタマイズ）］の順にクリックし、Okta Browser Pluginの青いアイコンをブラウザーツールバーの希望する場所にドラッグします。
  • Internet Explorer：ツールメニュー（）で、［Manage add-ons（アドオンの管理）］［Okta toolbar（Oktaツールバー）］に移動し、［有効にする］をクリックします。
  • Safari：ブラウザーツールバーの任意の場所を右クリックし、［Customize Toolbar（ツールバーをカスタマイズする）］をクリックしてから、Your Apps（使用アプリ）アイコンをブラウザーツールバーの希望する場所にドラッグします。

この手順を開始する

  1. Oktaアカウントを信頼し、Account Chooserに追加する
  2. Account Chooserを表示し、アカウントを切り替える

Oktaアカウントを信頼し、Account Chooserに追加する

  1. すでにOktaにサインインしている状態で、このブラウザーで信頼 されていない、あるいはまだアクセスしていない別のOktaアカウントにサインインします。アカウントが信頼されると、Okta Browser Pluginはエンドユーザーをアカウントにサインインさせ、［Your Apps（使用アプリ）］リストにアカウントのアプリケーションが表示されるようになります。

    2. Okta End-User Dashboardにバナーが表示され、アカウントを信頼するかどうか指定するよう求められます。ダッシュボード上のアプリによっては、アカウントを信頼した後でのみアクセス可能になるものがあります。

  2. ブラウザーツールバーにあるOkta Browser Pluginの青いアイコンをクリックします。
  3. [Your Apps (使用アプリ)]メニューで[Trust / Reject (信頼する／拒否する)]オプションが表示された場合、該当するものを選択します。
    • [Trust (信頼する)]をクリックすると、赤色のバナーが消え、ダッシュボード上の残りのアプリがアクセス可能になり、そのアカウントがアカウントチューザーのリストに追加されます。
    • アカウントを信頼しない場合、またはプラグインによってアカウントにアクションが実行されないようにする場合、[Reject (拒否する)]をクリックします。後でアカウントを信頼する場合は、ページをリフレッシュするか、またはアカウントに再度サインインして[Trust (信頼する)]オプションを表示することができます。

      • 注：Chrome、Edge、Firefoxのブラウザーで、Okta Browser Pluginの青いアイコンの上に赤い四角が表示される場合、信頼または拒否のアクションが必要であることを示しています。

Account Chooserを表示し、アカウントを切り替える

このタスクはオプションです。Account Chooserで、Okta Browser Pluginに追加された複数のアカウント間の切り替えが容易に行えます。

  1. ブラウザーツールバーにあるOkta Browser Pluginの青いアイコンをクリックします。
  2. Account Chooser (アカウントチューザー)アイコンをクリックします。

    3. アクティブなアカウントには横に青いリボンがあります。

  3. リスト内で切り替えるアカウントをクリックします。
  4. リストからアカウントを削除するには、アイコンをクリックします。
