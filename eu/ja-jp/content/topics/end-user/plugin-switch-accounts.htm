Okta Browser Pluginでアカウントを切り替える

Okta Browser PluginでAccount Chooserを使用すると、複数のOktaアカウントを切り替えることができます。Okta Browser Pluginを介してアカウントに初めてアクセスすると、ブラウザーを信頼するか拒否するかの確認を求められます。これを繰り返すうちに、プラグインを介してOktaアカウントのリストが構築され、さまざまなアプリに容易にアクセスできるようになります。

はじめに

Okta Browser Pluginがインストールされていることを確認します。ツールバーにOkta Browser Pluginアイコンが見つからない場合は、非表示になっている可能性があります。プラグインを表示する手順は次のとおりです。

  • Chrome：Chromeメニュー（）をクリックし、［More Tools（その他のツール）］［Extensions（拡張機能）］に移動してOkta Browser Pluginが有効であることを確認します。プラグインが有効であることを確認したら、ブラウザーの右上部分にあるパズルアイコンをクリックし、Okta Browser Pluginの横にあるピンアイコンを選択します。
  • Firefox：Firefoxメニュー（）をクリックして［Customize（カスタマイズ）］をクリックし、Okta Browser Pluginの青いアイコンをブラウザーツールバー上の目的の位置までドラッグします。
  • Internet Explorer：ツールメニュー（）で、［Manage add-ons（アドオンの管理）］［Okta toolbar（Oktaツールバー）］に移動し、［Enable（有効にする）］をクリックします。
  • Safari：ブラウザーツールバーの任意の場所を右クリックし、［Customize Toolbar（ツールバーをカスタマイズする）］をクリックしてから、［Your Apps（使用アプリ）］アイコンをブラウザーツールバーの希望する場所にドラッグします。

この手順を開始する

  1. Oktaアカウントを信頼してAccount Chooserに追加する
  2. Account Chooserを表示し、アカウントを切り替える

Oktaアカウントを信頼し、Account Chooserに追加する

  1. すでにOktaにサインインしている状態で、このブラウザーで信頼 されていない、あるいはまだアクセスしていない別のOktaアカウントにサインインします。アカウントが信頼されると、Okta Browser Pluginはエンドユーザーをアカウントにサインインさせ、［Your Apps（使用アプリ）］リストにアカウントのアプリケーションが表示されるようになります。

    2. Okta End-User Dashboardにバナーが表示され、アカウントを信頼するかどうか指定するよう求められます。ダッシュボード上のアプリによっては、アカウントを信頼した後でのみアクセス可能になるものがあります。

  2. ブラウザーツールバーにあるOkta Browser Pluginの青いアイコンをクリックします。
  3. ［Your Apps（使用アプリ）］メニューに［Trust / Reject（信頼する／拒否する）］オプションが表示されるときは、該当する方を選択します。
    • ［Trust（信頼する）］をクリックすると、赤いバナーが消えてダッシュボードに残されたアプリがアクセス可能になり、そのアカウントがAccount Chooserリストに追加されます。
    • アカウントを信頼しない、またはプラグインを介してそのアカウントでアクションを実行したくないときは、［Reject（拒否する）］をクリックします。後になってアカウントを信頼することにしたときは、ページをリフレッシュするか、アカウントにサインインし直すと、［Trust（信頼する）］オプションを表示できます。

      • 注：Chrome、Edge、Firefoxブラウザーで、Okta Browser Pluginの青いアイコン（）の上に赤い四角形が表示される場合、信頼または拒否のアクションが必要であることを示しています。

Account Chooserを表示し、アカウントを切り替える

このタスクはオプションです。Account Chooserを使用することで、Okta Browser Pluginに追加された複数のアカウントを容易に切り替えることができます。

  1. ブラウザーツールバー上のOkta Browser Pluginの青いアイコン（）をクリックします。
  2. ［Account Chooser（アカウントチューザー）］アイコンをクリックします。

    3. アクティブなアカウントの横には青いリボンが表示されます。

  3. リスト内の切り替え先アカウントをクリックします。
  4. リストからアカウントを削除するには、アイコンをクリックします。
