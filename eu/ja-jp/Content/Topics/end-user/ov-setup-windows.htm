WindowsデバイスでOkta Verifyを開始する

Okta Verifyでのアカウントの追加は、シンプルな2段階のプロセスで行います。まず、コンピューターで組織のOktaアカウントにサインインしてから、生体認証を使用して本人確認を行います。

開始する前に

管理者が認証要素としてOkta Verifyを構成した場合、デバイスですでに利用可能なアプリ、または管理者が利用可能としたアプリをインストールするよう求められます。IT部門がユーザーの代わりにデバイスでOkta Verifyアプリをインストールする場合もあります。

  • 複数のデスクトップデバイスでOkta Verifyを登録するには、別の要素、つまりYubikey、SMS、または追加デスクトップでサインインするために使用できるプッシュを使ってOkta Verifyなどで登録する必要があります。
  • 最初のOkta Verifyの登録がデスクトップデバイスで行われた場合、次のOkta Verifyの登録はモバイルデバイスで行うことができます。デスクトップで組織のOktaアカウントを開いて、[Settings (設定)] > [Extra Verification (追加認証)] > Okta Verifyを開き、[Set up (セットアップ)]をクリックします。次にデバイスの種類を選択します。画面にQRコードが表示されます。モバイルデバイスからQRコードをスキャンして登録します。

上記の考慮事項に従わずに、2つ目のデスクトップデバイスでOkta Verifyをセットアップしようとすると、Okta Verifyを使用して登録するよう繰り返し求められますが、この状況では利用できません。

このタスクを開始する

  1. Okta Verifyを開きます。
    タスクバーからOkta Verifyアイコンをダブルクリックします。
  2. [Add Account (アカウントを追加する)]をクリックします。
  3. 組織のサインインURLを入力して、[Next (次へ)]をクリックします。
    ブラウザーウィンドウが開きます。
    注意：サインインURLは、会社のポータルにアクセスするためのブラウザーで使用するウェブアドレスです。管理者から受信したようこそ/アクティベーション電子メールで利用できる場合もあります。
  4. ブラウザーで認証：
    • 電子メールアドレスを入力し、[Verify (確認)]をクリックします
    • パスワードを入力し、[Verify (確認)]をクリックします。
  5. デバイスが生体認証をサポートする場合、Okta VerifyでWindows Helloを有効にするよう求められます。[Enable (有効にする)]をクリックします。

    Windows Helloがデバイスでセットアップされていない場合、まずデバイスで生体認証をセットアップするよう求められます。[Setup (セットアップ)]をクリックしてから、Okta Verifyに戻りWindows Helloを有効にします。
    指紋リーダーで指をスキャンします。
    [More choices (その他の選択肢)]リンクをクリックする場合、暗号化キーを追加するよう求められます。キーを入力して確認し、[OK]をクリックします。
  6. アカウントが正常に追加されたことを確認するメッセージが表示されます。

    アプリにアクセスする際に、Okta Verifyを使用して認証できます。
