Automatically update Active Directory agents

Agent auto-update functionality keeps your agents up to date and helps ensure that your org has the latest features. You can auto-update single or multiple agents on demand, or you can schedule the updates to occur outside of business hours to reduce downtime and disruption to your users.

With agent auto-updates, you don't need to uninstall and then reinstall agents whenever a new agent is released. Instead, you select the agents you want to update and the installer removes and then installs the latest version of the agents. When an agent is auto-updated, a record of the event is added to the System Log.

To use agent auto-update functionality, you must download and install version 3.8.0 or later. See Download the latest agent version.
