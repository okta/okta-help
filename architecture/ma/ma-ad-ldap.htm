Integrate with a company that uses Active Directory or LDAP

If the company you acquired uses (AD) or Lightweight Directory Access Protocol (LDAP), deploy agents to sync users and groups with the Okta parent org.

Workflow

  1. Push users from the acquired company to your company’s Okta org. See Directory integrations.

  2. Provision applications.

  3. Configure authentication enrollment policies and rules.

  4. Configure authentication policies to define access to parent org resources.
