Import groups from Active Directory

You can import groups from any forest or domain connected to . The detects all groups in the domain or the Organizational Units (OUs) that you select. If you register an for more than one domain and you have the root OU selected for all domains, it imports all groups.

limits the total number of bytes that can be sent from an AD or LDAP agent to server in a single request. To avoid exceeding size limitations during data import, result sets containing multiple group objects are split into separately sized units and each unit is sent in a separate request.

A single group that exceeds the defined size limitation is still sent to , but a standard HTTP 413 (Payload Too Large) error might be returned. The length of the group distinguishedName (dn), the length of the user dn within the group, and the group membership size all contribute to the total bytes sent to .

If you receive a HTTP 413 (Payload Too Large) error, recommends splitting direct group membership into nested group membership or sub-groups to avoid the size limit limitation and allow the data to be sent in a single request.

does not support nested groups. imports all nested directories for group members and adds the user to each group in .
