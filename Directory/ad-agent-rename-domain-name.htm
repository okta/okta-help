Rename an Active Directory domain

Renaming an Active Directory (AD) domain without uninstalling the breaks the connection between the and . recommends that you avoid changing the AD domain name.

  1. Uninstall any that is on the domain controller that you are renaming. Refer to Update the [%=okta-variables.ProductName%] Active Directory agent.
  2. Rename the AD domain. See your Microsoft documentation.
  3. Reinstall the with the new domain name. Refer to Update the [%=okta-variables.ProductName%] Active Directory agent.

The renamed domain appears as a new AD application instance on the .
