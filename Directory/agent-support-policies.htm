agent support policies

agent support policies ensure the security and functionality of agents. When an agent is no longer supported, it is deprecated and not eligible for bug fixes or other corrective action by . If you experience functionality issues, make sure the installed agent is supported before contacting Okta support.

The agent support policy does not make existing agents obsolete or change existing functionality. If you're unable to resolve an issue with an unsupported agent using traditional troubleshooting methods and you require assistance from , you must upgrade the agent to a supported version.

The agent support policy changes frequently. To confirm if an agent is supported, recommends reviewing this topic often.
