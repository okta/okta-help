View the IWA Web agent status

  1. In the , go to Security > Delegated Authentication.
  2. View the status of all IWA Web agents in the In the IWA AGENTS area.

You can also check the system logs for information related to your IWA Web agents. For example, if your primary IWA Web agent goes offline, your system logs a report of when it happened and the promotion of the backup IWA Web agent. To view the report, go to Reports, and then click Active Directory Agent under System Logs.
