Install the

The Administrative Account used during IWA Agent installation must have Super admin permissions.

When you install the IWA agent the IP address of the client is added to the LegacyIPZone. See Define a network zone for IWA.

  1. In the , go to Settings Downloads .
  2. In the SSO IWA Agents area, click Download Latest.
  3. Browse to the location where you downloaded the installation file and double-click OktaSsoIwa-x.x.x .
  4. Click Next.
  5. In the Web Application Pool Identity dialog box, select Create or use the OktaService account, then click Next.
  6. Optional. When prompted, specify a proxy server for your IWA Web agent.
  7. On the Register Desktop Single Sign-On screen, select an environment (Production, Preview, or Custom), enter your customer subdomain name, and then click Next.
  8. On the Sign In page, enter your Super admin username and password, and then click Sign In.
  9. To grant permission to access the API, click Allow Access.
  10. When the message Installation completed appears, click Finish.
  11. In the , go to Security Delegated Authentication and in the On-Prem Desktop SSO area, confirm the is connected.
