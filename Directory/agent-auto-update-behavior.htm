Define the behavior for failed agent auto-updates

Define the behavior of agent auto-updates to determine what happens when an agent auto-update is interrupted. You can choose to pause or continue all remaining updates.

  1. In the , go to Directory Directory Integrations .
  2. Click an AD instance.
  3. Click the Agents tab.
  4. Select Update agents Manage auto-updates .
  5. Click Edit in the General settings pane and select one of these options:
    • Stop the rest of the auto-update job — Select this option to cancel all remaining updates when one of the agent auto-updates is unsuccessful.
    • Skip it and continue the rest of the auto-update job — Select this option to continue the remaining agent auto-updates when one of the agent auto-updates is unsuccessful.
  6. Click Save.
