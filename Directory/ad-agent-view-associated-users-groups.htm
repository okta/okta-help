View users and groups associated with an Active Directory instance

View users and groups to quickly confirm specific user or group assignment. recommends using the People page to locate users and view user information. To access the People page, click Directory People .

  1. In the , go to Directory Directory Integrations .
  2. Click Active Directory and then click the Assignments tab.
  3. Optional. To view only the people or groups associated with an Active Directory (AD) instance, click People or Groups in the Filters list.

View and assign ownership of an Active Directory instance

Admins can view and manage ownership of groups in Okta Directory and import an owner from Active Directory. This functionality streamlines assignation and other actions to group members. For example, Access Requests can now go directly to Group Owners, instead of all group members.

To assign ownership of a group, go to Directory Groups , select a group, and find the Owners tab on the Group page.

  1. Click the switch for Group Ownership for Okta Directory to enable admins to manage the owners of groups in Okta Directory.

  2. Click the switch for Group Ownership sourced from Active Directory to import the group owner using the ManagedBy attribute of the AD group.
