Cancel an on-demand agent auto-update

Cancel queued on-demand agent auto-updates to stop them from completing. When you cancel queued on-demand agent auto-updates, all agent auto-updates in progress are completed and all queued agent auto-updates are canceled.

  1. In the , go to Directory Directory Integrations .
  2. Click an AD instance.
  3. Click the Agents tab.
  4. Click Cancel in the Update agents dialog.
  5. Click Cancel update in the confirmation dialog.
