Download the latest agent version

If your org uses version 3.7.0 or earlier and you want to use agent auto-update functionality, you need to download the latest version of the agent, uninstall the existing agent, and then manually update the agent.

  2. Click an AD instance.
  3. Click the Agents tab.
  4. Click Update agents Download latest .
  5. In the AD Agents section, click Download Latest.
  6. Uninstall and then reinstall the AD agent. See Update the [%=okta-variables.ProductName%] Active Directory agent.
