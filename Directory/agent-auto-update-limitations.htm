Agent auto-update known limitations

The following are the known limitations that apply to agent auto-update functionality:

  • Agent auto-update functionality is available with version 3.8.0 and later. If your org uses versions earlier than 3.8.0, you need to update the agent to take advantage of agent auto-update functionality. See Update the [%=okta-variables.ProductName%] Active Directory agent.

  • Admins with custom admin roles cannot use agent auto-update functionality.
