Get started with CSV directory integration

Start here if you're new to CSV directory integrations, you want to review integration prerequisites, or you want to quickly locate support information.

The CSV directory integration is a lightweight out-of-the-box option that lets you build custom integrations for on-premises systems using the Provisioning agent.

Organizations commonly use CSV files containing user identities. Using the agent, can ingest these files on a recurring basis, importing users and their attributes into . You can expect all the functionality available in other profile sources such as Workday or Active Directory, which includes scheduled imports, attribute level sourcing, matching rules, and a rich profile of user attributes.
