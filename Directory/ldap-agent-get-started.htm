Get started with LDAP integration

Enterprise applications such as Customer Relationship Management (CRM), human resources, and email use the Lightweight Directory Access Protocol (LDAP) internet protocol to authenticate users and retrieve information from network servers. The allows delegated authentication to an on-premises LDAP server, meaning that users can authenticate to using their local LDAP credentials without replicating those credentials into the cloud. The can also make the main source of truth for your enterprise. This solution can be implemented without additional servers or firewall changes.
