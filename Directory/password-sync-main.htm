Synchronize passwords

Password synchronization helps you coordinate and manage user passwords and makes sure a user’s Active Directory (AD) password and their password always match. With password synchronization, your users use a single password to access applications and devices.

You can synchronize passwords from to AD or from AD to . The password synchronization methodology you choose is determined by which directory you are using currently to authenticate and provision users. To synchronize passwords from to AD, you enable Sync Password on the Admin Console Provisioning page. To synchronize passwords from AD to , you install the on all integrated domain controllers in your domain.
