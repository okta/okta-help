View LDAP agent status information

The Agent monitors page lists all Okta LDAP Agents installed in a domain. To be considered operational, the LDAP agent must have a stable connection to Okta servers (the LDAP agent is sending requests to the Okta server at regular intervals).

The LDAP agent must also have a valid configuration and must be integrated with the on-premises LDAP server.

You can view the agent operational status, connection state, and version information. You can also download, update, deactivate, and delete single or multiple agents.

The following icons in the Version column help you quickly identify the status of your installed agents:

  • Green check mark — the agent is up to date.
  • Yellow warning triangle — the agent is out of date and should be updated.
