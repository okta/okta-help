Auto-update multiple agents on demand

When a new agent is available and your existing agents are out of date, a warning icon appears in the Version column on the Admin Console Agent monitors page. Keeping your agents current helps ensure your org has the latest features and functionality.

When you start an agent auto-update, the agents are disconnected from . To avoid service disruptions for your users, recommends that you have two operational agents installed on the directory instance being updated.
