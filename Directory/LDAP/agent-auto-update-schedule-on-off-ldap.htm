Turn an agent auto-update schedule on or off

Turn an agent auto-update schedule on or off when you want to activate or stop an agent auto-update schedule.

  1. In the , go to Directory Directory Integrations .
  3. Click the Agents tab.
  4. Select Update agents Manage auto-updates .
  5. Select one of these options:
    1. Click Actions Turn schedule OFF next to an agent auto-update schedule to stop it running.
    2. Click Actions Turn schedule ON next to an agent auto-update schedule to activate it.
