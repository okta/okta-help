Download and install the Provisioning agent

To allow the exchange of user and group data between and your CSV directory, you need to install and configure an Provisioning Agent. The installation of the Provisioning Agent also lets you use additional provisioning functionality such as profile push, password push, user deactivation, group push, user import, and group import.

  1. In the , go to Directory Directory Integrations Add Directory .
  2. Select Add CSV Directory.
  3. In the Add CSV Directory page, enter a name for the CSV Directory in the Application label field and click Done.
  4. Go to Settings Downloads .
  5. Scroll to the Okta Provisioning Agents section, and download the Provisioning Agent for your operating system.
  6. Complete the agent installation.
