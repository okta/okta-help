Automations and hooks

Okta provides features that enable you to automate and customize your Okta processes. Automate lifecycle tasks using rule-based Automations. Integrate with other systems using custom code or create custom integrations for the Okta Integration Network using Event Hooks and Inline Hooks.

Automations
Respond to changes in your end users' lifecycles.
Inline Hooks
Integrate your custom code into Okta workflows.
Event Hooks
Trigger process flows in your own software systems.
Delegated flows
Run Okta workflows directly from the Admin Console.
