Register multiple domains to an Active Directory agent

You can register multiple domains to a single . However, these domains must be in the same forest and contain trust, otherwise the Service account (which the agent runs as) cannot connect to the other domains to register them.

To add more domains to the , you need to ensure the domains have an external trust and the domain service account has sufficient privileges in all domains.

  1. On the server running the , select Start All Programs Okta Okta AD Agent Okta AD Agent Manager .
  2. Select Domains.
  3. In the drop-down menu, select a domain and then click Register. Alternatively, type the domain name in the field, and then click Register.

A message appears stating that your new domain has been registered and you are prompted to restart the agent.
