View installed agents

The Agents page in the lists all installed agents. You can view the agent operational status, connection state, and version information.

Mobile admin, read only admin, or super admin permissions are required to access the Agents page.

Super admin or app admin permissions are required to manage the agents listed on the Agents page.

  1. In the , go to Dashboard Agents .
  2. Click a tab to view Active Directory, LDAP, On-premise, On-premise MFA, RADIUS, or IWA agent information.
  3. Optional. Click Load more to load and view more agent information.
  4. Optional. Click Download page to download and save agent information as a comma-separated values (CSV) file.
