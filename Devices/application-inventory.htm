Application Inventory for macOS

The Application Inventory page lists the applications that your users have installed on their macOS devices. This page helps you to:

  • Collect data for software audits, to avoid costs associated with exceeding software license terms .
  • Eliminate the need for tools that track contracts and licenses.
  • Assess the security state of end user macOS devices. If a bad app is detected, organizations can deprovision enterprise data from insecure devices.
  • Quickly identify compromised devices.
