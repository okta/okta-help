Browser Plugin Version History

This page displays current and past plugin versions for all supported browsers. This page is updated whenever a new plugin is released (GA or EA). For download, see Download the Okta Browser Plugin.

Version History

Plugin History shows current and past GA plugin versions and the release that they became available.

Note: Occasionally we release the GA version of a plugin in a phased rollout to orgs. In those cases, the latest GA version of the plugin is not available to all orgs at once.
