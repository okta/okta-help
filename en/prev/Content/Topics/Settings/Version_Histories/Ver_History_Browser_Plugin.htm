Version history

2019.10.0: Version 5.33.0 for all browsers

This version includes the following:

  • Security warning and anti-phishing whitelist
  • Reflection of real-time appAn abbreviation of application. Essentially, it is a web-based site used to perform any number of specific tasks, and requires authentication from end users by signing in. and profile changes in the end user dashboard
  • Custom URL domainA domain is an attribute of an Okta organization. Okta uses a fully-qualified domain name, meaning it always includes the top-level domain (.com, .eu, etc.), but does not include the protocol (https). support for the plugin (available in Preview orgs)
  • New look (available in beta)
  • Back-end enhancements

 

2019.09.0: Version 5.32.0 for all browsers

This version includes the following:

  • Custom URL domain support for the plugin (available through the EA Feature Manager)
  • Back-end enhancements

 

2019.06.0: Version 5.29.0 for all browsers

  • Quick Access apps tab (currently available as Early AccessEarly Access (EA) features are opt-in features that you can try out in your org by asking Okta Support to enable them.
