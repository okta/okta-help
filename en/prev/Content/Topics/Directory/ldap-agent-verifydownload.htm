  2. Import the file you create to the RPM database:
    rpm --import okta_rpm_gpg_key
  3. Validate the signature of the LDAP agent RPM file using the command below and the RPM file name:
    rpm -K OktaLDAPAgent_xx.xx.xx.x86_64.rpm

    rsa sha1 (md5) pgp md5 OK
