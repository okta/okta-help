Ubuntu

Installation

First, add the Advanced Server Access package repository:

# Add the  Advanced Server Access apt repo to your /etc/apt/sources.list system config file:
echo "deb http://pkg.scaleft.com/deb linux main" | sudo tee -a /etc/apt/sources.list

# Trust the repository signing key:
curl -C - https://dist.scaleft.com/pki/scaleft_deb_key.asc | sudo apt-key add -

# Retrieve information about new packages
sudo apt-get update

Then, install the scaleft-erver-tools package:

sudo apt-get install  scaleft-server-tools
