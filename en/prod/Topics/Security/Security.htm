VIDEO - A Quick Look at Securing Okta

Security

The security menu gives access to the following security functions. Follow the links below for details.

 

General – Set email, remember user, and mobile pin settings.

Policies – Configure sign on policies, mobile policies, group password policies, and wifi policies.

Authentication – Directory and authentication management options.

Identity Providers – Configure how external identity providers interact with Okta.

Network – Identify your network perimeter to Okta.

Administrators – Grant Admins access to all sections of the application, or limit their access to only certain apps.

API – Manage and create all Okta API tokens, and to add Origin URLs.

 

For information on Okta's approach to security overall, see Security at Okta.
