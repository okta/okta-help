iOS: Okta Safari extension

On iOS devices you can use the OktaSafari extension to sign in to apps directly from Safari without having to open Okta Mobile.

Before you begin

You can set up the Okta Safari extension on your device only if it's configured by your system administrator. If you are the system administrator, go to Okta Mobile Safari Extension.

Start this procedure

  • Task 1: Set up the Okta Safari extension
  • Task 2: Use the Okta Safari extension to sign in to apps

Task 1: Set up the Okta Safari extension

  1. Tap the open-in icon at the bottom of the Mobile Safari browser.

  2. Swipe right to reach the end of the options, then tap More.

  3. Locate the Okta extension and turn it on.

  4. Drag the Okta extension to the top of the list.

    5. The extension appears when you tap the open-in icon in Safari.

Task 2: Use the Okta Safari extension to sign in to apps

  1. On you mobile device, open Safari and go to the sign-in page of an app, such as Atlassian Cloud. Tap the open-in icon.

  2. Select the Okta extension.

  3. Complete one of the following procedures:

    • If you are already signed in to Okta:
      1. Confirm your Okta PIN.
      2. Wait for the credentials to load.
    • If you didn't sign in to Okta yet: 
      1. Enter your credentials on the sign-in screen.
      2. Complete the multifactor authentication if it's configured by your system administrator.
      3. When prompted, create and confirm a new Okta PIN.
      4. Wait for your credentials to load.
  4. Tap the app.

    5. If you are using Safari, you can use the Okta extension to fill out your sign-in forms. The credentials are auto-populated and you are signed in.
