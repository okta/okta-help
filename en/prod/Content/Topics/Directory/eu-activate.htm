Activate and deactivate users

Activate end user accounts

End user accounts can be in a variety of states that require administrative action. When end user accounts are in a Staged, Suspended, or Deactivated state, you might need to activate them.

Activating end user accounts in your organization changes the account status from Staged, Suspended, or Deactivated to Active. The left column of the People page enables you to filter your end user list based on their account status.

Activating individual users

If the end user account status is Staged, you can activate the account by clicking the Activate link, found in the Status column.

If the end user account status is Suspended or Deactivated, you can activate the account by selecting the user and clicking the Activate button found in the end user's profile page. Note that a deactivated end user account is removed from its provisioned applications. When you reactivate an account, you must also reassign its apps.

Activating batches of users

To activate multiple Staged or Deactivated end user accounts:

  1. Go to More Actions > Activate.
  2. Select Show deactivated users if needed.
  3. Click Activate All or select specific users and click Activate Selected.
  4. When the Activate People dialog appears, click Activate to activate the chosen end users, or click Cancel.

When activated, each user appears as Active in the Status column and an email is sent to the end user's primary or secondary email address, informing them that their accounts are active. Remember: deactivated end user accounts are removed from their provisioned applications. If you activate any previously deactivated accounts, you must also reassign their apps. Once active, they can access all the provisioned applications assigned to them.

Deactivate and delete users

Deactivating people in your organization changes their account status from Active to Deactivated. Deactivating end users revokes their appAn abbreviation of application. To deactivate an end user:

  1. From the More Actions menu, click Deactivate. The Deactivate People page appears.

  2. From the list of users, select the users you want to deactivate and then click Deactivate Selected.

  3. A Deactivate People dialog appears. Click Deactivate to confirm, or click Cancel to abort.

The end user is listed as Deactivated under the Status column. Admins will receive an email listing all users deactivated in the past 30 minutes. Once deactivated, a user's profile cannot be changed unless it is again made active.

Permanently delete an end user account

You can permanently delete a deactivated user with the Delete button that appears in the directory screen for that user. You cannot undo this deletion. After the deletion, the user is not visible on the People page and is not returned in API responses. However, any log entries that reference the user are maintained. After deletion you can reuse the user name and other identifiers.

Note: You cannot delete a user that is set as the technical or billing contact.
