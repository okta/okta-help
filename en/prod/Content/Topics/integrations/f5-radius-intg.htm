Configure F5 BigIP APM to interoperate with Okta via RADIUS

This guide details how to configure F5 BIG IP APM to use the Okta RADIUS Server Agent in conjunction with the Okta Integration Network (OIN) F5 BIG IP RADIUS for APM and VPN App.

 

Note

To integrate with Okta via SAML 2.0, add the F5 BIG IP SAML for APM and VPN SAML app in Okta by:

  1. Navigating to the Applications tab.
  2. Selecting Applications > Add Application.
  3. Searching for F5 BIG IP SAML for APM and VPN.
  4. And then click Add.

Typical workflow

Task

Description

Download the RADIUS agent
  • Download the Okta RADIUS Agent from the Settings > Downloads page your in Okta org. Note that there are both Windows and Linux agents.
  • For throughput, availability and other considerations, see Okta RADIUS Server Agent Deployment Best Practices.
Install the Okta RADIUS Agent.
  • Install either the Windows or Linux RADIUS agents as appropriate for your environment.
Configure application
  • In your Okta org, configure the F5 BIG IP RADIUS application.
Configure gateway
  • Using the F5 console , configure F5 BIG IP to work with RADIUS.
Configure optional settings
  • Configure optional settings as required, such as Client IP Reporting and Groups response.
Test
  • Test the newly integrated agent.
