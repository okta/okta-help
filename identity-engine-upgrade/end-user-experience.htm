End-user experience

Orgs that have successfully upgraded will unlock new functionality and capabilities. As part of this new functionality, end users will start seeing changes in the sign-in experience. Some of these experiences will change immediately after the upgrade and some others will appear once new features are turned on for the org.

Changes for end users:

  • Sign-in flows

  • Multifactor authentication enhancements

  • Enhanced password reset and account recovery

To better help prepare your users, make use of the End User Adoption Toolkit so they know what to expect during and after the upgrade.
