RADIUS and MFA

After you upgrade to , learn about the changes to RADIUS and multifactor authentication (MFA).

Change summary

: Supports MFA for Active Directory Federation Services (ADFS) and Personal Identity Verification (PIV).

: Doesn't support MFA for ADFS or PIV.
Admin experience

To configure authentication policies, go to Security .

If orgs that use MFA for ADFS in upgrade to and then configure the Security Question authenticator, it doesn't appear during authentication flows.

The On-Prem MFA agent supports both On-Prem MFA and RSA SecurID. Org admins who switch between these receive a permissions error. To avoid the error, perform the same action as a super admin.

When you switch between On-Prem MFA and RSA SecurID, an unsupported operation error can occur. Avoid this error by first disabling the current authenticator and then enabling the replacement.

When swapping an On-Prem MFA or RSASecurID authenticator, prompts you to disable the existing authenticator first.
