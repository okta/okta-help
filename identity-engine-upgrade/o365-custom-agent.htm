Office 365 Custom User Agent

Learn how Microsoft Office 365 Custom User Agent changes after the upgrade.

Change summary App sign-on policy rules with a Custom User Agent (as a client application) don't migrate fully. The Custom User Agent String is lost after migration and can't be recovered.
Admin experience You need to add the user agent to the authentication policy rule after the upgrade has been completed, and then activate the rule.

  • Before you upgrade: Capture the current setting in your org in the Office365 application under the authentication policy rule.

  • After you upgrade: Add the custom client filter in the authentication policy for Office 365.
