How do I set up Desktop in if it was also set up in ?

Since IWA was used for in , IWA agents stay in place until certificates are issued from the new platform:

  1. In , go to Security Device Integrations and configure the platform.
  2. Configure a new Certificate Authority (CA).
  3. Go to your device management software to deploy the new certification to all devices.
  4. Deploy to all devices.
  5. Remove the IWA agents.
  6. Remove the platforms.
  7. Don't revoke the certificate from the end-user devices.

What happens to the existing certificate after I deploy the new certificate?

The certificate works until you install on end-user devices and the users register their account by using the Okta Verify desktop application. After users create the account, the Certificate Authority requires the new certificate.

What happens if I don't roll out the new certificate to our managed devices?

The existing certificate continues to work until the users install and register on their devices. Users can't access apps if they must use a trusted device but don't have the new certificate.
