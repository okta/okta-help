Custom email templates

Email templates in switch from Expression Language to Velocity Template Language. This means that there’s now a single set of email templates: admins don’t have to choose between templates and templates.

The enhanced functionality that currently allows for One-Time Password (OTP) code remains in the following templates:

  • Active Directory Password Reset
  • Active Directory Password Unlock
  • LDAP Forgot Password
  • Reset Authenticator
  • Unlock Authenticator

Admins don't have to perform any manual tasks to upgrade custom email templates from to . During the upgrade, code applies Velocity syntax and then merges the templates. Admins should perform a one-time validation that their customizations are correct.

After upgrade, the only difference in the is a streamlined selection process. All templates now use Velocity Template Language only.
