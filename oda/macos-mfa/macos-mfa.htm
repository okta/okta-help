for macOS

Use (Multifactor Authentication) to strengthen the security of users' authentication to macOS computers. With this solution, you customize the sign-in flow so that users are prompted for MFA authentication after they access their computer. Users are prompted to set up an offline authentication factor that they can use to verify their identity when an internet connection isn't available.
