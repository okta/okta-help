Use (Multifactor Authentication) to strengthen the security of users' authentication to Windows computers. With this solution, you customize the sign-in flow so that users are prompted for MFA methods after they enter a Windows password. Users can set up offline sign-in methods that they can use for Windows when an internet connection isn't available.
