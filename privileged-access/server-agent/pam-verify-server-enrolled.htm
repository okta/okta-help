Verify server enrollment

You can check if a server is enrolled in a project from the dashboard or by running a command on a client.

To check if a server is enrolled in a project:

  1. Open the dashboard.

  2. Go to Resource Administration Resource Management and select the project where the server was enrolled.

  3. Go to the Servers tab.

If the server was successfully enrolled in the project, it appears in the list.
