Supported operating systems

Application

Operating System

client
  • Amazon Linux 2

  • Amazon Linux 2022 (x86)

  • Debian 10 to 11
  • Fedora 35, 36, 37
  • FreeBSD 12 to 13
  • macOS 10.15
  • macOS 11 (x86, ARM)
  • macOS 12 (x86, ARM)

  • MacOS 13

  • Red Hat Enterprise Linux 7, 8, and 9

  • Ubuntu 16.04, 18.04, 20.04, and 22.04 (x86, ARM)
  • Windows 10 and 11

server agent
  • Alma Linux 8.5

  • Amazon Linux 2022 (x86)

  • Amazon Linux 2 (x86, ARM)
  • Debian 10 to 11
  • FreeBSD 12 to 13
  • Red Hat Enterprise Linux 7, 8, and 9
  • SuSE Linux 12 and 15
  • Ubuntu 16.04, 18.04, 20.04 (x86, ARM), and 22.04 (x86, ARM)
  • Windows Server 2012 and 2012 R2
  • Windows Server 2016 (except Nano Server, which doesn't support RDP)
  • Windows Server 2019

The server agent isn't supported on Microsoft Active Directory domain controllers.

gateway
  • Amazon Linux 2

  • Amazon Linux 2022 (x86)

  • Alma Linux 8.5
  • Debian 10 to 11
  • FreeBSD 12 to 13
  • Red Hat Enterprise Linux 7, 8, and 9
  • Ubuntu 16.04, 18.04, 20.04 (x86, ARM), and 22.04 (x86, ARM)

Ubuntu 20.04, 22.04, RHEL 8, or RHEL 9 is required for AD-joined and RDP connections routed through a gateway.

Ubuntu 20.04, 22.04, RHEL 8, or RHEL 9 is required for RDP connections routed through a gateway.
