Create tokens and labels

Set up tokens are used to enroll gateways into and labels control access to server for a given project.

Prerequisites

  • You must be a member resource administrator group or be the owner of the resource group where the project is created.

Create an gateway setup token

A setup token is a Base64 encoded object that includes metadata used to enroll the gateway into an team. To create an setup token for a project:

  2. Click Set up gatewaysCreate setup token.
  3. On the dialog that appears do the following:
    1. Enter a token
    2. Select an existing label or create a new label. Labels must be a key-value pair (for example, environment:staging).
    3. Press the Tab or Enter key to finalize the label.
    4. Optional. Repeat this process to add additional labels.
    5. Click Save.

    6. Copy the token and then click Done. You will need this token when you perform setup using the configuration options. See Setup Token options

Edit labels

You can edit a gateway to add new labels or remove an existing label.

  2. Click Actions Edit, on the gateway you want to edit.
  3. Go to Labels and type in a lable name to select an existing label or to create a new one.
  4. To remove a label, select x on the label
  5. Click Save.
