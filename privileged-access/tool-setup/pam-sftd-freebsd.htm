Install the server agent on FreeBSD

To install the server agent:

  1. Download the latest version of server agent from the package repository.
  2. Install the server tools package, which includes the agent:

    pkg install -y scaleft-server-tools-<version>.pkg

  3. Remove any existing agent files from /var/lib/sftd/.

  4. Create a new or copy an existing enrollment token from and paste it into the file /var/lib/sftd/enrollment.token
  5. Run the command.

    sysrc sftd_enable=YES

  6. Start the server agent.

    service sftd start
