Install the client on macOS

To install the client:

  1. Go to the macOS client repository.
  2. Click the latest version to download.
  3. Run the installation package.

Uninstall the client

To uninstall the client:

  • Delete the application (/Applications/ScaleFT.app).
  • Delete the symbolic link to the client (rm /usr/local/bin/sft).
