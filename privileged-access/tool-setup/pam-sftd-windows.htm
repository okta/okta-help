Install the server agent on Windows

To install the server agent:

  1. Go to the Windows client repository.
  2. Click the latest version to download.
  3. Install the MSI by double-clicking the download or by running the command:

    msiexec.exe /qb /i ScaleFT-Server-Tools-#.msi

The server agent is not supported on Microsoft Active Directory domain controllers.

Automate agent installation

server tools can be automatically installed on Windows servers running in AWS and other cloud environments using a PowerShell user data script, or by a PowerShell command run locally or remotely.

You can find an open-source PowerShell module for managing server agent installations for Windows at https://github.com/okta-server-asa/scaleft-powershell/.
