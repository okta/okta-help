Use PuTTY for

Windows users can install PuTTY to help open Remote Desktop Protocol connections to Windows servers.

Before you begin

  • Download and install PuTTY.
  • Install the client on your local device. See Install the [%=okta-feature-names.Okta Privileged Access%] client.

Start the procedure

  1. Open the terminal.
  2. Identify an available server.
  3. Connect to the server with the following command:
    sft putty <servername>
