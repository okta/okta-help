Enroll the client

After teams have installed the client, they must enroll the client in an existing team. Enrolling the client adds it to the client inventory on the platform and authorizes it to take actions.

Teams can enroll clients individually or use silent enrollment to automate the enrollment process for larger groups of clients. See Silently enroll the [%=okta-feature-names.Okta Privileged Access%] client.

Before you begin

Install the [%=okta-feature-names.Okta Privileged Access%] client.

Start the task

  1. From a command prompt on the client, run the command: sft enroll
  2. From the browser window, select the team to enroll the client.
    Note: When run in a headless environment, the command outputs a URL. Open the URL in a browser.
  3. Click the team to enroll with, then click Approve.

Upon success, the client is enrolled with the team.
