Configure team-level group attributes

Teams can set and manage team-level group attributes from the dashboard.

  1. Open the dashboard.
  2. Go to Resource AdministrationSystem Configuration.
  3. Select the Users and Groups Attribute Settings tab.
  4. Click the gear and select Edit Team Attributes. The Update Team Attributes page appears.
  5. Specify updated attribute values.
  6. Click Save Group Settings.

saves the group attributes.
