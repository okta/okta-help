System Configuration

The dashboard provides several features and settings that can help teams better control access to server resources. To configure these settings, you must have the resource admin role.

Topic Description
Client Session The number of hours before clients are required to reauthenticate. Optionally, you can enable the feature to auto-approve clients when users enroll in the team.
Web Session Duration The number of hours before the user is required to reauthenticate.
Disconnected Mode Not in PAM beta.
Statistics Visual representation of the number of users, groups, projects, clients, and servers in the team.

Configure client session settings

Using this feature, you can auto-approve clients when users are enrolled in the team and configure the client session duration.

  3. Select the Client Session tab.
  4. Optional. Select the checkbox beside Approve Clients Without Interaction.
  5. Click the dropdown arrow and select the number of hours before clients are required to reauthenticate.
  6. Click Save Settings.

Configure web session duration

Using this feature, you can set a time frame after which users are required to reauthenticate.

  3. Select the Session Duration tab.
  4. Click the dropdown arrow and select the number hours before users are required to reauthenticate.
  5. Click Save Settings.
