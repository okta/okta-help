Configure for gateway connections

If a connection is being routed through an gateway, you'll need to configure additional settings in the client.

This process makes unable to verify if a connection is forwarded directly through the client.

  1. Open the client.
  2. In the left panel, open Application Default Settings .
  3. Right-click Remote Desktop Default Settings and select Properties.
  4. In the left menu, select Advanced, and go to the Authentication tab.
  5. Enable TLS Encrytpion.
  6. Click Apply & Close.

Create a dynamic folder for team servers

Dynamic folders allow to automatically add a list of servers that belong to a team. You can use this to start RDP and SSH sessions directly from .

  1. In , go to File New Document .
  2. Right-click the new document and select Add Dynamic Folder . The Dynamic Folder Settings window appears.
  3. Enter a name and description for the folder.
  4. Click Dynamic Folder Script.
  5. In the Interpreter dropdown, select Bash.
  6. Replace the default script with:sft list-servers-rjson
  7. Click Apply & Close.
  8. Right-click the dynamic folder and select Reload.
  9. If prompted for approval by , click Approve.

populates the folder with a list of available servers. To refresh this list, right-click the folder and select Reload.
