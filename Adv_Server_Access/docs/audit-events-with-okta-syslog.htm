Audit Events Integration with System Log

This is an . To enable it, contact Okta Support.

audit events integration with the System Log lets administrators access audit events directly through the System Log. administrators can use this feature to view, search, and query the System Log for specific audit events and filter them based on a specific value. This feature doesn't remove the current audit events log from .

All the audit events available in the System Log are documented in the Event Types page. The audit events are prefixed with the pam namespace for easy searching and selection by event types.

Prerequisites

  • Ensure that you're signed in to the .

  • Verify that you have the required permission in org-wide settings. See Standard administrator roles and permissions for details.

  • You must have your org connected to your team. See Connect your [%=okta-variables.ProductName%] org to [%=okta-feature-names.ScaleFT%].

View audit events in the Okta System Log

  1. recommends assigning the lowest privileges required to view the logs, such as the Report Administrator role, to administrators who need to view audit events in the System Log.

  2. Specify a date range to filter the report. retains events for 90 days, so the earliest available starting date range is three months prior.

  3. Use the search filter to find your audit events. For example, you can search "(PAM)" to list all audit events. See System Log filters and search for more information.

  4. Click the search icon to generate the report.

Results

The System Log page displays the following information for each audit event:

Field name Field description

Time

Timestamp of the event.

Actor

The user who initiated the event.

Event information

Details about the event. All audit events have the "(PAM)" prefix.

Target

App integration or user that received the event or action. Depending on the event type, there maybe one or more target type. For example, when you create a group, the group itself will be a target and the team will be another target.
