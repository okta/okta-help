Bind a sudo entitlement to a group

To bind a sudo entitlement to a group:

  1. Open the dashboard from an admin account.
  2. Click Projects.
  3. Select the project that contains the group to bind the sudo entitlement to.
  4. Click the Groups tab.
  5. Select the group to bind the entitlement to.
  6. Click Add Sudo Entitlement Binding.
  7. Select the sudo entitlement to bind to the group from the Sudo Entitlement dropdown.
  8. Click Submit to bind the sudo entitlement to the group.
