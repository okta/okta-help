Connect to servers through the command line

  1. Identify a server name.
  2. Open a terminal window.
  3. Enter the following command:
    sft rdp <server name>
    The client displays a list of available accounts.
  4. Specify a username.

If the AD connection isn't configured to use certificates, users may need to enter their AD password before they can connect.
