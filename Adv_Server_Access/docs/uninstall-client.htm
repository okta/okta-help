Uninstall the client

You can remove the client from any device.

Operating System Process
Ubuntu or Debian From the terminal, run  sudo apt-get remove scaleft-client-tools
Red Hat or Fedora From the terminal, run  sudo yum remove scaleft-client-tools
macOS
  1. Delete the client application located at /Applications/ScaleFT.app.
    See How to delete apps on your Mac.
  2. From the terminal, run the following command to delete the symbolic link to the client: rm /usr/local/bin/sft
Windows
  1. Open Control Panel.
  2. Click Programs and Features. The Uninstall or change a program page appears.
  3. Select ScaleFT, click Uninstall, then click Yes.
