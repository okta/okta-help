Team management

The dashboard provides access to several features and settings that can help teams better control access to server resources.

Topic Description
Projects projects consolidate and control access to server resources.
Groups groups consolidate and manage user accounts. Groups are assigned to one or more projects and control what actions and project resources are available to users.
[%=okta-feature-names.ScaleFT%] user and group attributes attributes are metadata that define information that is synced to servers. Attributes are set at the team-level by default, but teams can also attributes for the needs of each individual project.
Sudo entitlements Entitlements control what commands, directories, and executables are accessible to users.
Service users Service users allow teams to create dedicated accounts for use with automation or API access.
