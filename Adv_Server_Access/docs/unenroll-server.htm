Unenroll a server from

Teams can unenroll servers from by manually removing it from the associated project.

Teams can also stop the server agent service on the device. After four days, automatically removes the server from the project. See Manage the [%=okta-feature-names.ScaleFT%] server agent.

Manual unenrollment

You can manually unenroll a server from by removing it from the project that it's enrolled in.

To remove a server from a project:

  1. In the dashboard, click Projects and select the project to remove the server from.
  2. Select the Servers tab. This lists the servers that are currently enrolled in the project.
  3. Click the gear beside the server to unenroll and select Delete. Click Delete Server to remove the server from the project.
