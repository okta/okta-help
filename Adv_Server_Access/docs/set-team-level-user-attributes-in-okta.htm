Configure team-level user attributes

Teams can set and manage team-level user attributes from the Admin Console. Alternatively, teams can also use attributes with a default SAML configuration. This method is not recommended as teams lose the benefits of SCIM, including any custom mapping of attributes.

  1. From the Okta Admin Console, go to Applications > Applications.
  2. Click Okta Advanced Server Access to open the application settings window.
  3. Go to the Assignments tab.
  4. Identify an account in the list and click edit . The Edit User Assignment window opens
  5. Modify any of the available attributes.
  6. Click Save.
