Set project-level attribute overrides

You can override the default attribute values for user and group attributes at the project level. This allows teams to define unique values for each individual project.

  1. On the dashboard, click Projects.
  2. Click the project to open.
  3. Configure user attribute overrides.
    1. Go to the Users tab.
    2. Click the gear beside the user and select Edit Attributes. The Update Attributes page appears.
    3. Enable the checkbox for an attribute in the Project Value Override column.
    4. Specify an updated attribute value.
    5. Click Update.
  4. Configure group attribute overrides.
    1. Go to the Groups tab.
    2. Click the gear beside the group and select Edit Team Attributes. The Update Team Attributes page appears.
    3. Specify updated attribute values.
    4. Click Update.

Teams can remove project-level overrides by repeating this process and disabling the checkbox next to the modified values.
