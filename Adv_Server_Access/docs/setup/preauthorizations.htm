Create a preauthorization for a user

Teams can configure projects to require preauthorizations for users. This forces the project to only grant credentials to specific users within a limited time. This allows teams to better control access to server resources.

  1. Click Projects.
  2. Select the project to add preauthorizations to.
  3. Click Preauthorizations.
  4. Click Create Preauthorization.
  5. Enter the user to preauthorize in the User field.
  6. Enter the expiration date of the preauthorization in the first Expires At field in YYYY-MM-DD format. For example, to expire a preauthorization on March 1, 2022, enter 2022-03-01.
  7. Enter the time of day that the preauthorization will expire on the expiration date set the previous step. For example, to expire at 8PM, enter 08:00 PM.
  8. Click Submit to create the preauthorization.
