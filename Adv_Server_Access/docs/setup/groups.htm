Groups

Similar to assignments in , uses groups to explicitly assign users with associated permissions to projects, giving them access to required resources.

can sync your users and groups from the , which provides easier management of people, membership, and roles. See User Management and Configure SCIM for [%=okta-variables.ProductName%].

Two groups are automatically created for each team: everyone and owners.

  • everyone: Includes every user that belongs to the team.
  • owners: Initially includes only the user who created the team. This group can't be deleted.
