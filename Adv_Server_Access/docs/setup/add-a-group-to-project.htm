Add a group to a project

Members of a group cannot access project resources until the group is explicitly added to a project.

  1. Open the dashboard.
  2. Click Projects and open a project.
  3. Go to the Groups tab and click Add Group to Project. The Add Group to Project window appears.
  4. In the Group field, enter a group name.
  5. Optional. Select Admin to grant admin permissions.
    By default, group members only receive user-level permissions. On Linux, this grants sudo privledges.
  6. Click Add Group.
