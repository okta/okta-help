Install the client on Windows

To install the client:

  1. Go to the Windows client repository.
  2. Click a version to download.
  3. Install the client by running the installation MSI.

Uninstall the client

To uninstall the client:

  1. Open Control Panel.
  2. Click Programs and Features. The Uninstall or change a program screen appears.
  3. Select ScaleFT, click Uninstall, then click Uninstall.
