Connect to servers through the dashboard

  1. Open the dashboard.
  2. On the Home page, identify a server.
  3. Click Connect.
  4. In the Connect to Server window, select a username.
  5. Click Connect.

If teams don't configure the AD connection to use certificates, users must manually enter an AD password to connect.
