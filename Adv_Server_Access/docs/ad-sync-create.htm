Create a server sync job

Server sync jobs allow to automate the process of server discovery. Sync jobs use LDAP queries to locate servers that meet specific criteria and assign them to a project.

Before you begin

Create an Active Directory connection

Start this task

  1. Open the dashboard.
  2. Go to Connections and select an existing connection.
  3. Go to the Server Sync tab.
  4. Click Create Server Sync Job.
  5. In the job creation window, enter a job name.
  6. Configure assignment rules.See Assignment rules.
  7. Configure attributes. See Active Directory attribute mapping.
  8. Optional. Click Add to map additional attributes.
  9. Click Save & Continue.
