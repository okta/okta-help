Test a user sync job

Testing a user sync job is a highly recommended (but optional) step that ensures that the sync job can correctly discover users and their security identifiers (SIDs). After completing a test, creates a CSV file that lists any users that match your discovery rules.

Before you begin

  6. After completes the test, click View details.
