Uninstall the server agent

You can remove the server agent from any device.

Operating System Process
Ubuntu or Debian Run sudo apt-get remove scaleft-server-tools
Red Hat or Fedora Run sudo yum remove scaleft-server-tools
SuSE Linux Run sudo zypper remove scaleft-server-tools
FreeBSD Run pkg remove scaleft-server-tools
Windows
  1. Open Control Panel.
  2. Click Programs and Features. The Uninstall or change a program page appears.
  3. Select ScaleFT Server Tools, click Uninstall, then click Yes.
