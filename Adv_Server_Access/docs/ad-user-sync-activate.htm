Activate a user sync job

Activate a user sync job before it connects to your domain. During this process you must specify how often the job should attempt to discover users within the domain.

  2. Go to Connections and select an existing connection.
  3. Go to the User Sync tab.
  4. Identify an inactive job. See Create a user sync job.
  5. Optional. Run a test. See Test a user sync job.
  6. Set a schedule. To reduce server load for daily jobs, the system randomly starts the job within 60 minutes of the specified time.
  7. Click Activate.

After activation, the sync job runs immediately. To maintain performance, limits user discovery to 10,000 users per AD job.
