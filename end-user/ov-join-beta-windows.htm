Join the beta program on your Windows device

Be aware that beta versions of are experimental versions that might be less stable than released versions of . This means that the app might crash, some features might not work correctly, or you might lose some existing data.

Join the beta program

You can join the beta program at any time to test new features before they are released.

  1. Click the gear icon.
  2. Select the Join our beta program and get early access to the latest versions of Okta Verify checkbox.

  3. Optional. Click Back to return to the Accounts screen.

Leave the beta program

If you want to use the production version of , you can leave the beta program.

  1. Click the gear icon.
  2. Clear the Join our beta program and get early access to the latest versions of Okta Verify checkbox.
  3. Optional. Click Back to return to the Accounts screen.
