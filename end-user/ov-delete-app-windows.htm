Delete the app from a Windows device

You can delete the app from your Windows device at any time.

Remove all accounts from before uninstalling .

  1. If you have more than one operating system (OS) user profile with accounts, from each user profile, open and then remove all accounts from the app.

    2. See Remove an account from [%=okta-feature-names.Okta Verify%] on Windows devices.

  2. Close the app.
  3. Click Start Add or remove programs .
  4. For Okta Verify, click Uninstall.
  5. Optional. Confirm that was deleted from the device:
    1. In a browser, go to the .
    2. Click your username at the top-right of the screen, and then select Settings.
    3. In the Security Methods (or Extra Verification) section, under , verify that the applicable device is not listed. If it is, click Remove.
