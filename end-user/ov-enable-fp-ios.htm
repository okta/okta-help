Set up on iOS devices

If your organization uses , you can set up this authentication option when you add your account or later, when prompted by your technology department. With , you can access Okta-protected applications securely without providing your password.

If your organization asked you to set up , but this option isn’t available on the Account details page, go to your organization’s apps dashboard on another device. Click the arrow beside your name, select Settings Okta Verify , and then click Set Up. Follow the instructions to add a new account on your device. The new account comes with .

  1. Open and tap your account.
  2. On the Account Details page, go to Ways to Sign In Okta FastPass and tap Set Up.

  3. Read about using to sign in to apps and tap Set Up.
  4. To proceed with identity verification, tap Next or Sign In.
  5. If displays a code, tap Copy Code Continue to Verification . Otherwise, skip to the next step.

    The six-digit code that you copied might be required in a later step. If the code is hidden and you want to view it, tap the Reveal icon.
    The code is renewed every 30 seconds.


  6. Enter your email address. This is the address where you received or setup instructions from your organization.
  7. Confirm your identity. Select a security method:
    • If you select Use my password, enter your password and tap Verify.
    • If you select Enter a code, paste the code you copied in Step 4 and tap Verify.
  8. If prompted, enable Touch ID or Face ID. Tap Enable.
  9. Read any process permission requests and alerts.

    The Setup Complete screen confirms that you enabled .

  10. Tap Done to finalize the procedure.

Okta FastPass is now listed as an authentication option on the Account Details page.

If you have other accounts with within the same organization, you can choose which account is used when you sign in to apps with . See Set the default [%=okta-feature-names.Okta FastPass%] account on iOS devices.
