Send feedback from your Windows device

Send feedback to to share ideas about , or to report an issue.

Start this task

  1. From the Windows system tray, right-click the icon > Report Issue.
  2. Fill out the form: 
    • Enter a descriptive title for the issue.
    • Enter your email if you want.
    • Enter a detailed description of the issue. Consider including steps to reproduce the error, and information about your browser. Don't enter passwords or other private information.
  3. Click Submit report.

    4. If the button isn't available, you're not set up to share diagnostic information with Okta.
