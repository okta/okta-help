Lock and unlock

Lock the app to ensure that no one can access your apps if your mobile device is lost or stolen. To use the app, unlock it with a pin, or your fingerprint. On iPhone X, you can unlock with Face ID.

Before you begin

To check if you use , open and tap your account. If has a green checkmark, your organization uses the sign-in method.

Tasks

  • Unlock Okta Mobile with a PIN
  • Unlock Okta Mobile with a fingerprint
  • Unlock Okta Mobile with Face ID (iPhone X only)

Unlock with a PIN

When you access , you are prompted to set a PIN. You must enter the PIN each time you access , unless you set up fingerprint unlocking. If you forget your PIN, tap the Sign Out • Reset PIN link in the Enter your PIN window.

Unlock with a fingerprint

If your IT department enabled this feature, you can unlock with a fingerprint.

  • iOS: Ensure that you meet the following requirements:
    • You are using 5.10.1 for iOS or a later version.
    • Your device model is iPhone 5S or later, iPad (5th generation), iPad Pro, iPad Air 2, or iPad mini 3 or later.

    If you don't use yet:

    1. Set up Touch ID on your device.
    2. Download the latest version of from the Apple App Store.
    3. Turn on Touch ID verification when prompted.

    If you are already using :

    1. Set up Touch ID on your device.
    2. Update to the latest version of ( Settings  Downloads ).
    3. Sign out of , then sign back in.
    4. Turn on Touch ID verification when prompted.
  • Android: Ensure that you meet the following requirements:
    • You are using 3.2.2+ for Android.
    • Android 6.0+ device with a fingerprint sensor

    If you already have a fingerprint set up on your device, you are prompted to turn on fingerprint verification when you first download or update to 3.2.2 or higher. Otherwise, you must first set up a fingerprint on your device's Security settings. Then, go to Settings in and tap the Use fingerprint to sign in toggle.

    If you use to enroll in Okta Mobility Management with Android for work, you must re-enable fingerprint verification after enrollment. If you use Android 7.0+ devices, you must set up a separate fingerprint for your work profile. Then return to Settings in and turn on fingerprint verification.

    On Android 12, fingerprint verification is not supported if is in your work profile (Okta Mobility Management is set up).

Unlock with Face ID (iPhone X only)

If your IT department enabled this feature, and if you are using iPhone X, you can unlock with face recognition.

If you don't use yet:

  1. Set up Face ID on your device.
  2. Download the latest version of from the Apple App Store.
  3. Turn on Face ID verification when prompted.

If you are already using :

  1. Set up Face ID on your device.
  2. Update to the latest version of ( Settings  Downloads ).
  3. Sign out of , then sign back in.
  4. Turn on Face ID verification when prompted.
