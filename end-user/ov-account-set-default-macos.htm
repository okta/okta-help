Set a default account on macOS devices

If you have more than one account that uses , you can set the most frequently used account as your default account. You can also change the current default account to a different account at any time.

When you user enter you username or email during authentication, uses that account for authentication. If you do not enter a username or email, uses the default account, or a random account if a default account is not set up.

If your default account is invalid, an error appears on the Account page. To resolve this problem, re-enroll your account or choose a different account as the default.

See Re-enroll an [%=okta-feature-names.Okta Verify%] account on macOS devices.

  1. Open .

    For example, click the desktop shortcut, or if the app is running, from the menu bar extras, click the icon > Open.

  2. On the Account details screen, click the account you want as your default account.
  3. If the Account screen has a Default for indicator in the top right corner, the account you selected is already set as the default and your process is complete; otherwise, proceed to step 4.

  4. Click the Set as default for link.

    5. If you only have one account on your Account details screen, this link is not available.

    A message confirms that you successfully set the account as the default.
