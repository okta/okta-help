Use the with keyboard shortcuts

Use keyboard shortcuts to access the instead of clicking the plugin icon in your browser.

Open the plugin

Web browser Windows Mac
Chrome Alt + O Cmd + Shift + O
Internet Explorer Alt + O Cmd + Shift + O

Keyboard shortcuts for Firefox and Safari are not yet available.

Disable the plugin shortcut (only for Internet Explorer)

  1. Open Registry Editor.
  2. Go to HKEY_CURRENT_USER\Software\AppDataLow\Software\Okta\IE Plugin.
  3. Add a new DWORD DisablePopoverShortcut and set its value to one.

Close the plugin dialogs

Web browser Windows Mac
Chrome Alt + X Ctrl + Opt + X
Internet Explorer Alt + X n/a
Firefox Alt + Shift + X Ctrl + Opt + X
Safari Alt + X Ctrl + Opt + X
