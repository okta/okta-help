Manage settings on Android devices

From the Settings page, you can access documentation, send feedback about the app, join beta, or check what version is installed on your Android device.

Tasks

  • Set the display theme
  • Access the documentation
  • Send feedback
  • Join beta

Set the display theme

To choose between light and dark mode display, or apply the system display theme to , open the app, tap the menu (...) in the top-right corner and tap Settings Theme .

Access the documentation

To learn more about , access the documentation. Tap the menu (...) in the top-right corner, and go to Settings Help .

Send feedback

To share suggestions or ideas about , or report issues that you might encounter, tap the menu (...) in the top-right corner and go to Settings Send feedback .
