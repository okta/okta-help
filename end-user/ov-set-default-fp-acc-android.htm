Set the default account on Android devices

If you have multiple accounts with within the same organization, you can select one as your default. This is the account that’s used when you sign in to apps with .

If you don’t set any account as default, randomly selects an account with to verify your identity. This account becomes your default account that will be used for your subsequent sign-in attempts.

If your default account is invalid, a message appears on the Account details page. To resolve this problem, choose a different account as default, or re-enable your account if possible. To learn how to re-enable your account, see MFA reset issues.

Start this task

  1. Open and tap the account you want to use for signing in with .
  2. On the Account details page, go to Options to check if the account is already set as default for .
    • If this is already your default account, you see a green checkmark. Your task is complete. There are no further actions.

    • If this isn’t your default account, tap Set as default.
