BambooHR supported features

This table lists the features and functionality available with a BambooHR integration.

Feature Description

Import new users

Users created in BambooHR can be imported into .

Import profile updates

Updates made to a user's profile in BambooHR are downloaded and applied to the user profile.

Import user schema

Imports more user attributes from BambooHR. Also known as schema discovery.

Profile sourcing

Makes BambooHR the source for user profile attributes.

Push profile updates

Updates made to the user profile are pushed to BambooHR.
