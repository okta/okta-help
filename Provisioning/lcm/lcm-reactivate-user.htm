Reactivate a user profile

Reactivate a user profile to allow the user to access assigned applications.

  2. Enter the user name in the Search field.

    The Status column for deactivated users displays Deactivated.

  3. Click the name of the deactivated user.
  4. Click Activate.
  5. Click Activate in the Activate Person dialog.

    An activation email is sent to the user informing them that their account is active. You can edit the content of these emails on the customize email page.

A blue check mark on the Set Password and Activate button indicates Set by admin option was selected when the user was added.
