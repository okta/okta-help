Convert an individual assignment to a group assignment

When user app integration access and user profile attributes are converted from individually-managed to group-managed, they inherit the app attributes defined for members of the group. To be eligible for conversion, the user must be added to a group that is also assigned to the app.

You convert app assignments on the Assignments tab of an app integration.

  2. Enter the name of the app integration in the Search field and click the app integration name.
  3. Click the Assignments tab.
  4. Click People in the Filters list.
  5. Click Convert Assignments and select one of these options:

    • Select assignments to convert — select this option to move specific users from individually-managed to group-managed. Select individual users and click Convert selected. To convert all listed users, click Convert all assignments and click Convert all.

    • Convert all assignments — select this option and click Convert all to move all users from individually-managed to group-managed.

    6. In the Assignments tab, the entry in the Type column is now set to Group for the changed accounts.
