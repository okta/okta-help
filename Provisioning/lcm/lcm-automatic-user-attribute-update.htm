Automatically update user attributes

updates a user’s attributes in the downstream application when an app integration is assigned.

After the app integration is assigned to the user, any changes made to the user profile automatically overwrite the corresponding attribute value in the downstream application.

  2. Enter the app integration name in the Search field.
  3. Click the app integration name and click the Provisioning tab.
  4. Select To App in the Settings list.
  5. Click Edit in the Provisioning to App area.
  6. Click Enable for the Update User Attributes option and click Save.
