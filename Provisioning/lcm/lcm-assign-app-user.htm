Assign an app integration to a user

Assign the app integration to a user so the user can access and use the app integration.

  2. Enter the name of the app integration in the Search field.
  3. Click the app integration name to open the settings page.
  4. Click the Assignments tab.
  5. Click Assign and select Assign to People.
  6. In the Assign application name to People dialog box, click Assign next to a user name.
  7. Confirm the data is correct in the Assign application name to People dialog.
  8. Click Save and Go Back.

    The Assigned button for the user is disabled in the Assign application name to People dialog . This indicates that the app integration is now assigned to the user.

  9. Click Done.

    The Type column for the user displays individual.

  10. Optional. To confirm that the user was assigned to the app integration, go to Reports System Log and confirm that the Add user to application membership entry has a status of success.
