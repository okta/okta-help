Provision unprovisioned users

Users assigned to an app integration before provisioning is enabled for that app integration are not automatically provisioned.

After enabling provisioning in the app integration, you can manually provision these unprovisioned users.

  2. In the Search field, enter the name of the name of the app integration and click the app integration name in the search results.
  3. On the Assignments tab, select People in the Filters list.

    Unprovisioned users are identified with a red exclamation icon that you can select to view provisioning details.

  4. Click Provision User and then OK in the Provision User dialog box.
