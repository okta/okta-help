Automatically deactivate app users

When this functionality is enabled, automatically deactivates a user’s external application account when the user is unassigned from the app integration in or if the user's account is deactivated.

Accounts can be reactivated if the app integration is reassigned to the user in .

  4. Select To App in the Settings list.
  5. Click Edit in the Provisioning to App area.
  6. Click Enable for the Deactivate User option and click Save.
