Search for an existing app integration

Before you create a custom app integration that supports provisioning, see if an integration for the application is already available in the ().

  1. Open a web browser and go to the App Catalog (https://www.okta.com/integrations/).
  2. Enter the name of the application you want to integrate with in the Search catalog of integrations field and press Enter.

    If the app supports provisioning, the tile for the app indicates Provisioning. You can also narrow your search by selecting the Provisioning check box in the Capabilities list.
