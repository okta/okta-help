DocuSign supported features

This table lists the features and functionality available with a DocuSign integration.

Feature

Description
Push groups

Groups and their members can be pushed to integrated applications. See Manage Group Push.
Import new users

Users created in DocuSign can be imported into .

Import profile updates

Updates made to a user's profile in DocuSign are downloaded and applied to the user profile.

Push new users

Users created in are also created in DocuSign.

Push profile updates

Updates made to the user profile are pushed to DocuSign.

Push user deactivation

Deactivating a user or disabling application access in removes all user data and the user account in DocuSign. When a user is suspended, their data is not removed and they cannot access the application.

Import user schema

Imports additional user attributes from DocuSign. Also known as schema discovery.
