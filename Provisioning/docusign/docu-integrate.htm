Integrate DocuSign with

  1. Locate and record the DocuSign API account ID:
    1. Sign in to your DocuSign instance with administrator permissions.
    2. Select Settings on the top ribbon. If Settings is unavailable, click menu and select eSignature.
    3. In the left menu below Integrations, click Apps and Keys.
    4. Copy the value in the API Account ID field.
  2. If you've already added the DocuSign app to , go to step 3. If you haven't added it, add it now:
    2. Click Add Application.
    3. In the search field, enter DocuSign, and select DocuSign.
    4. Select Add.
    5. Complete the fields on the General Settings page and click Next.
    6. In the Sign On Methods section of the Sign-On Options pane, select a sign on option. If you select SAML 1.1 or SAML 2.0, click View Setup Instructions and follow the instructions.
    7. Click Done.
  4. Select DocuSign from the list of applications.
  5. Click the Provisioning tab, click Configure API Integration, and select the Enable API Integration check box.
  6. In the API Account ID field, enter the API account ID you copied in step 1.

    Error messages displayed when authenticating with DocuSign are often caused by an incorrect API Account ID. Make sure the API Account ID and DocuSign administrator credentials match.

  7. Generate the authentication token:
    1. Click Authenticate with DocuSign.
    2. Click Accept if a prompt appears.
    3. If required, enter your email address and click CONTINUE.
    4. If required, enter your password and click LOG IN.
  8. Optional. Clear the Import Groups check box if you don't want to import groups.
  9. Click Save.

  10. Optional. Change the to DocuSign provisioning settings:

    1. Click the Provisioning tab and select To App in the SETTINGS list.
    2. Click Edit.
    3. Select the check box for Create Users, Update User Attributes, or Deactivate Users to enable the functionality.
    1. Click Save.
  11. Optional. Change the DocuSign to provisioning settings:
    1. Click the Provisioning tab and select To Okta in the SETTINGS list.
    2. Click Edit in the General, User Creation & Matching, or Import Safeguard areas to edit the settings.
    1. Click Save.
  12. Optional. Assign users to the DocuSign app. See Assign applications to users.

    When you assign the DocuSign app to users, you can indicate if they can send envelopes and identify their permissions.

  13. Optional. Add custom app attributes. See Add custom attributes to apps, directories, and identity providers.
