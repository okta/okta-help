Meta Work Accounts

Follow this guide to configure provisioning for Meta Work Accounts.

Features

  • Import profile updates
  • Import user schema

Location attribute

By default, populates the user's Location with comma-separated address properties (street, city, state, and so on). If this behavior doesn’t fit your needs, you can add a Location field to AppUser through Schema Discovery.

  1. Click Refresh Attribute List.
  2. Find the Location field in the list of attributes, and then add it to the AppUser profile.
  3. Map the Location field from to Meta Work Accounts. For example: user.city>location
