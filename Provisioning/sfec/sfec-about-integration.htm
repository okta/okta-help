Learn about SAP SuccessFactors Employee Central integration

Integrate SAP SuccessFactors Employee Central with to automate provisioning and manage the employee life cycle including hiring, updating, termination, and post termination. Integration with also lets you push data originating in SAP SuccessFactors Employee Central to third-party applications.

You the use the Compound Employee application programming interface (API) for SAP SuccessFactors Employee Central to import employee data from Employee Central to . You can use one of these methodologies to import and provision employee data:

  • Import from SAP SuccessFactors Employee Central
  • SuccessFactors (EC)-driven IT provisioning (SuccessFactors EC-as-Source)

Although the focus of the integration procedure is SuccessFactors EC-driven IT provisioning, the procedures are also relevant if you want to import from SAP SuccessFactors Employee Central.

Import users from SAP SuccessFactors Employee Central

With this methodology, employee data is imported from SAP SuccessFactors Employee Central and it is used to create user profiles. After the employee data is imported into and the user profiles are created, changes made to an employee record in SAP SuccessFactors Employee Central are not replicated in the associated user profile.

SuccessFactors (EC)-driven IT Provisioning

With this methodology, user profiles are managed by SAP SuccessFactors Employee Central after the employee data is imported into and the user profiles are created. Changes made in SAP SuccessFactors Employee Central are replicated in and integrated third-party applications. This methodology is useful if you want to use SAP SuccessFactors Employee Central to manage employee and contractor access to applications.

Enterprise or Enterprise Plus versions of are required to enable SAP SuccessFactors Employee Central as a Profile Source and for custom attribute mappings.

This table lists the supported employee lifecycle events that are available with SuccessFactors EC-driven IT provisioning.

Lifecycle Event Description
New hire
  • A new employee is hired and a record is created in SAP SuccessFactors Employee Central
  • imports the new employee data and creates an user profile
  • creates accounts in integrated third-party apps (Active Directory included)
Updates
  • An employee’s attribute is changed in SAP SuccessFactors Employee Central
  • The attribute change is imported into
  • updates the attribute in integrated apps (Active Directory included)
Termination
  • An employee is terminated and the record is updated in SAP SuccessFactors Employee Central
  • The status change is imported into
  • The user profile is deactivated in and in integrated apps (Active Directory included)
Future Dated Hire
  • A new employee is hired with a future start date and a record is created in SAP SuccessFactors Employee Central
  • If the start date falls within the defined interval, the new employee data is imported into and a user profile with an active status is created
  • creates accounts in integrated third-party apps (Active Directory included)
Post Termination
  • An employee is terminated and the record is updated in SAP SuccessFactors Employee Central
  • The employee remains active in for the number of days defined in the post termination field
  • The user profile is deactivated in and in integrated apps (Active Directory included) after the number of days defined in the post termination field is reached
