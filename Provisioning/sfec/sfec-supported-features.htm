SAP SuccessFactors Employee Central supported features

This table lists the features and functionality available with a SAP SuccessFactors Employee Central integration.

Feature Description
Import New Users

Users created in a third-party application can be imported into .
Import Profile Updates

Updates made to a user profile in SAP SuccessFactors Employee Central are imported into the associated user profile.
Import User Schema Imports the user schema from SAP SuccessFactors Employee Central.
Push Profile Updates/Write-back functionality Only primary email and primary phone can be updated.

Profile Source

Makes SAP SuccessFactors Employee Central the profile source. Enterprise or Enterprise Plus versions of are required for this option.

Pre-start Interval

When Profile Sourcing is enabled, this feature allows importation of an user account into before the user's official start date. The interval is the number of days (prior to a SAP SuccessFactors Employee Central user’s stated Worker/Employee Date) before evaluates a SAP SuccessFactors Employee Central user for early import. When this feature is enabled, evaluates the pre-hire date and, if it falls within the set interval, user data is imported into and a user profile with an active status is created.

Post-Termination Interval

This feature permits late deactivation of SAP SuccessFactors Employee Central users. The user profile remains active after their end or termination date in SAP SuccessFactors Employee Central. The interval is the number of days (after a SAP SuccessFactors Employee Central user's stated end date) after evaluates a SAP SuccessFactors Employee Central user for deactivation. When this feature is enabled, evaluates the SAP SuccessFactors Employee Central Post Termination date, and the user profile remains active in for the defined number days, after the user's termination or end date.
