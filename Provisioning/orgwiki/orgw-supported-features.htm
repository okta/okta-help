OrgWiki supported features

This table lists the features and functionality available with a OrgWiki integration.

Feature

Description
Push new users

Users created in OrgWiki can be imported into .
Push profile attributes

Updates made to the user profile are pushed to OrgWiki.

Push user deactivation

Deactivating a user or disabling application access in removes all user data and the user account in OrgWiki. When a user is suspended, their data is not removed and they cannot access the application.
