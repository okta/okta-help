Integrate OrgWiki with

  1. If you've already added the OrgWiki app to , go to step 2. If you haven't added it, add it now:
    2. Click Add Application.
    3. In the Search field, enter OrgWiki, and select OrgWiki (SCIM).
    4. Select Add.
    5. Complete the fields on the General Settings page and click Done.
  2. In the , click Applications and select OrgWiki in the list of applications.
  3. Click the Provisioning tab, click Configure API Integration, and select the Enable API Integration check box.
  4. Generate the API authentication token:
    1. Click Authenticate with OrgWiki (SCIM).
    2. Click Authorize when the prompt appears.

  5. Click Save.
