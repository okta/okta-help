Add custom Rally Software attributes

Add custom attributes to the Rally Software application in to map additional user profile attributes that are not available in the standard schema. For example, the number of allowable failed log in attempts or the default landing page.

  2. Enter Rally in the Search field.
  3. Click Profile for Rally Software User.
  4. Click Add Attribute.
  5. Select additional attributes in the Pick Schema Attributes dialog box.
  6. Click Save.
  7. Click Mappings to map the new attributes.
  8. Select the Rally Software to Okta User or Okta User to Rally Software tabs.
  9. Choose an attribute or enter an expression for the attributes you added in step 5.
  10. Click Save Mappings.
