Rally Software supported features

This table lists the features and functionality available with a Rally Software integration.

Feature Description
Push New Users Users created in are also created in Rally Software.
Push User Deactivation Deactivating a user or disabling application access in removes all user data and the user account in Rally Software. When a user is suspended, their data is not removed and they cannot access the application.
Push Profile Updates Updates made to the user profile are pushed to Rally Software.
Import New Users New users created in Rally Software are downloaded and new AppUser objects are created and then compared with existing user profiles.

Import Profile Updates

Updates made to a user's profile in Rally Software are downloaded and applied to the user profile. If Rally Software is the system of record, changes made to core profile fields (email, first name, last name) are applied to the user profile. If Rally Software is not the system of record, only changes made to app-specific fields are applied to the local user profile.

Reactivate Users

Reactivating a user in reactivates the user in Rally Software.

Import User Schema

Imports additional user attributes from Rally Software. Also known as schema discovery.
