Configure Workday writeback for home and work contacts

When Workday writeback enhancement is enabled, makes separate calls to update work and home contact information. This feature requires the Home Contact Change and Work Contact Change business process security policy permissions in Workday.

Prerequisites

Set up provisioning for Workday. See Workday Provisioning

Procedures

Configure Domain Security Policy Permissions

  1. In Workday, search for Security Group Membership and Access.

  2. Search for and select your security group.

  3. Click the group name menu, and then click the group name to view the security group.

  4. Click the group name menu, and then select Security Group Maintain Domain Permissions for Security Group.

  5. Add the following domains to the Domain Security Policies permitting Put access integration permissions:

    • External Account Provisioning

    • Person Data: Home Contact Information (Optional. Add this domain to writeback home contact updates.)

    • Person Data: Work Contact Information

    • Workday Accounts

    • Worker Data: Public Worker Reports

  6. Add the following domains to the Domain Security Policies permitting Get access integration permissions:

    • Worker Data: All Positions

    • Worker Data: Business Title on Worker Profile

    • Worker Data: Current Staffing Information

    • Worker Data: Qualified Workers

    • Worker Data: Skills and Experience

    • Worker Data: Workers

  7. Click OK.

  8. Click Done.

Configure business process security policy permissions

Configure Work Contact Change

  1. In Workday, search for and select the Edit Business Process Security Policy task.
  2. Search for and select the Work Contact Change business process type, and then click OK.
  3. Scroll to the Change Work Contact Information (Web Service) section.
  4. Add your integration system security group to Security Groups.
  5. Click OK.
  6. Click Done.

Configure Home Contact Change

Optional. Complete the following steps if you want to writeback home contact updates.

  1. In Workday, search for and select the Edit Business Process Security Policy task.
  2. Search for and select the Home Contact Change business process type, and then click OK.
  3. Scroll to the Change Home Contact Information (Web Service) section.
  4. Add your integration system security group to Security Groups.
  5. Click OK.
  6. Click Done.
