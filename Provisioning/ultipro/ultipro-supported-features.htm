UKG Pro supported features

This table lists the features and functionality available with a UKG Pro integration.

Feature

Description
Import new users

Users created in UKG Pro can be imported into .
Import profile updates

Updates made to a user's profile in UKG Pro are downloaded and applied to the user profile.

Import user schema

Imports additional user attributes from UKG Pro. Also known as schema discovery.

Profile sourcing

Makes UKG Pro the profile source.

Push profile updates

Updates made to the user profile are pushed to UKG Pro.
