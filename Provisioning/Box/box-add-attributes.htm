Add additional attributes to a Box profile

I removed this topic from HOC .. per OKTA-313527

Add additional attributes to the Box profile to import or push additional Box user attributes.

  2. In the search field, enter Box.

  3. Click Profile for Box User.

  4. Click Add Attribute and complete the fields in the Add Attribute dialog box.

  5. Click Save.

  6. Click Mappings to map the new attributes.

  7. Select the Box to Okta User or Okta User to Box tabs.

  8. Choose an attribute or enter an expression for the attributes you added in step 4.

  9. Click Save Mappings.
