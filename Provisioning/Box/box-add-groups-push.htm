Assign Box to groups and configure group push

Use group push to use Active Directory (AD) or to manage Box groups and group membership. Group push lets you push groups to existing groups in Box, even if the and Box groups have the same name. If you have a Box instance with existing groups and you want to manage groups or group memberships in or AD, you migrate the Box groups to and you link the groups to the Box groups. This process makes the profile source for Box groups.

  4. Click the Assignments tab and click Assign > Assign to Groups.

  5. Click Assign to assign the group to Box.

  6. Optional. Repeat step 5 to assign additional groups to Box.
