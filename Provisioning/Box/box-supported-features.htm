Box supported features

This table lists the features and functionality available with a Box integration.

Feature

 Description
Push groups Groups and their members can be pushed to Box. See About Group Push.
Import new users Users created in Box can be imported into .
Import profile updates Updates made to a user's profile in Box are downloaded and applied to the user profile.
Push new users Users created in are also created in Box.

Push profile updates

Updates made to the user profile are pushed to Box.

Push user deactivation

Deactivating a user or disabling application access in removes all user data and the user account in Box.

Reactivate users

User accounts can be reactivated in Box.
