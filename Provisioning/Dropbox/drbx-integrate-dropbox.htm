Integrate Dropbox Business with

  1. Add the Dropbox Business app to if it has not been added previously:
    2. Click Add Application.
    3. In the Search for an application field, enter Dropbox.
    4. Select Add for Dropbox Business.
    5. Complete the fields on the General Settings page and click Next.
    6. In the Sign On Methods section of the Sign-On Options page, select a sign on option. If you select SAML 2.0, click View Setup Instructions and follow the instructions.
    7. Optional. In the Advanced Sign-on Settings area, select the Silent Provisioning check box to enable silent provisioning.
    8. Click Done.
  2. If you added the Dropbox Business app previously, on the click Applications and select Dropbox Business in the list of applications.
  3. Click the Provisioning tab, click Configure API Integration, and select the Enable API Integration check box.
  4. Click Authenticate with Dropbox Business.
  5. Enter your email and password and click Sign in.
  6. Click Save.

  7. Optional. Change the to Dropbox Business provisioning settings:

    1. Click the Provisioning tab and select To App in the SETTINGS list.
    2. Click Edit.
    3. Select the check box for Create Users, Update User Attributes, or Deactivate Users to enable the functionality.

    If you enable Deactivate Users functionality, additional deactivation options are available. The Dropbox user deactivation type list lets you select whether you want to suspend or remove users when they are deactivated in . recommends selecting Suspend.

    Select the Wipe data from linked devices check box to remove files from a user's Dropbox Business linked device when they are deactivated.

    When you select Remove in the Dropbox user deactivation type list, the File management upon user deletion list appears. Select Manage file transfer directly in Dropbox Business if you do not want to perform any action on user's files and want to manage files directly in Dropbox Business. Select Transfer files to destination team member account to manage file transfer in and complete the Dropbox email address of destination team member account and Dropbox email address of admin notification account fields.

    1. Click Save.
  8. Optional. Change the Dropbox Business to provisioning settings:
    1. Click the Provisioning tab and select To Okta in the SETTINGS list.
    2. Click Edit in the General, User Creation & Matching, or Import Safeguard areas to edit the settings.
    1. Click Save.
  9. Optional. Assign users to the Dropbox Business app. See Assign applications to users.
