Add attributes to a Salesforce profile

Schema Discovery lets to add attributes to the Salesforce profile, which makes them available to map to and from an user profile. If the Salesforce instance does not provide a list of attributes, but instead presents fields where you manually add attributes, then the Salesforce instance does not allow for Schema Discovery. You'll need to install a new instance from Okta Integration Network (OIN) to gain Schema Discovery capability.
