Salesforce supported features

This table lists the features and functionality available with a Salesforce integration.

Feature Description

Import new users

Users created in a third-party application can be pulled into .

Import profile updates

Updates made to a user profile in Salesforce are pulled into the counterpart user profile in .

Push new users

Users created in are pushed to Salesforce.

Push profile updates

Changes made to a user profile in are pushed to Salesforce.

Push user deactivation

Deactivating a user in deactivates the user's account in Salesforce.

Reactivate users

User accounts can be reactivated in Salesforce.
