Slack integration prerequisites

These are the prerequisites for integrating Slack with :

  • If you have previously integrated Slack with , run a full import to link existing assigned users to Slack users if you enable provisioning after assigning users to Slack single sign-on (SSO).

  • A Slack Plus plan for your organization to access the Slack SCIM API.
  • If you have previously integrated Slack with , reauthenticate existing Slack instances to allow schema discovery.

  • Customers needs to re-authenticate the provisioning integration every 5 years.

  • The latest profile mapping template. If you have previously integrated Slack with , contact support to request the latest profile mapping template, or use schema discovery to map new attributes. New Slack integrations use the latest profile mapping template by default.
