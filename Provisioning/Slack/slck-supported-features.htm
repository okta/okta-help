Slack supported features

Currently Multi-Channel Guest User Schema is not supported.

This table lists the features and functionality available with a Slack integration.

Feature Description
Push New Users

Users created in are also created in integrated third-party applications.
Push User Deactivation

Deactivating a user in removes a user from all organizations and teams in integrated third-party applications.
Push Profile Updates Updates made to the user profile are pushed to integrated third-party applications.
Import New Users New users created in integrated third-party applications are downloaded and new AppUser objects are created and matched with existing users.

Import Profile Updates

Updates made to a user's profile in integrated third-party applications are downloaded and applied to the user profile. If the app is the system of record, changes made to core profile fields (email, first name, last name, etc) are applied to the user profile. If the app is not the system of record, only changes made to app-specific fields are applied to the local user profile.

Group Push

Groups and group members can be pushed to remote systems. See About Group Push.

Reactivate Users

Reactivating a user in reactivates the user in integrated third-party applications.

Import User Schema

Imports additional user attributes from Slack. Also known as schema discovery
