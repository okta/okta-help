Troubleshoot Slack integrations

Use the information provided here to help you resolve common Slack integration issues.

Errors with provisioning

  • Make sure that your Slack organization has a Plus Plan subscription.
  • The Slack API doesn't support special characters (for example, +, which means entries like john+doe@email.com can't be used). Check for any special characters and remove them.

Updates don't appear in Slack

If you've previously integrated Slack with , are using group push enhancements, and updates aren't appearing in Slack, select Push Now in the . This synchronizes groups from to Slack and removes users assigned to a group in Slack but not in .
