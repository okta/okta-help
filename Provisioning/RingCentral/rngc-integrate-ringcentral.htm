    2. Click Add Application.
    3. In the Search for an application field, enter RingCentral.
    4. Select Add for RingCentral.
    5. Complete the fields on the General Settings page and click Next.
    6. In the Sign On Methods section of the Sign-On Options pane, select a sign on option. If you select SAML 2.0, click View Setup Instructions and follow the instructions.
    7. Click Done.
  2. If you added the RingCentral app previously, on the click Applications and select RingCentral in the list of applications.
  3. Click the Provisioning tab, click Configure API Integration, and select the Enable API Integration check box.
  4. Optional. Clear the Import Groups check box and click Continue in the Disable Import Groups dialog box if you do not want to add RingCentral groups to .
  5. Click Authenticate with RingCentral.

The RingCentral Sign In window appears.

  1. Enter your email address or phone number and click Next.
  2. Enter an optional extension, the password, and click Sign In.

  3. Optional. Edit the to RingCentral provisioning settings:

    2. In the Search field, enter RingCentral.
    3. Click the RingCentral link for the RingCentral app you want to edit.
    4. Click the Provisioning tab, select To App in the SETTINGS list.
    5. Click Edit in the right pane, edit the settings, and click Save.

  4. Optional. Edit the RingCentral to provisioning settings:
