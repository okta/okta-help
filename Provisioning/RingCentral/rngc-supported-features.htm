RingCentral supported features

This table lists the features and functionality available with a RingCentral integration.

Feature

 Description
Import New Users New users created in RingCentral are downloaded and new AppUser objects are created and then compared with existing user profiles.
Import Profile Updates Updates made to a user's profile in RingCentral are downloaded and applied to the user profile. If RingCentral is the system of record, changes made to core profile fields (email, first name, last name) are applied to the user profile. If RingCentral is not the system of record, only changes made to app-specific fields are applied to the local user profile.
Import User Schema Imports additional user attributes from RingCentral. Also known as schema discovery.
Push New Users Users created in are also created in RingCentral.

Push Profile Updates

Updates made to the user profile are pushed to RingCentral.

Push User Deactivation

Deactivating a user or disabling application access in removes all user data and the user account in RingCentral. When a user is suspended, their data is not removed and they cannot access the application.

Profile Source

Makes RingCentral the single source of truth for user data.
