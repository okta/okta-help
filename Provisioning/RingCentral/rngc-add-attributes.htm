Add custom RingCentral attributes

Add custom attributes to the RingCentral application in to map additional user profile attributes that are not available in the standard schema. For example, the number of allowable failed log in attempts or the default landing page.

  2. Enter RingCentral in the Search field.
  3. Click Profile for RingCentral User app you want to modify.
  4. Click Add Attribute.
  5. Select additional attributes in the Pick Schema Attributes dialog box.
  6. Click Save.
  7. Click Mappings to map the new attributes.
  8. Select the RingCentral to Okta User or Okta User to RingCentral tabs.
  9. Choose an attribute or enter an expression for the attributes you added in step 5.
  10. Click Save Mappings.
