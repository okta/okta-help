Create an instance of your on-premises app in

  1. Install the . See Install the [%=okta-feature-names.Okta Provisioning Agent%].
  2. Verify that every is listed with a green circle. The green circle indicates the agent is connected.
  3. Click Applications and enter the name of your app integration in the Search field.
  4. Click the name of the app integration and click the General tab.
  5. In the App Settings area, click Edit.
  6. Select the Enable on-premises provisioning check box.
