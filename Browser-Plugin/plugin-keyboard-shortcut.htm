Keyboard shortcuts for the Okta Browser Plugin

This topic has been replaced by Use the [%=okta-feature-names.Okta Browser Plugin%] with keyboard shortcuts

End-users can access Okta Browser Plugin using keyboard shortcuts as described below.

Open the plugin

End-users on Chrome and Internet Explorer can open the Okta Browser Plugin using the following keyboard shortcut.

Web browser Windows Mac
Chrome Alt + O Cmd + Shift + O
Internet Explorer Alt + O Cmd + Shift + O

Users see a one-time recommendation to use the shortcut when they click on the plugin popover window.

Keyboard shortcuts for Firefox and Safari are not yet available.

Disable the plugin shortcut (only for Internet Explorer)

For Internet Explorer, end-users can disable Okta Browser Plugin shortcut in Registry Editor. To disable:

  1. Open Registry Editor.
  2. Go to HKEY_CURRENT_USER\Software\AppDataLow\Software\Okta\IE Plugin.
  3. Add a new DWORD DisablePopoverShortcut and set its value to one.

Close the plugin dialogs

End-users can close the Okta Browser Plugin dialogs using the following keyboard shortcuts:

Web browser Windows Mac
Chrome Alt + X Ctrl + Opt + X
Internet Explorer Alt + X n/a
Firefox Alt + Shift + X Ctrl + Opt + X
Safari Alt + X Ctrl + Opt + X
