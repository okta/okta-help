Workflowsについては、新しい消費プランが2021年11月以降に利用可能になります。

  • Invite User to Channel（チャンネルへのユーザーの招待）

  • Send Message to Channel（チャンネルへのメッセージの送信）

接続を更新した後で、既存のカードを削除して、新しいバージョンのカードに置き換える必要があります。

影響の低いカード

次のカードは機能が拡張されますが、既存のフローには影響がありません。

  • Create Channel（チャンネルの作成）

  • New Message in Channel（チャンネルの新規メッセージ）

  • Read User（ユーザーの参照）

  • コマンドへの応答 - 遅延

  • チャンネルの検索

  • Send Direct Message（ダイレクトメッセージの送信）

  • スラッシュ・コマンド
