Okta Active Directoryパスワード同期 エージェントバージョン履歴

このページでは、Okta Active Directoryパスワード同期 エージェントの現在 および過去のバージョンを一覧表示しています。このページは、エージェントの新バージョンが一般提供（GA）および/または早期アクセス（EA）に対してリリースされるたびに更新されます。

最新のエージェントをダウンロードするには、管理者ダッシュボードから［Settings (設定)］>［Downloads (ダウンロード)］を選択し、目的のエージェントまでスクロールします。［Download (ダウンロード)］ボタンをクリックしてインストーラーを実行します。

現在のGAバージョン 1.3.6
現在のEAバージョン 該当なし

説明

リリース
1.3.6

このエージェントアップデートには、以下の修正が含まれています。

  • フォレストで重複しているSamAccountNameを検索した際に、正しいユーザーを特定できます。
  • リクエストのヘッダーにUser-Agentを含めることができます。
2018.15
1.3.5このバージョンには以下の内容が含まれています。サポートされているTLSの最小バージョンを1.2に、さらにWindows Serverの最小バージョンをWindow Server 2008に更新し、 デフォルトの設定を変更しました。2017.50
1.3.4このバージョンでは、サポートされるTLSの最小バージョンを1.2に、Windows Serverの最小バージョンをWindow Server 2008に更新しました。 2017.45
1.3.3

このバージョンには、以下が含まれています。

  • Oktaとの認証にデフォルトのUPNに加えてsamAccountNameの使用がサポートされるようになりました。
  • 特殊文字を含むパスワードのサポートを改善しました。
  • 最適化の改善
2017.40
1.3.1 このバージョンでは、特定の種類のイベントがエージェントのログファイルに記録されないという問題が修正されています。2016.10
1.3.0 このバージョンでは、SSL証明書のピン留めをサポートしています。
このバージョンでは、SSLピン留めが原因でOktaサーバーとの通信ができない環境でも新規インストールを完了できるように、管理者がWindowsレジストリを編集してSSLピン留めを無効化することができます。詳細については、Active DirectoryからOktaへのパスワードの同期をご覧ください。
また、このバージョンのWindowsインストーラーでは、インストール時にウイルススキャンの誤検出を引き起こす可能性を低減しています。		2015.52
1.2.0.2

Active Directoryパスワード同期エージェントのインストーラーのアップデートを提供します。このアップデートでは、以下の機能が更新されています。

  • Internet Explorer 7以上がマシン上にあることを確認します。
  • Windows Server 2003 R2のサポートを追加しました。

注：Windows Server 2003 R2にインストールする場合、ADパスワード同期エージェントにはInternet Explorer 7以上が必要です。

4/29/2013
1.2.0.1 バグ修正 03/26/2013
1.2.0.0
  • Active Directory Password Sync MSIファイルの提供を開始しました。
  • Active Directory Password Syncが複数のドメインコントローラーにインストールされている場合に発生する可能性のある競合状態を修正しました。
03/19/2013
1.1.1.1 以下の機能を追加しました。

03/04/2013

02/05/2013
