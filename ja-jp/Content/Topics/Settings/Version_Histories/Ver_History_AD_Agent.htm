Okta Active Directoryエージェントバージョン履歴

このページでは、Okta Active Directory (AD)エージェントの現在 および過去のバージョンを一覧表示しています。このページは、エージェントの新バージョンが一般提供（GA）または早期アクセス（EA）に対してリリースされるたびに更新されます。

最新のエージェントをダウンロードするには、Okta管理コンソールで［Settings (設定)］>［Downloads (ダウンロード)］を選択し、エージェントまでスクロールしてから［Download Latest (最新版のダウンロード)］をクリックしてインストーラーを実行します。

Okta ADエージェントの最新機能と最適なパフォーマンスを確保するために、Oktaでは最新バージョンのエージェントをダウンロードしてサーバーにインストールすることを推奨します。 複数のエージェントを実行している場合は、すべてのエージェントを同じバージョンに揃えてください。互いにバージョンの異なるエージェントが複数実行されている場合、全エージェントの機能が、最も古いバージョンと同程度に制限される可能性があります。

GAバージョンとEAバージョンが同じ場合、EAバージョンのOkta ADエージェントは利用できないため、代わりにGAバージョンをインストールする必要があります。
