バージョン6.1.0

リリース：2021年7月1日

バージョン6.1.0（SafariとIEを除くすべてのブラウザー）

ダッシュボードへのアクセス

  • Okta Browser Pluginのポップアップに表示される新しいOktaロゴをクリックして、エンド・ユーザー・ダッシュボードにアクセスできるようになりました。

Chromeユーザーの評価バナー

  • Chromeユーザーがプラグインを使用して5つ以上のアプリを起動すると、Okta Browser Pluginを評価するバナーがプラグインのポップアップに表示されます。
OKTA-368077

一部のエンド・ユーザーは、Okta Browser Pluginのトークン・エラーが原因で、Okta End-User Dashboardにアクセスできませんでした。

バージョン5.47.0

リリース：2021.06.2

バージョン5.47.0（IEを除くすべてのブラウザー）

  • Chrome、Firefox、Edgeで、プラグインのポップアップで歯車アイコンをクリックすると、プラグイン設定が新しいタブで開くようになりました。
  • 特定のアプリの設定にアクセスするには、ポップアップに表示されている各アプリの横の省略記号をクリックします。
OKTA-372430

OKTA-387202

開発者コンソールが開いていると、プラグインのポップアップによって一部のWebページの速度が低下していました。

OKTA-369511

プラグインのポップアップで、アプリを2回目に開いた後の表示に一貫性がありませんでした。

バージョン5.45.0

リリース：2021.05.0

バージョン5.45.0（すべてのブラウザー）

  • プラグインのポップアップの読み込み時間が短縮されました。
  • プラグインのデザインとイメージが更新されました。
OKTA-372649

OKTA-315155

バージョン5.40.0

リリース：2021.08.2

バージョン5.40.0（IEを除くすべてのブラウザー）

  • Okta Browser Pluginをインストールした後、エンド・ユーザーはOktaダッシュボードにリダイレクトされるようになりました。

バージョン5.39.1

リリース：2020.07.2

バージョン5.39.1（Firefox向け）

  • Okta Browser PluginがFirefox向けにバージョン53.9.1に更新されました。
  • [詳細設定]の[ローカルJavaScriptを使用]オプションが、IEを除くすべてのブラウザーで削除されました。

OKTA-315155

一部のFirefoxユーザーについて、最新のFirefox拡張サポート・リリースでOkta Browser Pluginが繰り返し失敗することがありました。

バージョン5.39.0

リリース：2020.07.0

バージョン5.39.0（すべてのブラウザー）

  • 新しいOkta End-User Dashboardに、新規ユーザーに対してダッシュボードを信頼するためにプラグイン・ポップアップを使用するように求める却下可能なバナーが表示されるようになりました。
  • プラグインの新機能が追加されたことをユーザーに警告するため、Okta Browser Pluginのアイコンに緑色の感嘆符（！）が表示されるようになりました。
  • Okta Browser Pluginの設定で、新しいオプトイン機能が利用可能になると強調表示されるようになりました。
  • Firefoxでは、ブラウザーの制限により、Okta Browser Pluginのプライバシー関連の権限を付与した後にユーザーに表示される[タブを閉じる]ボタンが削除されました。
  • Chromeでは、[パスワードの保存を提案]設定がグループ・ポリシーによって制御されている場合、ブラウザーでパスワードの保存を求められないようにするためのポップアップ設定は、エンド・ユーザーには表示されません。
  • Okta Browser Pluginのすべてのバージョンがバージョン5.39.0に更新されました。

OKTA-290791

Okta Browser Pluginのアプリ・セクションを切り替えたユーザーが、セクションの上部にリダイレクトされていませんでした。

バージョン5.38.1

リリース：2020.05.3

バージョン5.38.1（Firefox、Safari、およびEdge（Legacy）向け）

  • Safariでは、Okta拡張機能アプリのヘルプ・リンクがアプリのヘルプ・メニューに追加され、[印刷]サブメニュー項目が削除されました。

  • Okta Browser Pluginを使用してネイティブのブラウザー・プロンプトを無効にし、Oktaまたはアプリのパスワードを保存できるようになりました。「ブラウザーによってパスワード資格情報が保存されないようにする」

    を参照してください。
バージョン5.38.0

リリース：2020.05.1

バージョン5.38.0（ChromeおよびEdge（Chromiumベース）向け）

このバージョンには次のものが含まれています。

  • ユーザーは、Oktaにサインインするとき、またはOktaに保存されたアプリにサインインするときに、ネイティブ・ブラウザーのプロンプトを無効にしてOkta Browser Pluginからパスワードを保存するようにオプトインできるようになりました。「ブラウザーによってパスワード資格情報が保存されないようにする」を参照してください。
  • Firefoxで、Okta Browser Pluginのポップアップを開いたり閉じたりするためのキーボード・ショートカットが検索フィールドに追加されました。（Macの場合は⌘⇧O、PCの場合はCtrl + Shift + O）
  • 新しいOkta End-User Dashboardの場合： Okta Browser Pluginの検索機能が更新され、Okta End-User Dashboardと同じ検索精度になりました。 Okta Browser Pluginのポップアップのフォント・サイズも更新されました。
