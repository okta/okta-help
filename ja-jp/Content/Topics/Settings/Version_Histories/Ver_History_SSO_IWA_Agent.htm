Okta SSO IWA Web Appのバージョン履歴

このページには、現在 および過去のOkta統合 Windows Authentication (IWA) Web Appのバージョンが表示されます。このページは、アプリの新バージョンがリリースされるたびに更新されます（一般 提供 (GA)および/または早期アクセス (EA)）。 最新バージョンのOkta SSO IWA Web Appをダウンロードしてインストールすることで、最適なパフォーマンスが得られ、最新の機能や特徴を確保することができます。

最新のエージェントをダウンロードするには、管理者ダッシュボードから［Settings (設定)］>［Downloads (ダウンロード)］を選択し、［Download Latest (最新版のダウンロード)］をクリックして、インストーラーを実行します。

GAバージョンとEAバージョンが同じ場合、EAバージョンのOkta SSO IWA Web Appは利用できないため、代わりにGAバージョンをインストールする必要があります。
