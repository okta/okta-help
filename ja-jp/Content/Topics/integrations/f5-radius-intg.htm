F5 BigIP APMを構成してRADIUSを介したOktaとの相互運用を可能にする

本ガイドでは、APMおよびVPNアプリ向けにOkta Integration Network (OIN) F5 BIG IP RADIUSと組み合わせてOkta RADIUSサーバーエージェントを使用するためにF5 BIG IP APMを設定する方法を詳しく説明します。

注

SAML 2.0を介してOktaと統合するには、F5 BIG IP SAML for APM and VPN SAML アプリをOktaに追加します。

  1. ［Applications（アプリケーション）］タブに進みます。
  2. ［Applications（アプリケーション）］>［Add Applications（アプリケーションの追加）］を選択します。
  3. APMおよびVPN向けにF5 BIG IP SAMLを検索します。
  4. ［Add（追加）］をクリックします。

説明

RADIUSエージェントのダウンロード
  • Okta orgの ［Settings（設定） ］> ［Downloads（ダウンロード）］ページからOkta RADIUSエージェントをダウンロードします。WindowsおよびLinuxエージェントがあることにご注意ください。
  • スループット、可用性、その他の検討事項については、「Okta RADIUSサーバー エージェントのデプロイメントに関するベストプラクティス」を参照してください。
Okta RADIUSエージェントのインストール
  • 使用環境に適したWindowsまたはLinux RADIUSエージェントをインストールします。
アプリケーションの構成
  • Okta orgで、 F5 BIG IP RADIUSアプリケーションを設定します。
ゲートウェイの構成
  • F5コンソールで、F5 BIG IPを構成し RADIUSを操作します。
オプション設定の構成
  • 必要に応じて、クライアントIPレポートやグループ応答などのオプション設定を構成します。
テスト
  • 新しく統合されたエージェントをテストします。
