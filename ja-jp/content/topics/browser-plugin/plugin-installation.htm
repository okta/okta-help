プラグイン オンボーディング バナー

注： Okta Browser Pluginのオンボーディング・バナーは、エンド・ユーザー向けブラウザー・プラグインの改善されたインストールが有効になっている場合にのみ、新しいOkta End-User Dashboardに表示されます。この機能は、Chrome、Chromium Edge、Firefox上のプラグイン・バージョン5.40.0で動作します。ダッシュボードが開いているタブで自動的にフォーカスされ、更新されます。IEの場合、インストール・ヘルプの最後に[完了時にOktaダッシュボードを起動する]と表示されているチェックマークがあれば、インストールの完了時にログイン・プロンプトが開きます。Safariではこの機能ではサポートされておらず、インストール時に新しい動作はありませんが、将来のリリースでサポートされる予定です。


ChromeでOkta Browser Pluginをインストールする

ChromeでOkta Browser Pluginをインストールするには、次の手順を使用します。

  1. Chrome Storeで [Oktaブラウザー・プラグイン]ページを開きます。
  2. [Chromeに追加]をクリックします。
  3. プロンプトが表示されたら、[拡張機能を追加]をクリックする前に、Okta Browser Pluginに対してブラウザーで付与された許可を確認します。

    注意：アプリを追加したにもかかわらずタスクバーにOkta Browser Pluginが表示されない場合、タスクバーのパズルアイコンをクリックします。次に、[拡張機能を管理する]をクリックして、それが有効になっていることを確認します。 Okta Browser Pluginが有効になっていない場合は、セクションの右下にあるスイッチを切り替えてプラグインを有効にします。

  4. プラグインのインストール後、Okta End-User Dashboardにサインインしてインストールを完了します。サインインすると、Okta Browser Pluginのポップアップ・ウィンドウに組織のアプリが表示されます。

SafariでOkta Browser Pluginをインストールする

SafariでOkta Browser Pluginをインストールするには、次の手順を使用します。

  1. Mac App Storeで [Oktaブラウザー・プラグイン]ページを開きます。
  2. [Mac App Storeで表示]をクリックします。
  3. App Storeに移動したら、ウィンドウの右上にあるダウンロード・リンクをクリックして、ブラウザー・プラグインをダウンロードしてインストールします。完了すると、別のウィンドウが表示され、Oktaプラグインが正常にインストールされたことが通知されます。
  4. 別のウィンドウで[Safariを開く]をクリックします。
  5. 画面左上のツールバーにある[Safari] > [基本設定] > [拡張機能]に移動し、[Okta拡張機能アプリ]ボックスにチェックを入れて、拡張機能を有効にします。
  6. Okta End-User Dashboardにサインインしてセットアップを完了します。サインインすると、Okta Browser Pluginのポップアップ・ウィンドウに組織のアプリが表示されます。

FirefoxでOkta Browser Pluginをインストールする

FirefoxでOkta Browser Pluginをインストールするには、次の手順を使用します。

  1. Firefoxブラウザー用アドオンで [Oktaブラウザー・プラグイン]ページに移動します。
  2. [Firefoxに追加する]をクリックします。
  3. プロンプトが表示されたら、[拡張機能を追加]をクリックする前に、Okta Browser Pluginに対してブラウザーで付与された許可を確認します。
  4. プラグインのインストール後、Okta End-User Dashboardにサインインしてインストールを完了します。サインインすると、Okta Browser Pluginのポップアップ・ウィンドウに組織のアプリが表示されます。

Internet ExplorerでOkta Browser Pluginをインストールする

Internet ExplorerでOkta Browser Pluginをインストールするには、次の手順を使用します。

  1. Okta組織のEnd-User Dashboardに移動します。
  2. サインイン後、画面上部の[プラグインをインストール]プロンプトをクリックします。
  3. okta.swa.ie実行可能ファイルの[実行]または[保存]のいずれかを再度求められたら、[実行]を選択します。
  4. IE Okta Browser Pluginのインストーラーが開いたら、概要とウェルカムの両方の手順で[次へ]をクリックします。
  5. [インストールする]をクリックして、インストールを実行します。
  6. 処理が完了したら[完了]をクリックします。
  7. ブラウザーに戻り、ブラウザーによって開いた新しいウィンドウに表示される手順に従います。
  8. ブラウザを再起動した後、Okta End-User Dashboardにサインインしてセットアップを完了します。サインインすると、Okta Browser Pluginのポップアップ・ウィンドウに組織のアプリが表示されます。

EdgeでOkta Browser Pluginをインストールする

EdgeでOkta Browser Pluginをインストールするには、次の手順を使用します。

  1. Edge Storeで [Oktaブラウザー・プラグイン]ページを開きます。
  2. [取得]をクリックします。
  3. Microsoft Storeにリダイレクトされたら、[取得]をクリックしてプラグインをダウンロードし、インストールを開始します。
  4. Microsoft Storeのダウンロード・ページにリダイレクトされたら、インストールが完了するのを待ってから[起動]をクリックし、Edgeを開きます。
  5. ブラウザーで、ブラウザー・ウィンドウの右上にある省略記号をクリックして、設定ページを開きます。
  6. [拡張機能]を選択し、スイッチを切り替えてOkta Extensionをオンにします。
  7. 再度プロンプトが表示されたら、[オンにする]をクリックします。
  8. 最後に、Okta End-User Dashboardにサインインしてセットアップを完了します。サインインすると、Okta Browser Pluginのポップアップ・ウィンドウに組織のアプリが表示されます。

ChromiumでOkta Browser Pluginをインストールする

ChromiumでOkta Browser Pluginをインストールするには、次の手順を使用します。

  1. ChromiumベースのEdge Storeで [Oktaブラウザー・プラグイン]ページを開きます。
  2. [取得]をクリックします。
  3. プロンプトが表示されたら、[拡張機能を追加]をクリックする前に、Okta Browser Pluginに対してブラウザーで付与された許可を確認します。
  4. Okta End-User Dashboardにサインインしてセットアップを完了します。サインインすると、Okta Browser Pluginのポップアップ・ウィンドウに組織のアプリが表示されます。
