ダウンロードとバージョン履歴

Okta Admin Consoleの［Settings（設定）］>［Downloads（ダウンロード）］では、エージェント、Authenticator、ツールキット、Okta MobileとOkta Verifyアプリ（Androidのみ）、およびほとんどのWebブラウザー用のOkta Browser Pluginの最新バージョンをダウンロードできます。［Downloads（ダウンロード）］ページに表示されているように、一部のブラウザー用のOkta Browser PluginおよびiOS用のOkta Mobileアプリは、オンラインダウンロードストアからのみダウンロードできます。

ダウンロードを検証する

ここにリストされているエージェントの1つにファイルサイズとSHA-512ハッシュが表示されている場合は、それらを使用してファイルの整合性を検証できます。

    1. Admin Consoleで、［Settings（設定）］［Downloads（ダウンロード）］に移動します。
    2. ダウンロードするRADIUSインストーラーの横にある［Download Latest（最新をダウンロード）］リンクをクリックします。
    3. 次のいずれかのコマンドを使用して、ローカルマシンでハッシュを生成します。コマンド内のsetupは、ダウンロードしたエージェントへのファイルパスに置き換えてください。
      • Linux：sha512sum setup.rpm
      • macOS：shasum -a 512 setup.rpm
      • Windows：CertUtil -hashfile setup.exe SHA512
    4. 生成されたハッシュが［Downloads（ダウンロード）］ページのハッシュと一致することを確認します。

詳細については、エージェントのインストール手順を参照してください。
