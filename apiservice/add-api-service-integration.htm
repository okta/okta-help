Add an API Service Integration

The following API Service Integrations are available in OIN:

  • Openpath Security
  • Palo Alto Networks Cloud Identity Engine
  • Riot
  • Styra DAS

To add an API Service Integration:

  2. Click Browse App Catalog.

  6. Click Install & Authorize to grant the requested permissions and install the integration.
