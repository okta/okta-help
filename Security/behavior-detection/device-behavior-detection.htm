Add device behavior

A device behavior is used to define policies based on changes in the end user's device at login time. For example, admins can configure policies to prompt for MFA when an end user is attempting to authenticate from a new device.

Start this task

  1. In the , go to Security Behavior Detection .

  2. Click Add Behavior.
  3. From the dropdown list, select Device.
  4. In the Behavior Name field, enter a name for the behavior.
  5. In the Evaluate against past field, enter the number of recent authenticators used to evaluate this behavior.
  6. Click Save.
