Add IP behavior

An IP behavior is used to define policies based on changes in the end user's IP address at login time. For example, admins can configure policies to prompt for authenticators when an end user is attempting to authenticate from a new IP address.

Start this task

  3. From the dropdown, select IP.
  4. In the Behavior Name field, enter a descriptive name for the behavior.
  5. In the Evaluate against past field, enter the number of recent authenticators used to evaluate this behavior.
  6. Click Save.
