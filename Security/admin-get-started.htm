Get started with Okta

The Get Started guide provides five essential tasks to configure your Okta org.

When you first sign in, the Get Started guide is highlighted in the Admin Console. You can also see the guide by going to Dashboard > Getting Started.

Get Started walks you through these tasks, with links to the relevant documentation:

  1. Bring some users into Okta from Active Directory or LDAP
  2. Set up SSO (single sign on) for your first app
  3. Add another admin user
  4. Select MFA factors
  5. Create a sign on policy that requires MFA for your users
  6. Select authenticators
  7. Create a sign on policy that requires authenticators for your users

A check mark appears next to each task that you complete. When all five are complete, the Get Started option no longer appears on the main menu bar.
