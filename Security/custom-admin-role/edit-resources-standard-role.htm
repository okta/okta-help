Edit resources for a standard role assignment

Super admins can edit the resources assignments that are available to a standard admin role.

Before you begin

  • Ensure that you're signed in as a super admin.

  • See Best practices for creating a custom role assignment.

Start this task

  1. You can edit the resource assignments for a role or for an individual admin.
  2. To edit the resource assignments for a role:
    1. Go to the Roles tab. The Roles tab displays a list of standard and custom admin roles.
    2. For the standard role assignments whose resources you want to edit, select Edit View or edit assignments .
    3. On the Administrator assignment by role page, go to the Complete the assignment section.
    4. For the admin whose resource assignments you want to edit, click Edit.
    5. The Edit resources page opens, displaying a list of all the groups or applications that are assigned to the role.
  3. To edit the resource assignments for an admin:
    1. Go to the Admins tab. The Admins tab displays a list of all of the admins in your org
    2. For the admin whose resource assignments you want to edit, click Edit.
    3. The Edit resources to a standard role page opens, displaying a list of all the groups or applications that are assigned to the admin.
  4. On the Edit resources page you can:
    • Select Constrain this role to the entire organization to give the role access to all available resources of that type.
    • Use the Filter by group/application name field to search for a resource that the role is constrained to.
    • Use the Search by apps/groups field to search for a resource you want to constrain the role to. Then click Add.

      • The Edit resources to a standard role page displays 10 apps or 5 groups in the search results. If you don't see the app or group that you're looking for, try a more specific search term.

    • Use the Remove all resources button to remove all resource constraints from a role.

  5. Click Save.
