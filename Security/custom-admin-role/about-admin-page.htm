About the Administrators page

Use the Overview, Admins, Roles, and Resources tabs on the new Administrators page as a starting point for creating and editing admin role assignments. You can create and edit both custom and standard admin role assignments. You can also use it to:

  • Monitor your admin role assignments
  • Manage admins
  • Manage roles
  • Manage resources

Monitor your admin role assignments

Use the Overview tab on the Administrators page to monitor your admin role assignments at a glance. Here’s what you can do:

  • View role assignments for a specific user or group using the search.

  • Get a report of your admins, roles, and resource sets. Click Create Report. See Admin Role Assignments Report.

  • Monitor the changes made to admin role assignments in the last seven days using the Admin changes section. The numbers are updated every 24 hours.

  • Track the following items in the Overview section:

    Item Description Actions

    Total admins

    Total number of users with admin role assignments. This includes users who were granted their admin role assignments individually and through group membership.

    Click Total admins to go to the Admin role assignment reports page with the filter pre-applied for total number of admins.

    Individually assigned admins

    Total number of users who were granted their admin role assignments individually.

    Click Individually assigned admins to go to the Custom admin reporting page with the filter pre-applied for individually assigned admins.

    Group assigned roles

    Total number of groups with admin role assignments.

    Click Group assigned roles to go to the Custom admin reporting page with the filter pre-applied for group assigned roles.

    Custom roles

    Total number of custom roles created.

    Click Custom roles to go to the Roles tab on the Administrators page.

    Resource sets

    Total number of resource sets created.

    Click Resource sets to go to the Resource tab on the Administrators page.

Manage admins

Use the Admins tab on the Administrators page to:

  • View existing admins. You can filter admins by individually assigned admins and group assigned admins.
  • Create a new admin assignment.
  • Edit or delete an existing admin assignment.

Manage roles

Use the Roles tab on the Administrators page to:

  • View existing roles. You can filter roles by type and by name.
  • Create a new role.
  • Edit or delete an existing role.
  • Edit admin assignments by role.
  • Edit notification preferences.

Manage resources

Use the Resources tab on the Administrators page to:

  • View existing resources. You can filter resources by name.
  • Create a new resource set.
  • Edit or delete an existing resource set.
  • Edit admin assignments by resource set.

Notes

  • Super admin is the only role that can manage users or groups with admin privileges.
  • To prevent permission overrides, existing admin groups can be granted new roles through the Edit option only.
  • If a group admin is assigned access to a group that is later assigned an admin role, the group admin is no longer able to make any changes over the group or group members.
  • Admin roles can't be assigned to groups with more than 5,000 members.
  • Group rules don't work with admin groups. This prevents delegated admins from mistakenly increasing their or other user's administrative privileges.
  • Admins lose their permissions when they are deactivated. If you reactivate a former admin, you also need to reassign privileges to them.

  • When you modify a group’s admin role assignments or group membership, it might take a few minutes for those changes to take effect.

  • You can only get the admin reports from the Admin role assignment reports page in the Admin Console. Currently, getting reports using APIs isn’t supported.
