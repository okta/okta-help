About the Admin role assignment report page

Use the Admin role assignment report page to get reports about admin role assignments and components. You can filter by admin, role, and resource set components, a specification within these components, or a combination of both.

Use the reports to:

  • Monitor your org security.

  • Track who has access to various resources.

  • Ensure that only the right people have access.

  • Meet your audit requirements.
