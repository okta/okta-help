Authentication

Okta is the foundation for secure connections between people and technology. With Okta, you can manage MFA, password and sign-on policies, integration with existing user directories, role-based access to applications, and more.

Multifactor authentication
Add another layer of security to verifies a user's identity.
Sign-on policies
Configure policies to control how users authenticate.
Identity providers
Allow users to sign in with identity account credentials.
Delegated authentication
Allow users to authenticate with various credentials.
RADIUS integrations
Manage authorization and access to on-premises resources using the RADIUS protocol.
