Mobile administrators

Mobile admins can perform actions related to mobile policies, sign-on policies, mobile devices, and Okta Mobile.

Mobile admins have the following permissions:

  • Manage mobile devices
  • Manage mobile policies
  • Manage OMM applications
  • Manage OMM - Wifi
  • Manage sign-on policies
  • View users and groups
  • View apps and app instances
  • Run reports
  • Edit MFA factors
