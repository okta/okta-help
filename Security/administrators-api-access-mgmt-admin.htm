API Access Management administrators

API Access Management admins perform tasks for the Okta API Access Management product. For information on this product, see API Access Management.

API Access Management admins have the following permissions:

  • Create the authorization server
  • Create scopes
  • Create claims
  • Create access policies
  • Create rules for each access policy
  • Test your authorization server configuration
  • Delete an authorization server
  • Rotate keys manually
