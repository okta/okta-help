Define geolocation for a dynamic zone

You can create dynamic zones based on geographic regions.

An update to the universal ISO standard for region codes and country codes has resulted in some discrepancies between new codes for China and the codes that appear in . As a result, recommends that you update your region codes for China by editing any affected dynamic zones to prevent any issues.
