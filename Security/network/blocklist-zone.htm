Add a network zone to a blocklist

You can add IP zones and dynamic zones to a blocklist.

Clients from the blocked network zones can't access any URL for the org and requests are automatically blocked before any type of policy evaluation occurs.

  2. In the list of zones, click Edit for the network zone you want to modify.
  3. Select Block access from IPs matching conditions listed in this zone.
  4. Click Save.

Two network zones are created by default when the Multiple Network Zones feature is enabled. One of them can be used for blocklisting IP addresses.
