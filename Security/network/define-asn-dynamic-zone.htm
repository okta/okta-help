Include autonomous system numbers for dynamic zones

You can create a dynamic zone based on an autonomous system number (ASN) that uniquely identifies each network on the internet. An ASN represents an entire network of IP addresses.

Specifying an ASN can help you reduce overhead as an alternative to entering a list of multiple IP addresses.

Before you begin

Create a dynamic zone. See Create a dynamic zone for instructions.
