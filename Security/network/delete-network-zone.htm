Delete a network zone

When you delete an IP zone or dynamic zone, all rules that use the deleted zone are affected.

Before you Begin

  • If the network zone to delete is the only zone in any rule, you can't delete the zone and you receive an error message. Edit the rule to use a different zone, then attempt to delete the network zone again.
  • If the network zone to delete isn't the only zone in any rule you can delete the zone. The zone is removed from all the rules where it's specified.
  • If the network zone that you want to delete is active and is used by other rules, including rules in for customers who upgraded to , make the network zone inactive before you attempt to delete it.
