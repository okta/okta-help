Generate a Proxy IP report

A report of Proxy IP addresses can be generated to help you identify which proxies can be configured in s. This information is used by to correctly identify the client IP where the request originated. The proxy report lists all Proxy IP addresses that have been used to connect to your org, including proxies that may or may not be trusted.

Before you begin

Ensure that:

Any IP you add as a trusted proxy is trustworthy, due to the possibility that some Proxy IP addresses may potentially be malicious.

Generate a Proxy IP report:

  2. Go to Proxies.
  3. Click the Proxy IP Usage links to open the Generate Report dialog.
  4. Click Generate Report to generate the report.
