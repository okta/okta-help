Add IPs to a network zone from the System Log

You can add an IP address that appears in a System Log event directly to a network zone. This eliminates the need to copy the IP address and go to the Network menu.

You must have super or org admin permissions to perform the following task.

Add an IP address to a network zone from the System Log

  2. Find the event and IP address in the System Log.
  3. Click the More icon ( ) and Add to zone.
  4. In the Add to IP zone dialog, select the following:
    • Add to zone: Select the network zone to which to add the IP address.
    • IP type: Select from Proxy or Gateway.
  5. Click Save.
