Read-only administrators

Read-only admins have view access to most data in the . Like report admins, read-only admins are unable to edit data. The only settings that read-only admins can change are their own notifications and API token settings.

Read-only admins can view the following:

  • Users
  • Groups
  • Apps and app instances
  • Reports
  • Okta settings
  • System Log
  • Authorization server scope
  • Device Trust enablement settings
  • Mobile policies
  • Wifi policies
  • Device details
  • Sign-on policies
