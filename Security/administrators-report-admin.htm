Report administrators

Report admins have view-only access to reporting features.

Report admins can perform the following actions:

  • View and download reports
  • View and download the System Log

These are the only menu items available to report admins in . The report admin role is suitable for an auditor who needs to view data but not edit it. However, you can combine report admin permissions with other admin roles to expand their privileges. For example, assigning report admin permissions to a user who is also a group admin enables the user to view all reports and System Log events, but only drill down into report details for the groups they manage.
