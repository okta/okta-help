Reset multifactor authentication for end users

If you need to reset multifactor authentication (MFA) for your end users, you can choose to reset configured factors for one or multiple users.

This action resets all configured factors for any user that you select. End users are required to set up their factors again. This action applies to all factors configured for an end user. You can't select specific factors to reset.

Reset all factors for one or multiple users

  2. Click Reset multifactor.
  3. Select the users for whom you want to reset multifactor authentication.
  4. Click Reset Multifactor Authentication. A confirmation prompt appears. Click Reset to proceed. This action can't be undone.

Reset one or multiple factors for a single user

This action resets any configured factor that you select for an individual user. The user must set up their factors again.

From the Admin Console:

  2. Click the user whose multifactor authentication that you want to reset.
  3. Click More Actions > Reset Multifactor.
  4. Select the factors that you want to reset and then click either Reset Selected Factors or Reset All. A confirmation prompt appears. Click Reset to proceed. This action can't be undone.
