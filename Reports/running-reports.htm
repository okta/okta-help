Run reports

When you run a report, some reports are:

  1. Available only in the Admin Console UI:

    • SMS Usage
    • Yubikey
    • SAML Capable Apps
    • Provisioning Capable Apps

  2. Available for download in a comma-separated values (CSV) format:

    • Suspicious activity

    • Deprovision details

    • Proxy IP Usage

  3. Delivered by email:

    • Okta Usage

    • Okta Password Health

    • Current Assignments

    • MFA Usage

    • Application Usage
    • App Password Health
    • Admin Role Assignments

    See Receive reports by email.

Simply select a report and follow the steps as prompted.
