バージョン5.39.0

リリース：2020.07.0

バージョン5.39.0（すべてのブラウザー）

  • 新しいOkta End-User Dashboardに、新規ユーザーに対してダッシュボードを信頼するためにプラグインポップオーバーを使用するように求める却下可能なバナーが表示されるようになりました。
  • プラグインの新機能が追加されたことをユーザーに警告するため、Okta Browser Pluginのアイコンに緑色の感嘆符（！）が表示されるようになりました。
  • Okta Browser Pluginの設定で、新しいオプトイン機能が利用可能になると強調表示されるようになりました。
  • Firefoxでは、ブラウザーの制限により、Okta Browser Pluginのプライバシー関連の権限を付与した後にユーザーに表示される［Close tab（タブを閉じる）］ボタンが削除されました。
  • Chromeでは、［Offer to Save Passwords（パスワードの保存を提案）］設定がグループポリシーによって制御されている場合、ブラウザーでパスワードの保存を求められないようにするためのポップアップ設定は、エンドユーザーには表示されません。
  • Okta Browser Pluginのすべてのバージョンがバージョン5.39.0に更新されました。

OKTA-290791

バージョン5.38.1

リリース：2020.05.3

バージョン5.38.1（Firefox、Safari、およびEdge（Legacy）向け）

  • Safariでは、Okta拡張機能アプリの［Help（ヘルプ）］リンクがアプリの［Help（ヘルプ）］メニューに追加され、［Print（印刷）］サブメニュー項目が削除されました。

  • Okta Browser Pluginを使用してネイティブのブラウザープロンプトを無効にし、Oktaまたはアプリのパスワードを保存できるようになりました。「ブラウザーでパスワード資格情報が保存されないようにする」を参照してください。

バージョン5.38.0

リリース：2020.05.1

バージョン5.38.0（ChromeおよびEdge（Chromiumベース）向け）

このバージョンには次のものが含まれています。

  • ユーザーは、Oktaにサインインするとき、またはOktaに保存されたアプリにサインインするときに、ネイティブブラウザーのプロンプトを無効にしてOkta Browser Pluginからパスワードを保存するようにオプトインできるようになりました。「ブラウザーでパスワード資格情報が保存されないようにする」を参照してください。
  • Firefoxで、Okta Browser Pluginのポップアップを開いたり閉じたりするためのキーボードショートカットが検索フィールドに追加されました。（Macの場合は⌘⇧O、PCの場合はCtrl + Shift + O）
  • 新しいOkta End-User Dashboardの場合：Okta Browser Pluginの検索機能が更新され、Okta End-User Dashboardと同じ検索精度になりました。Okta Browser Pluginのポップオーバーのフォントサイズも更新されました。
