組織の読み取り

Zendeskから組織を読み取ります。

入力

フィールド定義タイプ必須
読み取り条件
Organization ID (組織ID)このカードが読み取る組織の一意の識別番号。IDは次の形式のドキュメントのURLで見つかります：http://<your domain>.zendesk.com/agent/organizations/< Organization ID (組織ID) >数値FALSE
External ID (外部ID)組織の一意のID文字列FALSE

出力

フィールド定義タイプ 
組織 
Organization ID (組織ID)ユーザーが関連付けられている組織のIDコード数値 
Organization URL (組織のURL)組織のWebサイトのURL文字列 
External ID (外部ID)組織の一意のID文字列 
Name (名前)組織の一意の名前文字列 
Created At (作成日時)組織の作成日時文字列 
Updated At (更新日時)組織の最終更新日時文字列 
Domain Names (ドメイン名)組織に関連付けられているドメイン名リスト 
Details (詳細)住所など、組織に関する詳細文字列 
Notes (メモ)組織に関するメモ文字列 
Group ID (グループID)この組織のユーザーが開いたチケットに割り当てられるグループID数値 
Shared Tickets? (共有チケット？)[true]の場合、組織のエンドユーザーはお互いのチケットを表示可能ブーリアン 
Shared Comments (共有コメント)[true]の場合、組織のエンドユーザーはチケットのお互いのコメントを表示可能ブーリアン 
Tags (タグ)組織に関連するタグリスト 
Custom Fields (カスタムフィールド)組織に関連付けられているカスタムフィールドの動的に生成されたリスト「カスタムフィールドタイプについて」を参照してください。各種
