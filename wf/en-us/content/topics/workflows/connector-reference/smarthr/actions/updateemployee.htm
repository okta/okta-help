Field Definition Type Required
Clear Empty Values?

Choose an option to update input values:

  • If No, keep an existing value if there's no new input value provided. Only fields with new input values are updated.

  • If Yes, clear an existing value even if no new input value is provided. All specified input fields are updated, even if there's no new value provided for a field.

    The following input fields can't be cleared, even if you select Yes for the Clear Empty Values? option. If you don't provide a new value, the existing value remains.
