Assign User to Application for SSO and Provisioning

Assign a user to an application with credentials and an application-specific profile. Profile mappings defined for the application are first applied before applying any profile properties specified in the request.

The input and output fields in this card are dynamically generated based on your instance.

Options

Field Definition Type Required
Application Choose an application to which you want to assign the user. Choose Enter Application ID to select an application by its ID.

The maximum applications displayed is 100 application instances. When a timeout occurs due to server or network connectivity issues, the Enter Application ID is the only option that displays.
