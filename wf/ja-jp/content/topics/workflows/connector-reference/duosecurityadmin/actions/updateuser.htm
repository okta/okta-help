Status（ステータス） グループの認証ステータス。active、bypass、またはdisabled。

テキスト
Mobile OTP Enabled? (モバイルOTPが有効？) trueの場合、グループ内のユーザーはDuo Mobileによって生成されるパスコードを使用して認証できます。falseの場合、グループ内のユーザーはDuo Mobileによって生成されるパスコードを使用して認証できません。Note（注）：Duo Mobileパスコードがグローバルで無効な場合、この設定は無効です。

True/False
Push Enabled? (プッシュが有効？) trueの場合、グループ内のユーザーはDuo Pushを使用して認証できます。falseの場合、グループ内のユーザーはDuo Pushを使用して認証できません。Note（注）：Duo Pushがグローバルで無効な場合、この設定は無効です。

True/False
SMS Enabled? (SMSが有効？) trueの場合、グループ内のユーザーはSMSパスコードを使用して認証できます。falseの場合、グループ内のユーザーはSMSパスコードを使用して認証できません。Note（注）：SMSパスコードがグローバルで無効な場合、この設定は無効です。

True/False
Voice Enabled? (音声が有効？) trueの場合、グループ内のユーザーは音声コールバックを使用して認証できます。falseの場合、グループ内のユーザーは音声コールバックを使用して認証できません。Note（注）：音声コールバックがグローバルで無効な場合、この設定は無効です。

True/False
Is Enrolled?（登録？） ユーザーが認証に使用可能な電話、ハードウェアトークン、U2Fトークン、またはセキュリティキーを持っている場合はtrue。それ以外の場合はfalse。

True/False
Last Directory Sync（最終ディレクトリ同期日時） ディレクトリ同期によるユーザーへの最終更新のタイムスタンプ。ユーザーが外部ディレクトリと同期したことがない場合、またはユーザーを最初に作成したディレクトリがDuoから削除されている場合はnull。

テキスト
Last Login（前回のログイン） このユーザーの最終ログイン日時（UNIXタイムスタンプ）。ユーザーがログインしたことがない場合はnull。

テキスト
Status（ステータス） ユーザーのステータス。
  • ［active（アクティブ）］：ユーザーは二次認証を完了しなければなりません
  • ［bypass（バイパス）］：ユーザーは一次認証を完了後、二次認証をバイパスします
  • ［disabled（無効）］：ユーザーはログインできません
  • ［locked out（ロックアウト）］：過剰な認証試行回数のためにユーザーは自動的にロックアウトされました
  • ［pending deletion（削除保留中）］：Duo管理者によって管理パネルから、非アクティブのためシステムにより、またはディレクトリ同期により、ユーザーに削除マークが付けられました。7日以内に復元されない場合、ユーザーは恒久的に削除されます。

テキスト

True/False

  • ［mobile_otp（モバイルOTP）］：デバイスはDuo Mobileを使用してパスコードを生成できます
