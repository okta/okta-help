Search Users（ユーザーの検索）

1人以上ユーザーを検索します。

オプション

フィールド 定義 タイプ 必須
Return [Single user (単一のユーザー)]か[All matching users (一致するすべてのユーザー)]を選択します。

Single user (単一のユーザー): こちらを選択した場合、このカードは検索条件に最初に一致したユーザーを返します。特定のユーザーを探している場合にこのオプションを使用します。

All matching users (一致するすべてのユーザー): こちらを選択した場合、このカードは検索条件に一致するユーザーの集合を返します。フィルタリングされたユーザーのセットを探している場合にこのオプションを使用します。

 ドロップダウン TRUE
