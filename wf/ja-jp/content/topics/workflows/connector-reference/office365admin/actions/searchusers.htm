ユーザーの検索

プロパティ値を使ってAzure Active Directoryからユーザーを検索します。

サポートされる検索処理には、［Exact Match（完全一致）］と［Starts With（先頭一致）］が含まれます。［Exact Match（完全一致）］はプロパティが完全に一致するユーザーを返すのに対して、［Starts With（先頭一致）］はプロパティが指定の値から始まるユーザーを返します。

Search Usersアクションカードは、最大で4,000ユーザーを返します。

オプション

フィールド 定義 タイプ 必須
Result Set（結果セット）

［First Matching Record（最初の一致レコード）］を返すか、［All Matching Records（すべての一致レコード）］を返すかを選択します。結果は、カードの出力フィールドにオブジェクトのリストとして返されます。

 ドロップダウン TRUE
Search By（検索条件）

  • [Exact Match (完全一致)]は、プロパティーが入力値と完全に一致するユーザーを返します。

  • [Starts With (先頭一致)]は、プロパティーの先頭が入力値と一致するグループを返します。

 ドロップダウン TRUE
