Asanaコネクター

Asanaを使用してチームの作業を整理および追跡します。

  • Asanaアカウントの承認
  • Asana connectorアクションカード

Asanaアカウントの承認

Asanaカードを初めてフローに追加すると、接続の構成が求められます。構成をセットアップすることで、Asanaアカウントへの接続、情報の保存、コネクターを利用した次回のフロー構築時の構成の再利用が可能になります。このコネクターを使用してフローを作成するたびにサインインする必要がなくなります。AsanaコネクターはOAuth 2.0プロトコルを使用します。「承認」を参照してください。

Asana connectorアクションカード

アクション

説明

Add Task（タスクの追加）

Asana組織内にタスクを作成します。

Add User to Team（チームへのユーザーの追加）

Asanaインスタンス内のチームにユーザーを追加します。

Add Users to Project（プロジェクトへのユーザーの追加）

廃止。Asanaインスタンス内のプロジェクトにユーザーを追加します。

Create Membership（メンバーシップを作成）

Asanaインスタンスにメンバーシップを作成します。

Create Project（プロジェクトの作成）

Asana組織内に新規プロジェクトを作成します。

Create User（ユーザーを作成）

Asanaインスタンスにユーザーを追加します。

Custom API Action（Custom API Action）

Asana APIに対してカスタムの認証済みHTTP呼び出しを行います。

Deactivate User（ユーザーの非アクティブ化）

Asanaインスタンス内のユーザーを無効化します。

Delete Membership（メンバーシップを削除）

メンバーシップIDを使ってAsanaインスタンス内のメンバーシップを削除します。

List Tasks（タスクの一覧表示）

Asanaインスタンス内のタスクを一覧表示します。

プロジェクトの参照

プロジェクトのデータを参照します。

Read Project Tasks（プロジェクトタスクの参照）

プロジェクトに関連付けられたタスクを参照します。

Read User（ユーザーの参照）

ユーザープロファイルを参照します。

ユーザータスクの参照

Asana インスタンス内のユーザータスクを参照します。

チームからのユーザーの削除

Asanaインスタンス内のチームからユーザーを削除します。

Remove Users from Project（プロジェクトからのユーザーの削除）

廃止。Asanaインスタンス内のプロジェクトからユーザーを削除します。

Search Projects（プロジェクトの検索）

プロジェクトを名前で検索し、プロジェクトのIDを特定します。

Search Tasks（タスクの検索）

Asanaインスタンス内のタスクを検索します。

Search Teams（チームの検索）

チームを名前で検索し、チームのIDを特定します。

Search User（ユーザーの検索）

ユーザーを名前またはメールで検索し、ユーザーのAsana IDを特定します。

Update Project（プロジェクトを更新）

Asanaインスタンス内の既存のプロジェクトを更新します。

Update Task（タスクを更新）

タスクのデータを更新します。

Update User（ユーザーの更新）

Asanaインスタンス内のユーザーの情報を更新します。
