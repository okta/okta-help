メールを検索する

Office 365 Mailでメールを検索します。

オプション

フィールド 定義 タイプ 必須
メール ユーザーメールの種類を指定します。

  • My Mail（マイメール）：接続ユーザーが所有します。

  • User Mail（ユーザーメール）：接続ユーザーと共有または委任されたユーザー。

 ドロップダウン TRUE
Result Set（結果セット） レスポンスで返されるメールの数を指定します。

検索結果をフィルタリングする方式を選択します。

  • ［First Matching Record（最初の一致レコード）］：一致する最初のレコードが返されます。
  • First 200 Matching Records（最初に一致する200レコード）：最初に一致する200レコードが返されます。
  • ［Stream Matching Records（一致レコードのストリーミング）］：一致するすべてのレコードが親フローからヘルパーフローに渡されます。カードに［Streaming（ストリーミング）］入力セクションが追加され、カスタム拡張可能フィールドのストリーミングと追加のためのヘルパーフローを選択できます。
 ドロップダウン TRUE
Search By（検索条件） 入力テキスト値の比較方法を指定します。すべてのテキストフィールドを検索します。検索に使用する演算子を選択します。
  • Exact Match（完全一致）：検索が入力されたテキスト/数字の値に完全に一致する場合にメールを返します。例：「重要」を含む件名をすべて検索します。
  • Contains（次の値を含む）：入力されたテキスト/数字の値に部分的に一致するメールを返します。例：［Sender（送信者）］フィールドで「Smith」を含む単語を検索します。
  • Starts With（指定の値で始まる）：検索が入力されたテキスト/数字の値で始まる場合にメールを返します。
 ドロップダウン TRUE
Date Filter（日付フィルター） レスポンスでメールの日付を指定します。すべてのテキストフィールドを検索します。検索に使用する演算子を選択します。
  • Greater Than（より長い）：［Start Date Filter（開始日フィルター）］フィールドよりも長い期間に送受信されたメールを返します。たとえば、日付が2022年1月1日より後のメールをすべて検索します。
  • Less Than（より短い）：［End Date Filter（終了日フィルター）］フィールドよりも短い期間に送受信されたメールを返します。たとえば、［End（終了）］フィールドで「3/19/2022」より小さい値を検索します。
  • Range（範囲）：日付の値が特定の日付範囲内にあるメールを返します。
  • None（なし）：日付検索は使用されません。
 ドロップダウン TRUE

多数のレコードを返す方法については、「ヘルパーフローによる一致のストリーミングオプションのセットアップ」を参照してください。
