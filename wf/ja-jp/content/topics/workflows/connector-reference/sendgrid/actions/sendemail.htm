Send Email（メールの送信）

SendGridを使用してメールを送信します。

オプション

フィールド 定義 タイプ 必須

Use Template? (テンプレートを使用？)

メールのテンプレートを使用するかどうかを示します。

［Yes（はい）］に設定すると、［Template ID（テンプレートID）］および［Dynamic Template Data（動的なテンプレートデータ）］フィールドが入力として追加されます。

［No（いいえ）］に設定すると、［Subject（件名）］、［Substitutions（代替）］、［Content（コンテンツ）］フィールドが入力として追加されます。

ドロップダウン

TRUE

With Bypass List Management?（リスト管理のバイパスあり？）

［Yes（はい）］に設定すると、スパムレポートリストをバイパスし、メールが確実に受信者に届くようにできます。バウンスとサブスクライブ解除のリストはチェックされたまま残ります。ただし、スパムレポート以外のリストのアドレスではメッセージを受信しません。

このフィルターをbypass_list_managementフィルターと組み合わせることはできません。

［Yes（はい）］に設定すると、［Bypass List Management（リストの管理のバイパス）］入力フィールドが追加されます。

［No（いいえ）］に設定すると、［Bypass Spam Management?（スパム管理をバイパス？）］、［Bypass Bounce Management?（バウンス管理をバイパス？）］、［Bypass Unsubscribe Management?（サブスクライブ解除管理をバイパス？）］が入力として追加されます。

［Bypass List Management（リスト管理のバイパス）］入力フィールドをその他の［Bypass（バイパス）］フィールドと組み合わせることはできません。

ドロップダウン

TRUE

With Spam Check?（スパムチェックあり？）

［Yes（はい）］に設定すると、スパムレポートリストをバイパスし、メールが確実に受信者に届くようにできます。バウンスとサブスクライブ解除のリストはチェックされたまま残ります。ただし、スパムレポート以外のリストのアドレスではメッセージを受信しません。

［Yes（はい）］に設定すると、［Threshold（しきい値）］および［Post to URL（ポスト先URL）］フィールドが必須になります。

［No（いいえ）］に設定すると、入力に［Span Check（スパンチェック）］は表示されません。

ドロップダウン

TRUE

入力

フィールド 定義 タイプ 必須
メール

Template ID（テンプレートID）

メールのテンプレートIDを指定します。

件名とコンテンツ（テキストまたはHTML）が含まれるテンプレートは、パーソナライズまたはメッセージレベルで指定される件名とコンテンツの値をオーバーライドします。

文字列

TRUE

From（送信元）

メッセージ配信に使用される「From（送信元）」メールアドレス。

このアドレスは、Twilio SendGridアカウント内の検証済み送信者である必要があります。

文字列

TRUE

To（宛先）

意図した受信者のメールアドレス。

文字列

TRUE

CC

メールのコピーを受信する受信者の配列。この配列の各オブジェクトには、受信者のメールアドレスが含まれている必要があります。

メールは、最大で1,000人のCC受信者に送信できます。

テキストのリスト

FALSE

BCC

メールのブラインドカーボンコピーを受信する受信者の配列。この配列の各オブジェクトには、受信者のメールアドレスが含まれている必要があります。

メールは、最大で1,000人のBCC受信者に送信できます。

テキストのリスト

FALSE

Headers (ヘッダー)

JSONのキーと値のペアを使用して、メールの取り扱い指示を指定できます。次のヘッダーは上書きできません。

  • x-sg-id

  • x-sg-eid

  • received

  • dkim-signature

  • Content-Type

  • Content-Transfer-Encoding

  • To

  • 開始

  • 件名

  • Reply-To

  • CC

  • BCC

オブジェクト

FALSE

Substitutions (代替)

代替により、動的なトランザクション型テンプレートを使用せずにデータを挿入できます。

このフィールドを動的なトランザクション型テンプレートと併用しないでください。このテンプレートは、template_idがd-で始まることで識別されます。

このフィールドは、"substitution_tag":"value to substitute"のパターンに従うキー/値ペアの集合です。

オブジェクト

FALSE

Dynamic Template Data（動的なテンプレートデータ）

動的なテンプレートデータは、動的なトランザクション型テンプレートでHandlebars構文を使用することで利用できます。

このフィールドを動的なトランザクション型テンプレートと併用しないでください。このテンプレートは、template_idがd-で始まることで識別されます。

このフィールドは、"variable_name":"value to insert"のパターンに従うキー/値ペアの集合です。

オブジェクト

FALSE

Custom Arguments（カスタム引数）

このパーソナライズに固有の値で、メールやそのアクティビティデータと共に搬送されます。

このパラメーターに入力される任意の文字列は、使用するカスタム引数とみなされます。

このフィールドの最大長さは10,000バイトです。

オブジェクト

FALSE

Send At（送信時刻）

メールを配信する時刻を指定するUNIXタイムスタンプを受け入れます。

72時間より先の送信はスケジューリングできません。ピーク時間を避けてメールをスケジューリングすることをお勧めします。ほとんどのメールは正時または半時の最初にスケジューリングされ、送信されます。ピーク時間を避けてメールをスケジューリングすると（たとえば、10時53分）、遅延率を抑えられる可能性があります。

数値

FALSE

Categories（カテゴリー）

このメールに関連付けるカテゴリー。

テキストのリスト

FALSE

Batch ID (バッチID)

同時に送信されるメールのバッチを表すID。

リクエストにバッチID値を含めることで、そのメールをそのバッチに含めることができます。そのバッチの配信をキャンセルまたは一時停止することもできます。

文字列

FALSE

Reply To（返信先）

返信やバウンスが返されるメールアドレス。

文字列

FALSE

Attachments（添付ファイル）

含める添付ファイルを指定できる、オブジェクトの配列。

オブジェクトのリスト

FALSE
  • content（コンテンツ）：添付するファイル。

ファイル

TRUE

  • type（タイプ）：添付するコンテンツのMIMEタイプ。

    例：text/plain、text/html

文字列

FALSE
  • filename（ファイル名）：添付ファイルのファイル名。

文字列

TRUE

  • disposition（処置）：添付ファイルのコンテンツの処置。添付ファイルの表示方法を指定します。

    たとえば、inlineを指定すると、添付ファイルは自動的にメッセージ内に表示されます。attachmentを指定すると、添付ファイルが表示されるまでに、ファイルを開く、またはダウンロードするなど、何らかのアクションが必要になります。

文字列

FALSE
  • content_id（コンテンツID）：メールのテンプレートID。件名とコンテンツ（テキストまたはHTML）が含まれるテンプレートは、パーソナライズまたはメッセージレベルで指定される件名とコンテンツの値をオーバーライドします。

文字列

FALSE

Subject (件名)

メールの件名。このフィールドは、個別化で設定された件名行により上書きされることがあります。

文字列

TRUE

Content（コンテンツ）

メールのコンテンツを指定するためのオブジェクトの配列。

  • type（タイプ）

  • value（値）

オブジェクト

TRUE

Spam Check (スパムチェック)

Threshold (しきい値)

フィルタリングのレベルを示す割り当て済みのSpamAssassin値。

範囲は1～10で、数値が小さいほどフィルタリングが厳しくなります。

ドロップダウン

FALSE

フッターのコンテンツ。

文字列

 数値

IP Pool Name（IPプール名）

メールの送信元となるIPプールの名前。

文字列

FALSE

Bypass List Management?（リスト管理をバイパス？）

Trueに設定すると、すべてのサブスクライブ解除グループと抑制がバイパスされ、すべての受信者にメールが確実に送信されます。

このフィールドは、すべての受信者がメールを受信することが必要な緊急時にのみご利用ください。このフィルターを、その他のバイパスフィルターと組み合わせることはできません。

ブール値

FALSE
