Read Employee（従業員の参照）

IDとメールアドレスに基づいて従業員に関する情報を取得します。

入力

フィールド 定義 タイプ 必須
Employee（職員）
ID or Email（IDまたはメール）

ユーザーの一意の識別子。識別子は、Personioインスタンス内の従業員のURLを使って特定できます。

たとえば、URLがhttps://example.personio.de/staff/details/1234567であれば、従業員IDは1234567となります。

 テキスト TRUE

出力

フィールド 定義 タイプ
結果
ID 従業員の一意のID テキスト
First Name（名） 従業員の名 テキスト
Last Name（姓） 従業員の姓 テキスト
Email（メール） 従業員のメールアドレス。これは、インポート時の従業員データの割り当てに使用される一意の識別子として事前に定義されます。 テキスト
Gender（性別） 従業員の性別 テキスト
Status（ステータス） 従業員のステータス：Onboarding（オンボーディング）、Active（アクティブ）、Inactive（非アクティブ）、Leave（休職中） テキスト
Position（役職） 社内での現在の役職 テキスト
Supervisor（上司）

従業員の上司

  • ID（テキスト）- 上司の一意のID

  • First Name（名、テキスト）- 上司の名

  • Last Name（姓、テキスト）- 上司の姓

  • Email（メール、テキスト）- 上司のメール

 オブジェクト
Employment Type（雇用形態） 従業員が内部で雇用されているか、外部で雇用されているかを示します テキスト
Weekly Hours（1週間あたり労働時間） 従業員が契約上合意している1週間の労働時間。この値は従業員の給与にリンクされます。 テキスト
Hire Date（雇用日） 労働契約が有効になった日付 日時
Contract End Date（契約終了日） 有期労働契約が終了する日付 日時
Termination Date（終了日） 雇用が終了する日付。この日付を超えると、従業員のステータスは自動的にInactive（非アクティブ）に設定されます 日時
Termination Type（終了タイプ） 雇用終了の種別 テキスト
Termination Reason（終了理由） 終了の理由 テキスト
Probation Period End Date（試用期間終了日） 試用期間の終了日 日時
Created At（作成日時） 従業員レコードの作成日 日時
Last Modified（最終変更） 従業員レコードが最後に更新された日付 日時
Subcompany（子会社）

子会社がセットアップされている場合に、従業員が勤務する子会社

  • ID（テキスト）- 子会社の一意のID

  • Name（名前、テキスト） - 子会社の名称

 オブジェクト
Office（オフィス）

従業員が勤務するオフィス

  • ID（テキスト）- オフィスの一意のID

  • Name（名前、テキスト）- オフィスの名称

 オブジェクト
Department（部署）

従業員が勤務する部署

  • ID（テキスト）- 部署の一意のID

  • Name（名前、テキスト）- 部署の名称

 オブジェクト
Cost Center（コストセンター）

関連するコストセンター。複数のコストセンターを選択し、それぞれに異なる請求割合を割り当てることができます。

  • ID（テキスト）- コストセンターの一意のID

  • Name（名前、テキスト）- コストセンターの名称

  • Percentage（割合、数値）- コスト割り当ての割合（パーセント）

 オブジェクトのリスト
Public Holidays（祝祭日）

従業員の祝祭日

  • ID（テキスト）- 祝祭日の一意のID

  • Name（名前、テキスト）- 祝祭日の名称

  • Country（国、テキスト）- 祝祭日が適用される国

  • State（都道府県、州）- 祝祭日が適用される都道府県、州

 オブジェクト
Absence Entitlements（欠勤資格）

従業員の欠勤資格

  • ID（テキスト）- 欠勤資格の一意のID

  • Name（名前、テキスト）- 欠勤資格の名称

  • Category（カテゴリ、テキスト）- 欠勤資格のカテゴリ

  • Entitlement（資格、テキスト）- 欠勤資格の種別

 オブジェクトのリスト
Work Schedule（勤務スケジュール）

従業員の勤務スケジュール

  • ID（テキスト）- 勤務スケジュールの一意のID

  • Name（名前、テキスト）- 勤務スケジュールの名称

  • Valid From（有効開始、テキスト）- 勤務スケジュールの開始

  • Monday（月曜、テキスト）- 予定勤務時間

  • Tuesday（火曜、テキスト）- 予定勤務時間

  • Wednesday（水曜、テキスト）- 予定勤務時間

  • Thursday（木曜、テキスト）- 予定勤務時間

  • Friday（金曜、テキスト）- 予定勤務時間

  • Saturday（土曜、テキスト）- 予定勤務時間

  • Sunday（日曜、テキスト）- 予定勤務時間

 オブジェクト
Fixed Salary（固定給）

従業員の固定給

  • Value（値、数値）- 従業員の給与

  • Currency（通貨、テキスト）- 通貨の種別

 オブジェクト
Salary Interval（給与間隔） 給与の支払い間隔 テキスト
Hourly Salary（時給）

従業員の時給

  • Value（値、数値）- 従業員の給与

  • Currency（通貨、テキスト）- 通貨の種別

 オブジェクト
Vacation Day Balance（残り休暇日数） 利用できる休暇日数 数値
Last Day of Work（勤務終了日） 雇用の最終日 日時
Profile Picture（プロフィール画像） 従業員のプロフィール画像のURL テキスト
Team（チーム）

従業員のチーム

  • ID（テキスト）- チームの一意のID

  • Name（名前、テキスト）- チーム名

 オブジェクト
Custom Fields（従業員カスタム項目） ユーザー定義の従業員項目。接続ごとに動的に生成されます。 各種
