ボードの一覧表示

monday.comの選択したワークスペースにあるすべてのボードを一覧表示します。

オプション

フィールド 定義 タイプ 必須
Workspace（ワークスペース）

ワークスペースを動的に検索するか、ワークスペース識別子を入力するかを選択します。

  • ［-- Enter Workspace ID --（-- ワークスペースIDを入力 --）］：アカウント内のすべてのワークスペースを動的に取得します。

  • ワークスペースのIDと名前を組み合わせた文字列のリストを表示します。例：HRワークスペース（2309811）。メインワークスペースはメインワークスペース（デフォルト）と表示されます。

 ドロップダウン TRUE
Result Set（結果セット）

検索が返す結果を選択します。

  • First 100 Records（最初の100レコード）

  • ［Stream Matching Records（一致レコードのストリーミング）］：このオプションを選択すると、一致するすべてのユーザーレコードが親フローからヘルパーフローに渡されます。［Streaming（ストリーミング）］入力セクションがカードに表示されます。このセクションでは、ストリーミングおよびカスタム拡張可能フィールドの追加に使用するヘルパーフローを選択できます。

 ドロップダウン TRUE

入力

フィールド 定義 タイプ 必須
Workspace（ワークスペース）

この入力フィールドは、［-- Enter Workspace ID --（-- ワークスペースIDを入力 --）］をオプションとして選択すると表示されます。

メインワークスペースにはID値はありません。デフォルトワークスペース内のすべてのボードを取得する場合は、［Workspace（ワークスペース）］ オプションで［Main Workspace(default)（メインワークスペース（デフォルト）］を選択します。

ID ワークスペース識別子。 数値 TRUE

Board（ボード）

State（状態）

ボードの現在の状態：

  • All（すべて）

  • Active（アクティブ）（デフォルト）

  • Archived（アーカイブ済み）

  • Deleted（削除済み）

ドロップダウン TRUE
Kind（種類）

ボードの種類：

  • Public（パブリック）

  • Private（プライベート）

  • Share（共有）

ドロップダウン FALSE
Order By（並べ替え順序）

ボードを取得するときに使用される検索順序：

  • Created At（作成日時）

  • Used at（使用日時）

ドロップダウン FALSE
Streaming（ストリーミング）
Flow（フロー）

このフィールドは、［Result Set（結果セット）］オプションでStream Records（レコードのストリーミング） を選択した場合に表示されます。

［Choose Flow（フローを選択）］をクリックして、検索結果ストリームを受信する宛先ヘルパーフローを選択します。［Choose（選択）］をクリックして確定します。

オプションとして、［Click or drop here to create（クリックまたはここにドロップして作成）］の下の空のフィールドをクリックし、データをヘルパーフローに渡すカスタム拡張可能フィールドを追加します。これらのフィールドは、ヘルパーフロー内の[State (状態)]出力オブジェクトの下にキーと値のペアとして追加されます。

 フロー FALSE
Record limit（レコード制限）

このフィールドは、［Result Set（結果セット）］オプションでStream Records（レコードのストリーミング） を選択した場合に表示されます。

ストリーミングするレコードの数を指定します。

  • ［Record Limit（レコード制限）］フィールドに0を設定すると、レコードは返されません。

  • ［Record Limit（レコード制限）］フィールドに0より大きい値を設定すると、最大で指定された最大レコード数までのレコードが返されます。

  • ［Record Limit（レコード制限）］フィールドが空、null、または未選択の場合、すべてのレコードが返されます。

  • デフォルト値は500,000です。

  • 有効な範囲は0～500,000です。

 数値 FALSE
