Read Employee（従業員の参照）

従業員のIDを使ってBambooHRから従業員情報を読み取ります。

［出力］セクションには、拡張可能フィールドを使ってその他のユーザーフィールドを追加できます。特定のIDまたはエイリアスを使って追加すると、カードはそれを出力として返します。

入力

フィールド 定義 タイプ 必須

ID

BambooHRによって自動的に割り当てられる従業員ID。

この入力は読み取り専用です。

テキスト

TRUE

出力

フィールド

定義

タイプ

ACA Status（ACAステータス）

従業員のACA（Affordable Care Act）ステータス。

このフィールドを直接更新することはできませんが、employmentStatusテーブルにある従業員ステータスのマッピングに基づいて計算されます。

状態の種類：contractor（契約）、full-time（フルタイム）、intern-eligible（インターン適格）、intern-ineligible（インターン不適格）、part-time-eligible（パートタイム適格）、part-time-ineligible（パートタイム不適格）、seasonal（季節）。

テキスト

Address Line 1（住所行1）

従業員の住所の1行目。

テキスト

Address Line 2（住所行2）

従業員の住所の2行目。

テキスト

Age（年齢）

従業員の年齢。この値を変更するには、［Birth Date（生年月日）］フィールドを更新する必要があります。

テキスト
