ユーザーの検索

プロパティー値によってOffice 365のユーザーを検索します。サポートされている検索処理には完全一致および先頭一致が含まれます。「完全一致」はプロパティーが完全に一致するユーザーを返し、「先頭一致」はプロパティーが該当する値でユーザーを返します。

ユーザーの検索アクションカードは最大で4,000個のユーザーを返します。

オプション

フィールド 定義 タイプ 必須
Result Set（結果セット） 最初の一致レコードを返すか、すべての一致レコードを返すかを選択します。これはカード上の出力フィールドにオブジェクトのリストとして返します。 ドロップダウン TRUE
Search By（検索条件） [Exact Match (完全一致)]は、プロパティーが入力値と完全に一致するユーザーを返します。

[Starts With (先頭一致)]は、プロパティーの先頭が入力値と一致するグループを返します。

 ドロップダウン TRUE

［Exact Match（完全一致）］ で表示される入力フィールドのプロパティーは、［Starts With（先頭一致）］ のものとは異なります。
