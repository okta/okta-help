ユーザーの作成

Duo Security Adminの新規ユーザーを作成します。

入力

フィールド定義タイプ必須
User (ユーザー)
Username (ユーザー名)ユーザーのプライマリーDuoユーザー名。文字列TRUE
First Name (名)ユーザーの名。文字列FALSE
Last Name (姓)ユーザーの姓。文字列FALSE
Full Name (フルネーム)ユーザーのフルネーム。文字列FALSE
Email (メール)ユーザーのメールアドレス。文字列FALSE
Status (ステータス) ユーザーのDuoスタータス。
  • [active (アクティブ)]：ユーザーは二次認証を完了する必要があります
  • [bypass (バイパス)]：ユーザーは一次認証を完了後、二次認証をバイパスします
  • [disabled (無効)]：ユーザーはログインできません
ドロップダウンFALSE
Alias 1 (エイリアス1)追加のユーザー名エイリアス。文字列FALSE
Alias 2 (エイリアス2)追加のユーザー名エイリアス。文字列FALSE
Alias 3 (エイリアス3)追加のユーザー名エイリアス。文字列FALSE
Alias 4 (エイリアス4)追加のユーザー名エイリアス。文字列FALSE
注追加のユーザー情報用のフリーテキストフィールド。文字列FALSE

出力

フィールド定義タイプ
User（ユーザー）
User ID (ユーザーID)

ユーザーの一意識別子。

文字列
Username (ユーザー名)ユーザーのプライマリーDuoユーザー名。文字列
First Name (名)ユーザーの名。文字列
Last Name (姓)ユーザーの姓。文字列
Full Name (フルネーム)ユーザーのフルネーム。文字列
Email (メール)ユーザーのメールアドレス。文字列
Alias 1 (エイリアス1)追加のユーザー名エイリアス。文字列
Alias 2 (エイリアス2)追加のユーザー名エイリアス。文字列
Alias 3 (エイリアス3)追加のユーザー名エイリアス。文字列
Alias 4 (エイリアス4)追加のユーザー名エイリアス。文字列
Created（作成日時）

オブジェクトが作成された日時。

文字列
Groups (グループ)
Name（名前）

グループの名前。

文字列
ID (ID)グループのID文字列
Description (説明)グループの説明。文字列
Status (ステータス)グループの認証ステータス。[active (アクティブ)]、[bypass (バイパス)]、または[disabled (無効)]。文字列
Mobile OTP Enabled? (モバイルOTPが有効？) trueの場合、グループ内のユーザーはDuo Mobileによって生成されるパスコードを使用して認証できます。falseの場合、グループ内のユーザーはDuo Mobileによって生成されるパスコードを使用して認証できません。

Duo Mobileパスコードがグローバルで無効な場合、この設定は無効です。

文字列
Push Enabled? (プッシュが有効？) trueの場合、グループ内のユーザーはDuo Pushを使用して認証できます。falseの場合、グループ内のユーザーはDuo Pushを使用して認証できません。

Duo Pushがグローバルで無効な場合、この設定は無効です。

文字列
SMS Enabled? (SMSが有効？) trueの場合、グループ内のユーザーはSMSパスコードを使用して認証できます。falseの場合、グループ内のユーザーはSMSパスコードを使用して認証できません。

SMSパスコードがグローバルで無効な場合、この設定は無効です。

文字列
Voice Enabled? (音声が有効？) trueの場合、グループ内のユーザーは音声コールバックを使用して認証できます。falseの場合、グループ内のユーザーは音声コールバックを使用して認証できません。

音声コールバックがグローバルで無効な場合、この設定は無効です。

文字列
Is Enrolled? (登録？)ユーザーが認証に使用可能な電話、ハードウェアトークン、U2Fトークン、またはセキュリティキーを持っている場合はtrue。それ以外の場合はfalse。文字列
Last Directory Sync (最終ディレクトリ同期日時)ディレクトリ同期によるユーザーへの最終更新のタイムスタンプ。ユーザーが外部ディレクトリと同期したことがない場合、またはユーザーを最初に作成したディレクトリがDuoから削除されている場合はnull。文字列
Last Login (最終ログイン日時)このユーザーの最終ログイン日時（UNIXタイムスタンプ）。ユーザーがログインしたことがない場合はnull。文字列
Status (ステータス) ユーザーのステータス。
  • [active (アクティブ)]：ユーザーは二次認証を完了する必要があります
  • [bypass (バイパス)]：ユーザーは一次認証を完了後、二次認証をバイパスします
  • [disabled (無効)]：ユーザーはログインできません
  • [locked out (ロックアウト)]：過剰な認証試行回数のためにユーザーは自動的にロックアウトされました
  • [pending deletion (削除保留中)]：Duo管理者によって管理パネルから、非アクティブのためシステムにより、またはディレクトリ同期により、ユーザーに削除マークが付けられました。7日以内に復元されない場合、ユーザーは恒久的に削除されます。
文字列
Phones (電話)
Activated? (有効化？)Duo Mobileによって電話が有効化されているかどうかを示します。[true (true)]または[false (false)]。文字列
Capabilities (機能) デバイスで使用可能な要素のリスト。次のいずれかです。
  • [push (プッシュ)]：デバイスはDuo Pushで有効化されます
  • [phone (電話)]：デバイスは電話を受けることができます
  • [sms (sms)]：デバイスはSMSパスコードのバッチを受信できます
  • [mobile_otp (モバイルOTP)]：デバイスはDuo Mobileを使用してパスコードを生成できます
文字列
Encrypted (暗号化)AndroidまたはiOSデバイスファイルシステムの暗号化ステータス。[Encrypted (暗号化)]、[Unencrypted (非暗号化)]、または[Unknown (不明)]のいずれか。他のプラットフォームの場合は空白。文字列
Extension (内線)内線電話番号文字列
Fingerprint (指紋)AndroidまたはiOS電話機が生体認証用に構成されているかどうかを示します。[Configured (構成済み)]、[Disabled (無効)]、または[Unknown (不明)]のいずれか。他のプラットフォームの場合は空白。文字列
Last Seen (最終確認)Duoのサービスと電話機にインストールされた有効なDuo Mobileアプリの間の最終連絡のタイムスタンプ。デバイスがDuo Mobileを有効化したことがない場合、またはプラットフォームがDuo Mobileをサポートしていない場合は空白。文字列
Model (モデル)電話機のモデル。文字列
Name (名前)電話機のラベル。文字列
Number (番号)Phone Number (電話番号)。文字列
Phone ID (電話ID)電話機のID。文字列
Platform (プラットフォーム)電話機のプラットフォーム。[unknown (不明)]、[google android (Google Android)]、[apple ios (Apple iOS)]、[windows phone 7 (Windows Phone 7)]、[rim blackberry (RIM BlackBerry)]、[java j2me (Java J2ME)]、[palm webos (Palm webOS)]、[symbian os (Symbian OS)]、[windows mobile (Windows Mobile)]、または[generic smartphone (汎用スマートフォン)]のいずれか。文字列
Postdelay (ポスト遅延)内線番号のダイヤル後、入力要求を発話する前の待機時間 (秒)。文字列
Predelay (プレ遅延)電話番号の取得後、内線番号をダイヤルする前の待機時間 (秒)。文字列
Screenlock (画面ロック)AndroidまたはiOS電話機で画面ロックが有効かどうかを示します。[Locked (ロック)]、[Unlocked (ロック解除)]、または[Unknown (不明)]のいずれか。他のプラットフォームの場合は空白。文字列
SMS Passcode Sent? (SMSパスコードの送信？)SMSパスコードが電話機に送信されたかどうかを示します。[true (true)]または[false (false)]のいずれか。文字列
Tampered (改ざん)iOSまたはAndroidデバイスが脱獄またはルート化されているかどうかを示します。[Not Tampered (改ざんなし)]、[Tampered (改ざんあり)]、または[Unknown (不明)]のいずれか。他のプラットフォームの場合は空白。文字列
Type (タイプ)電話機のタイプ。[unknown (不明)]、[mobile (携帯電話)]、または[landline (固定電話)]のいずれか。文字列
Tokens (トークン)
Token ID (トークンID)ハードウェアトークンの一意識別子。文字列
タイプハードウェアトークンのタイプ。文字列
Serial (シリアル)ハードウェアトークンのシリアル番号。type (タイプ)とともにハードウェアトークンを一意に識別するために使用します。文字列
TOTP Step (TOTPステップ)サポートされているすべてのトークンタイプに対してnull。文字列
U2F Tokens (U2Fトークン)
Date Added (追加日)U2FトークンがDuoに登録された日付。文字列
Registration ID (登録ID)U2Fトークンの登録識別子。文字列
WebAuthn資格情報
Credential Name (資格情報名)WebAuthn資格情報のラベル。文字列
Date Added (追加日)WebAuthn資格情報がDuoに登録された日付。文字列
Label (ラベル)WebAuthn資格情報のタイプを示します。[Security Key (セキュリティキー)]または[Touch ID (タッチID)]のいずれか。文字列
WebAuthn Key (WebAuthnキー)WebAuthn資格情報の登録識別子。文字列
注追加のユーザー情報。文字列
