フローのエクスポートとインポート

個別のフローまたは複数のフローを格納したフォルダーをエクスポートおよびインポートできます。

自身が複数の組織に所属している場合、または組織外の他者とフローまたはテーブルを共有したい場合、エクスポートコマンドとインポートコマンドを使用して、フローとテーブルを自身と共有できます。

個別のフロー：

  • 個別のフローに対するエクスポートコマンドの使用

  • フローの使用 - 個別のフローからのフローカードのエクスポート

  • 個別のフローに対するインポートコマンドの使用

フォルダー：

  • フォルダーに対するエクスポートコマンドの使用

  • フォルダーに対するインポートコマンドの使用

ヘルパーフロー：

  • フロー制御の使用 - ヘルパーフローのフローカードの呼び出し

個別のフローに対するエクスポートコマンドの使用

任意のフォルダーのリストビューまたはタイルビューで、[Export（エクスポート）]アイコンをクリックし、任意のローカルフォルダーを選択します。フローが「.flow」ファイルとしてエクスポートされます。

フローの使用 - 個別のフローからのフローカードのエクスポート

関数カードを使用し、個別のフローをJSONオブジェクトとしてエクスポートするには、Export Flowを参照してください。

個別のフローに対するインポートコマンドの使用

個別のフローをWorkflowsインスタンスにインポートするには：

  1. .flowファイルがあるか確認します。
  2. インポートしたフローを格納するフォルダーを選択し、フォルダー名の隣の歯車アイコンをクリックします。
  3. フォルダーメニューから[Import（インポート）]をクリックします。
  4. [Upload（アップロード）]をクリックして.flowファイルを選択するか、ファイルをダイアログにドラッグアンドドロップして、アップロードを開始します。

フォルダーに対するエクスポートコマンドの使用

エクスポートコマンドは、フォルダーメニューにあります。フォルダー名の隣の歯車アイコンをクリックすると、メニュー アイコンが有効になります。

フォルダー内のすべてのフロートテーブルをエクスポートできます。フォルダーの中身の一部をエクスポートすることはできません。

エクスポートコマンドを使用すると、ファイルが生成され、コンピューターにダウンロードされます。ファイル名は.folderで終わり、フォルダー名には、エクスポートされたフローとテーブルがすべて含まれます。ファイルを別のユーザーと共有したり、自分でファイルを使用して、フローとテーブルを別の組織にインポートしたりできます。

エクスポートコマンドで中身を共有すると、すべてのアカウント情報が削除されます。共有するのはフローのロジックとテーブルのスキーマだけです。受信者は、フローを使用するためには、新しいアカウントを指定する必要があります。

注

フロー間のリンクは維持されます。たとえば、1つのフローが子フローを呼び出してリストを処理した場合、フロー間のリンクが維持されます。リンクされたフローがエクスポートされたフォルダーに格納されていない場合、警告が表示されます。リンクされたフローを格納しなければ、インポートやエクスポートは行えません。

フォルダーのFolders - Export Folderカードの使用

関数カードを使用し、1つ以上のフローを含むフォルダーを単一のJSONオブジェクトとしてエクスポートするには、Export Folderを参照してください。

フォルダーに対するインポートコマンドの使用

フォルダーの中身をWorkflowsインスタンスにインポートするには：

  1. .folderファイルがあるか確認します。
  2. インポートしたフローを格納するフォルダーを選択し、フォルダー名の隣の歯車アイコンをクリックします。
  3. フォルダーメニューから[Import（インポート）]をクリックします。
  4. [Upload（アップロード）]をクリックして任意のファイルを選択するか、ファイルをダイアログにドラッグアンドドロップして、アップロードを開始します。

インポートしたフローは、新しいフローとして表示されます。このフローにはデータも、アプリケーションへの接続も含まれていません。インポートされたテーブルもすべて空です。すべてのアプリケーションを再接続する必要があります。

インポートしたフローがエクスポートに含まれていなかったフローを参照している場合（たとえば、リストのアイテムを処理するためにヘルパーフローが呼び出される場合）は、代わりのフローを指定して、インポートを完了する必要があります。

フロー制御の使用 - ヘルパーフローのフローカードの呼び出し

Call Flow関数カードを使用すると、ヘルパーフローの呼び出しと、親フローとヘルパーフロー間の移動が簡単に行えます。

  1. 親フローで、[Function（関数）]メニューから[Flow Control - Call Flow card（フロー制御 - フロー カードの呼び出し）]を選択します。

  2. [Call Flow card（フローカードの呼び出し）]で、[Choose Flow（フローの選択）]をクリックします。

  3. [Select flow（フローの選択）]ダイアログで、ヘルパーフローを格納しているフォルダーに移動してこれを選択し、[Choose（選択）]をクリックします。子フローが親フローに追加されます。

    任意で、[Select flow（フローの選択）]ダイアログのいずれかのフローの隣の[Open（開く）]をクリックして、フローを開き、更新します。ヘルパーフローをCall Flow関数カードから直接開くこともできます。
