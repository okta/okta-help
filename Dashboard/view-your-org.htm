View your org at a glance

Use the Overview section to view the number of active users, groups, and SSO app integrations in your org. Numbers that exceed 9,999 are rounded to the nearest ten thousand. These numbers refresh daily at midnight (PST/PDT). You can see the weekly changes for active users and groups, and tasks for your SSO app integrations, using a View by type link.

The number of SSO apps doesn't include first-party apps (for example, , Browser Plugin, and Dashboard), as they don't count against any app integration limits.

The group count includes only Okta-sourced groups, not those defined in on-premise user management applications such as Active Directory.

Use the Org changes section to monitor the org changes made by admins in the last seven days. Click the item to view details.

Only administrators with super admin permissions can view org changes.

To go directly to the People, Groups, or Apps pages, click the People, Groups, or Apps heading.
