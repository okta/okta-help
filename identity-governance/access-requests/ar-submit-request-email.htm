Submit requests by email

Each team is automatically assigned an email address. Requesters can contact a team's email to create requests if you have one or more active Request Types.

For example, a requester can email finance-team@akto.com to create a request for the Finance team. They can also review the email request from their web app.

If an email request doesn't have enough information or the system is unable to determine a Request Type, the requester receives an email prompting them to finish submitting the request from the web app.

Requesters can cancel a submitted request from the web app, if a team member or approver hasn't taken any action on it. They receive a notification about the cancellation based on the notification preferences that they’ve set up.

also cancels a request if the requester's access to the web app is revoked or their status in changes to suspended or deactivated.
