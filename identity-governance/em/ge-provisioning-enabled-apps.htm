Enable

Enable Governance Engine to manage third-party application entitlements in Okta.

Before you begin

Sign in as a super admin and ensure that you're assigned to the Okta application.

Start this task

  2. Select the application that you want to enable Governance Engine for.

  3. On to the General tab, scroll to the Identity Governance section.

  4. Click Edit.

  5. From Governance Engine dropdown menu, select Enabled.

  6. Click Save. Refresh the page to view the Governance tab for the application.

To switch off Governance Engine for an application, select Disabled from the Governance Engine dropdown. Switching Governance Engine on or off is logged as an event in the System Log.

When you disable Governance Engine for an application, all existing entitlements are deleted and users lose access to their entitlements and entitlement bundles. You must recreate entitlements when you enable Governance Engine.
