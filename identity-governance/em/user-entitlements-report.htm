User entitlements report

Use this report to view entitlements assigned to users. For example, you can filter reports to include all entitlements assigned to users for an application or a specific entitlement, entitlement value, or bundle assigned to users.

These reports can also help you meet audit and compliance requirements.

Before you begin

  • Sign in as a super admin, org admin, read-only admin, mobile admin, or reports admin.

  • Ensure that you have apps with Governance Engine enabled and have created entitlements.

  • Disable your browser’s pop-up blocker.

Get report

  2. In the Entitlements and Access section, click User Entitlements.

Field

Value
Application Enter the name of the application to include.
Entitlement Select an entitlement. You must select an application before searching for the entitlement.
Entitlement Value Select an entitlement value. You must select an entitlement before searching for the entitlement value.
Entitlement Bundle Select an entitlement bundle. You must select an application before searching for the entitlement bundle.
