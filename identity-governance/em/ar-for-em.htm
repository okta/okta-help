Access Requests automates the process of requesting access to apps, resources, and entitlement bundles.

You can’t assign bundles directly to your users from the Admin Console. Complete the following tasks to allow your users to request bundles:

  • Create an entitlement bundle

  • Configure a Request Type for entitlement bundles

After a user requests an entitlement bundle, you or the request assignee must handle the request. Review the Past access requests report for all entitlement bundles requested by your users.
