Manage bundles

Edit or delete entitlement bundles to directly update or revoke the entitlements you assigned to users as a part of the bundle. When you delete a bundle, it's also removed from the Access Requests configuration lists that use it. The entitlements themselves remain unaffected.

Before you begin

  • Check that you’ve enabled Governance Engine for the application.

  • Check that you have existing bundles and determine which configuration lists and Request Types use them.

Edit bundles

  1. Select the bundle and open the Actions dropdown menu associated with it.

  2. Click Edit to make your changes.

  3. Click Save bundle.

The changes take effect everywhere the bundle is referenced, including Request Types in Access Requests.

Delete bundles

  1. Select the bundle and open the Actions dropdown menu associated with it.

  2. Click Delete.

  3. Click Delete bundle.

Deleting a bundle revokes the entitlements that you granted to users as a part of the bundle. The bundle is also removed from existing campaigns, Request Types, and policies that reference it.

To confirm that the user’s entitlements were revoked, check the Events table in the System Log.
