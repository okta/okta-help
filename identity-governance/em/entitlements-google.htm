Google Workspace requirements

To work with , requires that you set the Manage licenses on create and update and Manage roles on create and update attributes for each user. You can set these attributes for an app instance or for individual users.

Set attributes for an app instance

Setting the attributes for the app instance applies the attributes to all users to whom the app is assigned.

  2. Find your app instance, and then click Mappings.

  3. Select Okta User to Google Workspace.

  4. Set manageLicenses to true to enable Manage licenses on create and update.

  5. Set manageRoles to true to enable Manage roles on create and update.

  6. Click Save Mappings.

Set attributes for a user

  2. Select the app instance.

  3. Go to the Assignments tab.

  4. From the options menu for the user, select Edit user assignment.

  5. Set Manage licenses on create and update to true.

  6. Set Manage roles on create and update to true.

  7. Click Save user profile.
