Create a bundle

Create entitlement bundles to reference them in Request Types.

Before you begin

  • Enable Governance Engine for the app and create entitlements. See Enable [%=okta-feature-names.Governance Engine%].

Start this task

  1. Click Create bundle.

  2. Enter a name and description for the bundle.

  3. In the Choose entitlements section, search for and select an entitlement and its corresponding value from the Entitlement and Values dropdown menus.

  4. Optional. Click + Add entitlement and repeat the previous step to add more entitlements to the bundle.

  5. Click Create.

You can view the bundles on the Bundles tab. Select a bundle to view the entitlements grouped in it. You can manually sync your Access Requests resources to use a bundle that you created. See Sync resources from Okta.

You must be assigned to the Request Access app to use the bundles in .
