Microsoft Office 365 requirements

To work with , Microsoft Office 365 requires that you set the Manage licenses on create and update attribute for each user. You can set this attribute for an app instance or for individual users.

Set attributes for an app instance

Setting the attributes for the app instance applies the attributes to all users to whom the app is assigned.

  2. Find your app instance and click Mappings.

  3. Select Okta User to Microsoft Office 365.

  4. Set manageLicenses to true to enable Manage licenses on create and update.

  5. Click Save Mappings.

Set attributes for a user

  2. Select your Microsoft Office 365 app instance.

  3. Go to the Assignments tab.

  4. From the options menu for the user, select Edit user assignment.

  5. Set Manage licenses on create and update to true.

  6. Click Save user profile.
