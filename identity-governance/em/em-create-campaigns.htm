Create campaigns to audit entitlements

Use the steps specified in Create campaigns with the Campaign settings topic as a reference. For campaigns that include entitlements in the review, see resource settings specified here instead.

Resource settings for resource campaigns

Field Value
Type
Applications
  1. Enable the Review entitlements toggle to include entitlements in the campaign.

  2. Select an app that has Governance Engine enabled.

  3. Select the scope as either of the following options:

    • All entitlements and bundles

    • Specific entitlements and bundles

      1. Select entitlements and bundles that should be reviewed in this campaign.

      2. Optional. Choose Include all entitlement values for an entitlement to include all of its values, including any new values that are added before the campaign launches.

Groups Select one or more groups.

Don’t select Group if you want to include entitlements in the campaign. The Review entitlements toggle isn’t available if you select Group.

Resource settings for user campaigns

Field Value
Resource scope
All apps and groups Select this option to include all apps and groups assigned to the users you selected earlier.
All apps Select this option to include all apps assigned to the users you selected earlier.
All groups Select this option to include all groups assigned to the users you selected earlier.

Don’t select All groups if you want to include entitlements in the campaign. The Only include individually assigned entitlements option isn’t available if you select All groups.
Include options
Only include individually assigned apps Select this checkbox to restrict the resource scope to apps that were individually assigned to users.

Apps assigned by a group aren't included. Use this option to avoid redundant reviews if you're reviewing both apps and groups assigned to a user

(since the group that assigns apps and groups has already been reviewed).

Only include individually assigned entitlements Select this checkbox to exclude entitlements that were assigned by entitlement policy.
Only include individually assigned groups Select this checkbox to restrict the resource scope to groups that were individually assigned to users.

Any groups assigned by a group rule aren't included. This option is helpful when you're confident about the resources assigned by group rules and only want to review groups that were assigned outside of the group rules.
Exclude options
Exclude specific apps from the campaign Select this checkbox and identify apps that should be excluded from the campaign.
Exclude specific groups from the campaign Select this checkbox and identify groups that should be excluded from the campaign.
