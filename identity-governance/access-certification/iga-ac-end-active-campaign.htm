End an active campaign

As a super admin or an access certifications admin, you can end an active campaign before it's scheduled end date.

Here are some scenarios in which you might want to end an active campaign:

  • You need to relaunch the campaign with a different configuration.
  • Most the reviews have been completed and the remaining items don't need to be reviewed.
  • There's an error in the campaign.

While scheduling the campaign, the remediation options that you configure affect what happens when the campaign ends. If you select Remove from resource as the remediation option for when the reviewer doesn't respond and then, manually end the campaign, the system removes all those users from the resource whose access certification wasn't reviewed. See Remediation settings.

Before you begin

  • Sign in as a super admin or access certifications admin.
  • Check that you have an active campaign.

Start this task

  1. In the , go to Identity Governance .
  2. Go to the Active tab and select the campaign that you want to end.
  3. Open the Actions dropdown and select End.
  4. Click End campaign.
