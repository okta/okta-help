Past Campaign Summary report

Use this report to see high-level configurations and status of access certification campaigns. For example, you can review the types of resources covered in each campaign and its completion percentage, among other data points. See Access Certifications.

Before you begin

  • Sign in as a super admin, org admin, read-only admin, mobile admin, or reports admin.
  • Check that you have completed access certification campaigns. See Create an access certification campaign.
  • Disable your browser’s pop-up blocker.

Get report

  2. In the Access Certification Campaigns section click Past Campaign Summary.
Field

Value
Campaign name Enter the name of the campaign to include or exclude.
Campaign started Select a date if the operator is before or after.
Campaign ended Select a date if the operator is before or after.
Resource type Select the type of resource reviewed.
Percentage completed Enter a number from 1 to 100.
